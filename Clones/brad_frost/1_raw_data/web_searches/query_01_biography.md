### Atomic Design Methodology

**Definition:** A methodology for creating and maintaining effective interface design systems, inspired by chemistry.

**Core Concept:** Breaking down user interfaces into fundamental building blocks:
1. **Atoms** - Basic HTML elements (inputs, buttons, labels)
2. **Molecules** - Simple groups of UI elements (search form = label + input + button)
3. **Organisms** - Complex UI components (header with logo + nav + search)
4. **Templates** - Page-level objects placing components into layout
5. **Pages** - Specific instances of templates with real content

**Philosophy:** Creating thoughtful hierarchies within user interfaces, building and maintaining design systems that promote consistency and scalability.

**Origin:** Started as blog post, expanded into definitive book
