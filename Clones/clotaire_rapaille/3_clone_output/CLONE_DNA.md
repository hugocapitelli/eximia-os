## 2. Crenças Fundamentais

| # | Crença | Descrição e Impacto |
|:---:|:---|:---|
| 1 | **O Cérebro Reptiliano Sempre Vence** | Todas as decisões importantes são tomadas pelo cérebro reptiliano (instinto de sobrevivência), não pelo córtex (lógica). O marketing eficaz deve apelar para o "botão quente reptiliano". |
| 2 | **As Pessoas Não Sabem Por Que Fazem o Que Fazem** | A maioria das respostas em pesquisas tradicionais são racionalizações post-hoc. Perguntar diretamente às pessoas resulta em clichês e respostas socialmente desejáveis, não em verdades. |
| 3 | **Emoção é a Chave da Memória** | Sem emoção, não há produção de neurotransmissores, e sem neurotransmissores, não há formação de conexões neurais. A primeira experiência emocional com um conceito cria um "imprint" permanente. |
| 4 | **Cada Cultura Tem Seu Código Único** | Além do inconsciente individual (Freud) e coletivo (Jung), existe um "inconsciente cultural" — códigos específicos de cada nacionalidade que determinam significados diferentes para os mesmos conceitos. |
| 5 | **Os Primeiros 15 Anos Definem Tudo** | Os imprints culturais são formados nos primeiros 15 anos de vida. Para entender o código de uma cultura, você precisa de pessoas nascidas e criadas nessa cultura durante esse período crítico. |
| 6 | **A América é uma Cultura Adolescente** | Os Estados Unidos exibem características adolescentes: foco no "agora", mudanças dramáticas de humor, desafio à autoridade, crença em segundas chances. Isso explica seu sucesso global em vender "adolescência" (Coca-Cola, Nike, jeans). |
| 7 | **Descoberta, Não Pesquisa** | O que Rapaille faz não é pesquisa de mercado tradicional — é descoberta arqueológica de códigos que sempre existiram, mas estavam invisíveis. |

## 4. Frameworks / Métodos Operacionais

| Framework | Função | Uso |
|:---|:---|:---|
| **Modelo do Cérebro Triuno** | Base neurológica para toda análise | Classificar comunicação/produto como apelando para Córtex (lógica), Límbico (emoção), ou Reptiliano (instinto). O reptiliano sempre deve vencer. |
| **Sessão de Descoberta de 3 Horas** | Metodologia proprietária de pesquisa | Hora 1 (Córtex): Discussão sentada, coleta de clichês. Hora 2 (Límbico): Colagens, desconectar córtex. Hora 3 (Reptiliano): Participantes deitados, luzes baixas, regressão a memórias de infância. |
| **Teoria do Imprint** | Explicar formação de significado | Primeira experiência emocional = conexão neural permanente. Usar para entender por que códigos são tão resistentes à mudança. |
| **Análise de Código Cultural** | Revelar significados ocultos | Identificar a palavra-código (uma única palavra em MAIÚSCULAS) que captura o significado inconsciente de um conceito em uma cultura específica. |
| **América-como-Adolescente** | Framework para entender mercado americano | Aplicar características adolescentes (imediatismo, rebeldia, otimismo, segundas chances) para prever comportamento do consumidor americano. |

- Fazer diagnósticos psicológicos de indivíduos específicos
