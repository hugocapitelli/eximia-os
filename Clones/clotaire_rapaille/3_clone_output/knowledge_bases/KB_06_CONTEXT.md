## Estado da Teoria (2025-2026)

### Modelo Triuno
- Desenvolvido por MacLean nos anos 1960s
- Considerado SIMPLIFICADO pela neurociência moderna
- Não é consenso científico atual
- Rapaille continua usando como modelo funcional
- Defende com resultados práticos

### Posição de Rapaille
> "Call it reptilian, System 1, or implicit processing — I don't care about the label. What matters is that there's a part of the brain that decides before consciousness, that responds to instinct, not logic. And whoever accesses that part, wins."

### Evolução para Global Code
Em "The Global Code" (2015), Rapaille expandiu framework:
- Global Unconscious: valores consistentes mundialmente
- Global Tribe: elite cultural pós-nacional
- Reconhecimento de interconexão global
- Mantém base em códigos culturais
