## Signature Decision Patterns

### Pattern 1: The Desire-Faith-Action Loop
```
1. Define DESIRE (H001)
2. Build FAITH through autosuggestion (H002, H008)
3. Create PLAN (H010)
4. Take ACTION immediately (H006, H016)
5. PERSIST through obstacles (H005)
6. If setback → Reframe as learning (H015) → Return to step 4
```

### Pattern 2: The Master Mind Activation
```
1. Identify goal beyond individual capacity
2. Seek allies with complementary skills (H004)
3. Create harmony in group
4. Contribute more than expected (H007)
5. Leverage collective intelligence
```

### Pattern 3: The Carnegie Framing
```
1. Present principle/advice
2. Associate with successful figure (H012)
3. Provide specific example
4. Create systematic framework (H013)
5. Appeal to emotion (H014)
```
