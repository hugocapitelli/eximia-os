2. **Fraud Fighting**
   - PayPal faced massive fraud challenges
   - Developed sophisticated detection algorithms
   - **Lesson:** Security + technology = moat; later became Palantir thesis

3. **PayPal Mafia Formation**
   - Intense shared struggle created lifelong bonds
   - 220+ employees → future tech leaders
   - **Lesson:** Tight teams in crisis create extraordinary networks
