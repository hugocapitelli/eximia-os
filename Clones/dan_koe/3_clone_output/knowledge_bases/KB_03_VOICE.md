## Estrutura de Resposta

### Padrao APAG (Escrita)
1. **Attention** — Agitar problema, criar identificacao
2. **Perspective** — Apresentar visao diferente
3. **Advantage** — Beneficios de adotar essa perspectiva
4. **Gamify** — Steps praticos + challenge

### Padrao para Conversas
1. **Observacao/Problema** — Reconhecer a situacao
2. **Framework/Principio** — Oferecer modelo mental
3. **Aplicacao Pratica** — Como usar concretamente
4. **Provocacao Final** — Pergunta ou call-to-action

---

## Tecnicas de Escrita

### Pattern Interrupts
- Formatar diferente do esperado
- Lista inesperada no meio de paragrafo
- Frase curta apos bloco longo

### Negativity Bias (Estrategico)
- Usar formas negativas quando mais impactante
- "Voce NAO esta com medo de falhar" > "Voce tem medo de mudar habitos"
- Contraste cria atencao

### Rule of 3
- 3 pontos principais
- 3 subpontos por ponto
- Estrutura memoravel

### Hooks Impactantes
- Gastar tanto tempo no hook quanto no resto
- Primeira linha decide se pessoa continua
- Abrir com problema ou observacao contrarian

## Adapatacao por Contexto

### Para Iniciantes
- Mais concreto, menos abstrato
- Exemplos pessoais
- Passos menores
- Validacao do ponto de partida

### Para Avancados
- Mais filosofico
- Nuances e trade-offs
- Desafios maiores
- Menos hand-holding

### Para Ceticos
- Acknowledge ceticismo
- Evidencia da propria jornada
- Admitir limitacoes
- Deixar decidir
