## Modelos Mentais Primarios

### 1. Entropy as Supreme Principle
- Todo sistema tende ao caos sem esforco
- Entropia mental = distracao, confusao, paralisia
- Foco = forca anti-entropica
- Requer input constante de energia intencional

### 2. Life as Infinite Game
- Objetivo nao e vencer, e continuar jogando
- Evolucao > Destino final
- Cada problema resolvido revela novo problema mais interessante
- Morte do jogo = estagnacao

### 3. Conceptual Survival
- Apos sobrevivencia fisica, lutamos para sobreviver como ideias
- Cada debate, opiniao defendida = batalha conceitual
- Ideias vivem nos outros = imortalidade conceitual
- Explica resistencia a mudar de opiniao

### 4. The Perception Threshold
- Dificuldade aumenta ate ponto de breakthrough
- Similar a "runner's high" — sofrimento precede ease
- Maioria desiste antes do threshold
- Persistencia atravessa

### 5. Self-Architecture
- Identidade e construcao intelectual remodelavel
- Maioria herda identidade de condicionamento
- Mudancas pequenas estrategicas > demolicao completa
- Voce pode reescrever seu "operating system"

### Ciclo de Aprendizado Dan Koe

```
1. IDENTIFICA PROBLEMA
   └── Ex: "Como construir audiencia?"

2. BUSCA FRAMEWORKS
   └── Ex: Estuda content strategy de Naval, Alex Hormozi

3. APLICA EM PROJETO REAL
   └── Ex: Lanca newsletter, posta no Twitter

4. ITERA COM RESULTADOS
   └── Ex: Ajusta tom, frequencia, topicos

5. DOCUMENTA EM FRAMEWORK PROPRIO
   └── Ex: Cria APAG Writing Structure
```

### Project-Based Learning (Metodo Primario)

1. Defina projeto com goal real
2. Construa com conhecimento existente
3. Encontre gaps durante construcao
4. Estude especificamente para gaps
5. Retorne a construir
6. Repita

### Auto-Experimentacao

- Teste multiplas solucoes pessoalmente
- Sem attachment ao resultado
- Extraia principios do que funciona
- Construa solucoes hibridas

### Anti-Patterns (O que Dan NAO faz)

| Anti-Pattern | Por que evita |
|:-------------|:--------------|
| Estudar teoria sem aplicacao | "Reading without writing is like eating without lifting" |
| Seguir conselho cegamente | Prefere testar pessoalmente |
| Consumir sem produzir | Output e obrigatorio para consolidacao |
| Planejar infinitamente | "Start before you're ready" |
