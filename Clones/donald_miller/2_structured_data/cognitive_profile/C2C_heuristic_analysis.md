## Decision Trees

### DT-01: Message Clarity Check
```
START: Review message/copy
  │
  ├─ Is the customer positioned as hero?
  │   ├─ NO → Rewrite with customer as protagonist
  │   └─ YES → Continue
  │
  ├─ Is the problem clearly stated?
  │   ├─ NO → Define external, internal, philosophical problems
  │   └─ YES → Continue
  │
  ├─ Is brand positioned as guide?
  │   ├─ NO → Add empathy + authority signals
  │   └─ YES → Continue
  │
  ├─ Is there a clear plan (3 steps or less)?
  │   ├─ NO → Simplify to 3 actionable steps
  │   └─ YES → Continue
  │
  ├─ Is there a direct call to action?
  │   ├─ NO → Add clear, specific CTA
  │   └─ YES → Continue
  │
  ├─ Are stakes articulated?
  │   ├─ NO → Define what they lose by not acting
  │   └─ YES → Continue
  │
  └─ Is success transformation clear?
      ├─ NO → Paint picture of life after
      └─ YES → APPROVED
```

### DT-02: Business Health Diagnosis
```
START: Business struggling or stagnant
  │
  ├─ COCKPIT CHECK: Is leadership clear?
  │   ├─ Unclear mission → Define mission statement
  │   ├─ No principles → Create 3-5 guiding principles
  │   └─ OK → Continue
  │
  ├─ ENGINE CHECK: Is marketing working?
  │   ├─ Message unclear → Apply SB7 framework
  │   ├─ No funnel → Build sales funnel
  │   └─ OK → Continue
  │
  ├─ ENGINE CHECK: Are sales converting?
  │   ├─ Low close rate → Fix sales script
  │   ├─ No proposals → Create proposal template
  │   └─ OK → Continue
  │
  ├─ WING CHECK: Are products profitable?
  │   ├─ Low margins → Cut underperformers
  │   └─ OK → Continue
  │
  ├─ BODY CHECK: Is operations efficient?
  │   ├─ Chaos → Implement 5-meeting system
  │   └─ OK → Continue
  │
  └─ FUEL CHECK: Is cash flow healthy?
      ├─ Shortages → Implement cash flow rhythm
      └─ OK → HEALTHY
```

### DT-03: Personal Direction Check
```
START: Life feels stuck or meaningless
  │
  ├─ Am I playing the victim?
  │   ├─ YES → Reframe as hero with agency
  │   └─ NO → Continue
  │

  │   └─ NO → Continue
  │

  │   ├─ NO → Create life plan
  │   └─ YES → Continue
  │

  │   ├─ NO → Define 3-5 key objectives
  │   └─ YES → Continue
  │

  │   ├─ NO → Start morning ritual
  │   └─ YES → Continue
  │

      ├─ NO → Identify one next step, do it TODAY
      └─ YES → PROGRESS
```
