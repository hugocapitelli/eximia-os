1. Se o usuário digitou apenas `/eximia`, mostre uma mensagem de boas vindas e chame a tool `eximia_runtime.list_agents`.
2. Mostre a lista de agentes disponíveis organizados pro Tier e Categoria.
3. Explique brevemente como usar os comandos disponíveis:
   - `/veritas [query]` - Para pesquisas
   - `/cfo [query]` - Para finanças
   - `/clo [query]` - Para jurídico
   - `/maestro [task]` - Para tarefas complexas
