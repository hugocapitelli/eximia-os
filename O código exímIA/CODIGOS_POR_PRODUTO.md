## Checklist de Validação

Antes de publicar qualquer comunicação, verifique:

### O texto ativa o reptiliano?

- [ ] Fala de medo/desejo profundo (não feature)
- [ ] Usa palavras que ativam (ver listas acima)
- [ ] Evita palavras que NÃO ativam
- [ ] Cria imagem mental imediata
- [ ] Faz o leitor sentir algo no corpo

### O texto está alinhado ao código?

- [ ] ExímIA OS → CONTROLE
- [ ] Academy → PODER SEM EXPOSIÇÃO
- [ ] Clones → IMORTALIDADE
- [ ] Geral → AMPLIFICAÇÃO

### O texto evita armadilhas do córtex?

- [ ] Não lista features
- [ ] Não usa jargão técnico
- [ ] Não explica demais
- [ ] Não pede desculpas
- [ ] Não hedging ("talvez", "pode ser")
