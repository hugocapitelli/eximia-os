# KB_10 — Financial Restructuring

## 1. Fundamentos de Reestruturação

### O Que é Reestruturação Financeira
Segundo Gilson (Creating Value Through Corporate Restructuring, 2010), reestruturação é o processo de mudar significativamente a estrutura financeira, operacional ou organizacional de uma empresa em dificuldades.

### Tipos de Reestruturação

| Tipo | Objetivo | Contexto |
| :--- | :--- | :--- |
| **Financial** | Ajustar capital structure | Alavancagem excessiva |
| **Operational** | Melhorar eficiência | Margens baixas |
| **Strategic** | Mudar modelo de negócio | Disrupção |
| **Judicial** | Proteção legal | Insolvência |

## 2. Sinais de Distress

### Early Warning Signs (Altman Z-Score, 1968)

```
Z = 1.2(WC/TA) + 1.4(RE/TA) + 3.3(EBIT/TA) + 0.6(MVE/TL) + 1.0(Sales/TA)

Z > 2.99: Safe Zone
1.81 < Z < 2.99: Grey Zone
Z < 1.81: Distress Zone
```

### Red Flags Financeiros

| Indicador | Threshold | Ação |
| :--- | :--- | :--- |
| **ICR < 1.5x** | EBITDA/Interest | Covenant breach risk |
| **Debt/EBITDA > 5x** | Total leverage | Refinancing risk |
| **FCF Negative** | 2+ quarters | Liquidity crisis |
| **Current Ratio < 1.0x** | CA/CL | Working capital stress |
| **Covenant Breach** | Any | Lender negotiation |

## 3. Opções de Reestruturação

### Out-of-Court Options

| Opção | Descrição | Timing |
| :--- | :--- | :--- |
| **Refinancing** | Nova dívida mais favorável | Early distress |
| **Amendment** | Waiver ou covenant reset | Covenant breach |
| **Exchange Offer** | Troca de dívida por equity | Moderate distress |
| **Distressed Sale** | Venda de ativos/empresa | Any stage |

### In-Court Options (Brasil)

| Opção | Lei | Descrição |
| :--- | :--- | :--- |
| **Recuperação Judicial** | Lei 11.101/2005 | Reorganização |
| **Recuperação Extrajudicial** | Lei 11.101/2005 | Acordo pré-negociado |
| **Falência** | Lei 11.101/2005 | Liquidação |

## 4. Processo de Turnaround

### Fases (McKinsey, Turnaround, 2018)

```
Fase 1: STABILIZATION (0-90 dias)
├── Cash management intensivo
├── Comunicação com stakeholders
├── Quick wins (custos)
└── Assessment inicial

Fase 2: RESTRUCTURING (90-180 dias)
├── Renegociação de dívida
├── Rightsizing operacional
├── Asset sales
└── Plano de longo prazo

Fase 3: GROWTH (180+ dias)
├── Investimentos seletivos
├── Reconstrução de confiança
├── Novas fontes de capital
└── Monitoramento contínuo
```

### Cash Management Emergencial

| Ação | Impacto | Timeline |
| :--- | :--- | :--- |
| 13-week cash forecast | Visibilidade | Imediato |
| Payment prioritization | Preservar cash | Imediato |
| AR acceleration | Gerar caixa | 30 dias |
| AP stretch | Preservar cash | 30 dias |
| Inventory liquidation | Gerar caixa | 60 dias |

## 5. Renegociação de Dívida

### Estratégias com Credores

| Estratégia | Credor Preference | Empresa Preference |
| :--- | :--- | :--- |
| **Maturity Extension** | Medium | High |
| **Rate Reduction** | Low | High |
| **Principal Haircut** | Very Low | Very High |
| **Debt-Equity Swap** | Medium | High |
| **Amend & Extend** | High | Medium |

### Waterfall de Credores

```
Priority Order (Brasil):
1. Trabalhista (até 150 SM por credor)
2. Garantias reais (secured)
3. Tributário
4. Quirografário (unsecured)
5. Subquirografário
```

### Term Sheet de Reestruturação

| Elemento | Descrição |
| :--- | :--- |
| Coupon adjustment | Taxa nova |
| Maturity | Prazo novo |
| Amortization | Novo cronograma |
| Covenants | Novos + waivers |
| Equity kicker | Warrants, conversão |
| Security | Garantias adicionais |

## 6. Recuperação Judicial (Brasil)

### Processo (Lei 11.101/2005)

```
Dia 0: Pedido de RJ
├── Demonstração de crise
├── Última contabilidade
└── Lista de credores

Dia 60: Deferimento
├── Stay automático (180 dias)
├── Administrador judicial
└── Verificação de créditos

Dia 150: Plano de Recuperação
├── Proposta aos credores
├── Classes: trabalhista, garantia, quirografário
└── Assembleia Geral de Credores

Dia 180: Homologação
├── Aprovação judicial
├── Novação das dívidas
└── Início da execução
```

### Classes de Credores

| Classe | Quorum |
| :--- | :--- |
| I - Trabalhista | Maioria simples (cabeças) |
| II - Garantia Real | Maioria simples + 60% créditos |
| III - Quirografário | Maioria simples + 60% créditos |
| IV - ME/EPP | Maioria simples (cabeças) |

## 7. Valuation em Distress

### Ajustes Necessários (Damodaran, 2012)

| Fator | Ajuste |
| :--- | :--- |
| WACC | +5-10% distress premium |
| Terminal Value | Conservador ou zero |
| FCF | Normalized, post-turnaround |
| Probability weights | Bear/Base/Bull com default |

### Liquidation Value

```
Liquidation Value = Σ(Asset × Recovery Rate) - Liabilities

Recovery Rates (típicos):
- Cash: 100%
- AR: 50-80%
- Inventory: 30-60%
- PP&E: 20-50%
- Intangibles: 0-20%
```

## 8. Stakeholder Management

### Comunicação em Crise

| Stakeholder | Mensagem Key |
| :--- | :--- |
| **Bancos** | Plano, timeline, covenant |
| **Fornecedores** | Continuidade, pagamentos |
| **Clientes** | Serviço mantido |
| **Funcionários** | Transparência, futuro |
| **Investidores** | Valor residual, recovery |

### BATNA para Credores
Best Alternative to Negotiated Agreement:
- Liquidation value é o floor
- Credor só aceita se > liquidation

## Invariantes de Reestruturação

1. **13-week cash forecast** obrigatório em crise
2. **Liquidation value é o floor** de negociação
3. **Trabalhista primeiro** no waterfall Brasil
4. **Stay de 180 dias** na RJ (prorrogável)
5. **60% + maioria** para aprovar plano
6. **Distress premium: 5-10%** no WACC
