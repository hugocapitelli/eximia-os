**Uses (Para onde vai o dinheiro):**
| Use | Components |
| :--- | :--- |

| Refinance Existing Debt | Current debt payoff |
| Transaction Fees | 2-3% of EV |

### IRR Calculation

```
IRR = (Exit Value / Entry Equity) ^ (1/n) - 1

Target IRR by Fund Type:
- Large Cap: 15-20%
- Mid Market: 20-25%
- Small Cap: 25-30%
```

## 4. Due Diligence for PE

### Commercial DD

| Area | Key Questions |
| :--- | :--- |
| Market | TAM, growth, competition |
| Customers | Concentration, churn, NPS |
| Products | Differentiation, lifecycle |
| Pricing | Power, elasticity |

### Financial DD

| Area | Focus |
| :--- | :--- |
| Quality of Earnings | Normalized EBITDA, addbacks |
| Working Capital | Net working capital peg |
| CapEx | Maintenance vs growth |
| Off-BS Items | Leases, pensions, contingencies |

### Management Assessment (Kaplan, 2009)
1. Track record
2. Skin in the game (rollover equity)
3. Incentive alignment (carry structure)
4. Bench strength
