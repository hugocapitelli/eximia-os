## 2. Synergy Analysis

### Types of Synergies (McKinsey, 2020)

| Type | Examples | Realization Probability |
| :--- | :--- | :--- |
| **Cost Synergies** | Headcount, Facilities, Procurement | 60-80% |
| **Revenue Synergies** | Cross-sell, Market access | 20-40% |
| **Financial Synergies** | Lower WACC, Tax shields | 70-90% |

### Haircut Guidelines

Segundo McKinsey (Valuation, 2020), aplicar haircuts:

```
Synergy Type      Announced â†’ Realized
Cost synergies:   -30% to -40%

```

### Brasil Case: Ambev-InBev (2004)
- Announced synergies: $200M
- Realized synergies: ~$150M (75%)
- Timeline: 3 anos (vs 2 prometidos)

## 3. Due Diligence Framework

### Financial DD Checklist (Big 4 Standard)

| Area | Key Items | Red Flags |
| :--- | :--- | :--- |
| **Revenue Quality** | Customer concentration, Churn, NRR | Top 3 clients >50% |
| **Margin Sustainability** | Gross margin trend, One-offs | Declining margins |
| **Working Capital** | DSO, DIO, DPO | Increasing WC needs |
| **Cash Flow** | FCF conversion, CapEx needs | OCF << Net Income |
| **Debt** | Covenants, Maturities | Covenant breaches |
| **Off-Balance Sheet** | Leases, Guarantees, Contingencies | Material contingencies |

### Brazil-Specific DD
- Trabalhista contingencies (often understated)
- Tax contingencies (ICMS, ISS disputes)
- Related party transactions
- Minority squeeze-out rules (Lei das S.A.)

## 5. 100-Day Integration Plan

### Framework McKinsey (2017)

**Day 1-30: Stabilization**
- Retain key people (retention bonuses)
- Communicate to customers
- Quick wins identification

**Day 31-60: Foundation**
- Synergy capture begins
- Systems integration planning
- Org structure decisions

**Day 61-100: Acceleration**
- Major synergy realization
- Culture integration
- Performance tracking

### Success Metrics
| Metric | Target | Timeline |
| :--- | :--- | :--- |
| Key talent retention | >90% | Day 90 |
| Customer retention | >95% | Day 180 |
| Synergy capture | 50% | Year 1 |
| Synergy capture | 80% | Year 2 |

## 6. Deal Structuring

### Consideration Types
| Type | Buyer Preference | Seller Preference |
| :--- | :--- | :--- |
| All Cash | Risk transfer | Liquidity, certainty |
| Stock | Conserve cash | Upside participation |
| Earnout | De-risk valuation | Higher total value |

### Earnout Best Practices (NVCA, 2020)
- Metrics must be objective (Revenue, EBITDA)
- Maximum 2-3 years
- Threshold + stretch targets
- Escrow arrangements
