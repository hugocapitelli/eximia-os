### Reguladores Principais

| Entidade | Função | Foco |
| :--- | :--- | :--- |

## 4. Regime Fácil (CVM 231/232)

### Vigência: Janeiro 2026

Objetivo: Facilitar acesso de empresas menores ao mercado.

| Critério | Limite |
| :--- | :--- |
| **Elegibilidade** | Receita ≤ R$500M |
| **Captação Anual** | R$300M máximo |

### Benefícios

| Área | Simplificação |
| :--- | :--- |
| **Formulário** | "Formulário Fácil" vs Referência |
| **Resultados** | Semestrais (vs trimestrais) |
| **Sustentabilidade** | Isenção de relatório |
| **Cancelamento** | OPA com quórum reduzido |

## 5. BACEN — Regulação Cambial

### Operações de Câmbio

| Tipo | Regulação |
| :--- | :--- |
| **Importação/Exportação** | Registro automático |
| **Investimento Estrangeiro** | ROF (Registro Operação Financeira) |
| **Empréstimo Externo** | ROF + conditions |
| **Remessa de Dividendos** | Livre, com registro |

### Thin Capitalization (Lei 12.249/2010)
- Debt/Equity ratio: 2:1 (partes relacionadas)
- Excesso de juros: indedutível para IR

| Câmbio (geral) | 0.38% |

| Empréstimo externo >1 ano | 0% |
| Investimento estrangeiro | 0% |

## 6. Lei das S.A. (Lei 6.404/1976)

### Direitos de Minoritários

| Direito | Threshold | Descrição |
| :--- | :--- | :--- |
| **Tag-Along** | Varies | Direito de venda junto |
| **Conselho Fiscal** | 10% | Instalação obrigatória |
| **Acionista Relevante** | 5% | Disclosure obrigatório |
| **Voto em separado** | 10% ON | Eleição de conselheiro |

### Dividendos Obrigatórios
- Mínimo: 25% do lucro líquido ajustado
- Pode ser alterado no estatuto

## Invariantes de Regulação Brasil

1. **CVM 400 vs 476**: know quando usar cada
2. **Tag-along 100%** no Novo Mercado
3. **ITR: 45 dias** após trimestre
4. **Free float mínimo: 25%** nos níveis diferenciados
5. **Thin cap: 2:1** debt/equity (relacionadas)
6. **Dividendo mínimo: 25%** (padrão estatutário)
7. **IRPJ + CSLL = 34%** corporate tax rate
