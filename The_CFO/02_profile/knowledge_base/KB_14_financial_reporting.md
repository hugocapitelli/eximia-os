# KB_14 — Financial Reporting

## 1. IFRS vs US GAAP (2024)

### Philosophical Difference (KPMG, CFA Institute, 2024)

| Framework | Approach | Implicação |
| :--- | :--- | :--- |
| **US GAAP** | Rules-based | Guidance específica, menos julgamento |
| **IFRS** | Principles-based | Guidelines amplas, mais julgamento |

> *"Differences between US GAAP and IFRS are accelerating in 2024 annual reports."*
> — CFA Institute, 2024

### Key Differences (Abacum, Wall Street Prep, 2024)

| Área | US GAAP | IFRS |
| :--- | :--- | :--- |
| **Inventory** | LIFO permitido | Apenas FIFO |
| **Asset Revaluation** | Historical cost | Fair value opcional |
| **Leases** | Operating vs Finance | Single model (finance) |
| **Cash Flow Interest** | Operating | Flexibility (Op/Fin) |
| **Rev Recognition** | Industry guidance | Principles only |

### IFRS 18 (Deloitte, 2024)
Nova norma de apresentação de demonstrações financeiras:
- Categorias específicas na DRE
- Subtotais definidos obrigatórios
- Maior comparabilidade

## 2. Estrutura de Demonstrações

### Three Statements

```
INCOME STATEMENT (DRE)
├── Revenue
├── COGS
├── Gross Profit
├── Operating Expenses
├── Operating Income (EBIT)
├── Financial Income/Expense
├── EBT
├── Income Tax

BALANCE SHEET (BP)
├── Current Assets
├── Non-Current Assets
├── Total Assets
├── Current Liabilities
├── Non-Current Liabilities
├── Shareholders' Equity
└── Total Liabilities + Equity

CASH FLOW STATEMENT (DFC)
├── Operating Activities
├── Investing Activities
└── Financing Activities
```

## 6. Brasil Reporting

### CVM Requirements

| Documento | Frequência | Prazo |
| :--- | :--- | :--- |
| **ITR** | Trimestral | 45 dias |
| **DFP** | Anual | 3 meses |
| **Formulário de Referência** | Anual | Update contínuo |
| **Fato Relevante** | Ad hoc | Imediato |

### CPC/IFRS Brasil
- CPCs convergem com IFRS
- CPC 47 = IFRS 15 (Revenue)
- CPC 06 = IFRS 16 (Leases)
- CPC 48 = IFRS 9 (Financial Instruments)

### Regime Fácil (CVM 231/232, vigência 2026)
Para empresas até R$500M de faturamento:
- Formulário Fácil (simplificado)
- Resultados semestrais (vs trimestrais)
- Isenção relatório sustentabilidade
- Captação anual até R$300M
