# KB_07 — Private Equity & LBO

## 1. Fundamentos de Private Equity

### O Que é Private Equity
Segundo Kaplan & Strömberg (Journal of Finance, 2009), Private Equity envolve aquisição de empresas usando significativa alavancagem, com objetivo de melhorar operações e vender com lucro.

### Tipos de PE

| Tipo | Stage | Check Size | Holding |
| :--- | :--- | :--- | :--- |
| **Buyout (LBO)** | Mature | $100M+ | 4-7 anos |
| **Growth Equity** | Expansion | $20-100M | 3-5 anos |
| **Venture Capital** | Early | $1-20M | 5-10 anos |
| **Distressed** | Turnaround | Varies | 2-4 anos |

## 2. LBO Fundamentals

### Estrutura de Capital LBO (Rosenbaum & Pearl, 2020)

```
┌─────────────────────────────────────────────────┐
│          TYPICAL LBO CAPITAL STRUCTURE          │
├─────────────────────────────────────────────────┤
│                                                  │
│  Senior Secured (Term Loan A/B)    40-50%       │
│  ├── Lowest cost, first priority                │
│  │                                               │
│  Subordinated/Mezzanine            10-20%       │
│  ├── Higher cost, junior to senior              │
│  │                                               │
│  Equity (Sponsor + Management)     30-40%       │
│  └── Highest return requirement                  │
│                                                  │
└─────────────────────────────────────────────────┘
```

### Key Ratios

| Ratio | Definition | Typical Range |
| :--- | :--- | :--- |
| **Debt/EBITDA** | Total Debt / EBITDA | 4-6x |
| **Senior/EBITDA** | Senior Debt / EBITDA | 3-4x |
| **Interest Coverage** | EBITDA / Interest | >2.0x |
| **Equity Contribution** | Equity / Purchase Price | 30-40% |

## 3. LBO Model Mechanics

### Sources & Uses

**Sources (De onde vem o dinheiro):**
| Source | Typical % |
| :--- | :---: |
| Revolver | 0-5% |
| Term Loan A | 15-25% |
| Term Loan B | 20-30% |
| High Yield Bonds | 10-20% |
| Sponsor Equity | 30-40% |
| Management Rollover | 5-10% |

**Uses (Para onde vai o dinheiro):**
| Use | Components |
| :--- | :--- |
| Purchase Enterprise Value | EBITDA × Multiple |
| Refinance Existing Debt | Current debt payoff |
| Transaction Fees | 2-3% of EV |
| Financing Fees | 2-3% of debt |

### IRR Calculation

```
IRR = (Exit Value / Entry Equity) ^ (1/n) - 1

Target IRR by Fund Type:
- Large Cap: 15-20%
- Mid Market: 20-25%
- Small Cap: 25-30%
```

### Value Creation Levers (McKinsey, 2018)

| Lever | Contribution |
| :--- | :--- |
| **EBITDA Growth** | 40-50% |
| **Multiple Expansion** | 20-30% |
| **Debt Paydown** | 20-30% |
| **Dividend Recap** | 0-10% |

## 4. Due Diligence for PE

### Commercial DD

| Area | Key Questions |
| :--- | :--- |
| Market | TAM, growth, competition |
| Customers | Concentration, churn, NPS |
| Products | Differentiation, lifecycle |
| Pricing | Power, elasticity |

### Financial DD

| Area | Focus |
| :--- | :--- |
| Quality of Earnings | Normalized EBITDA, addbacks |
| Working Capital | Net working capital peg |
| CapEx | Maintenance vs growth |
| Off-BS Items | Leases, pensions, contingencies |

### Management Assessment (Kaplan, 2009)
1. Track record
2. Skin in the game (rollover equity)
3. Incentive alignment (carry structure)
4. Bench strength

## 5. Exit Strategies

### Exit Routes

| Exit Type | Typical Timing | Buyer Type |
| :--- | :--- | :--- |
| **Strategic Sale** | 4-7 years | Corporate acquirer |
| **Secondary Buyout** | 4-6 years | Another PE fund |
| **IPO** | 5-8 years | Public markets |
| **Recap** | 2-4 years | Dividend to shareholders |

### Exit Multiple Drivers

Segundo Bain (PE Report, 2023):
- EBITDA growth → Multiple expansion
- Market conditions → Timing matters
- Strategic synergies → Premium in M&A


## 8. Red Flags in LBO

### Deal Breakers (Rosenbaum & Pearl, 2020)

| Red Flag | Risk |
| :--- | :--- |
| Customer concentration >40% | Revenue risk |
| EBITDA declining | Debt service risk |
| Aggressive addbacks | Earnings quality |
| Management not rolling | Alignment concern |
| Leverage >7x EBITDA | Default risk |
| Industry disruption | Secular decline |

---

## Invariantes de PE/LBO

1. **Debt/EBITDA ≤ 6x** para deals saudáveis
2. **Equity contribution ≥ 30%** (skin in the game)
3. **Target IRR: 20-25%** para mid-market
4. **Holding period: 4-7 years** típico
5. **Management rollover: 5-10%** alinhamento
6. **Exit multiple ≤ Entry + 2x** para ser conservador
