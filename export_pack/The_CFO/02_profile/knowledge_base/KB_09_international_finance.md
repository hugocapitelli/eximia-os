## 3. Hedging Strategies

### Instrumentos de Hedge

| Instrumento | Uso | Custo |
| :--- | :--- | :--- |
| **Forward Contract** | Lock rate futuro | Spread |
| **Futures** | Padronizado, exchange-traded | Margin + spread |
| **Options** | Proteção com upside | Premium |
| **Swaps** | Longo prazo, interesse + principal | Spread |
| **Natural Hedge** | Match revenues/costs em moeda | Zero |

### Hedge Ratio

```
Hedge Ratio = Exposure Hedged / Total Exposure

Conservative: 70-100%
Moderate: 40-70%
Aggressive: 0-40%
```

### Decision Framework

| Situação | Estratégia |
| :--- | :--- |
| High certainty exposure | Forward/Futures |
| Uncertain exposure | Options |
| Recurring exposure | Swaps |
| Long-term exposure | Natural hedge |

## 5. Transfer Pricing

### Arm's Length Principle (OECD, 2022)
Transações entre partes relacionadas devem refletir preços de mercado.

### Métodos

| Método | Descrição | Uso |
| :--- | :--- | :--- |
| **CUP** | Comparable Uncontrolled Price | Best if available |
| **Resale Price** | Resale minus margin | Distribution |
| **Cost Plus** | Cost plus markup | Manufacturing |
| **TNMM** | Net margin comparison | Most common |
| **Profit Split** | Split based on contribution | Intangibles |

### Documentação (Brasil IN 1312/2012)
- Master File: Grupo global
- Local File: Entidade brasileira
- CbCR: Country-by-Country Reporting

## 6. Capital Structure Internacional

### Subsidiary Funding

| Source | Pros | Cons |
| :--- | :--- | :--- |
| **Intercompany Loan** | Tax efficient | Thin cap rules |
| **Equity Injection** | No restrictions | No tax shield |
| **Local Debt** | Natural hedge | May be expensive |
| **Parent Guarantee** | Lower cost | Exposure |

### Thin Capitalization Rules (Brasil)
- Debt/Equity ratio: 2:1 (related parties)
- Excess interest: Non-deductible
- Fonte: Lei 12.249/2010

## Invariantes de International Finance

1. **Hedge ≥50% de exposição conhecida** (prudência)
2. **Country Risk Premium: adicionar no WACC** (Damodaran)
3. **Transfer Pricing: arm's length** (OECD, BACEN)
4. **Thin Cap Brasil: Debt/Equity ≤ 2:1** (Lei 12.249)
5. **BRL volatility: 20-30% anual** (histórico)
6. **ROF/RDE: obrigatório para FDI** (BACEN)
