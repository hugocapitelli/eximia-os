## Atividades por Formato

### Presencial (Sala de Aula)

| Atividade | Tempo | Participantes | Materiais |
|-----------|-------|---------------|-----------|
| **World Café** | 45-90 min | 12-100 | Flip-charts, mesas |
| **Open Space** | 2-8 horas | 5-2000 | Espaços, post-its |
| **Aquário (Fishbowl)** | 20-30 min | 8-30 | Cadeiras |
| **Jigsaw** | 30-60 min | 12-35 | Material segmentado |
| **Peer Teaching** | 15-30 min | 4+ | Variado |
| **Speed Dating** | 15-30 min | 10+ | Timer |

---

### Virtual Síncrono (Zoom, Teams)

| Atividade | Tempo | Ferramenta |
|-----------|-------|------------|
| **Breakout Rooms** | 10-20 min | Zoom/Teams |
| **Polls/Enquetes** | 2-5 min | Mentimeter, Slido |
| **Whiteboard colaborativo** | 15-30 min | Miro, Mural, Figjam |
| **Annotation** | 5-10 min | Zoom |
| **Chat storm** | 2-5 min | Chat nativo |
| **Video response** | 5 min | Loom, Flipgrid |

**Dica:** Alternar atividades a cada 10-15 minutos para manter atenção.

## Matriz: Complexidade Bloom x Atividade

| Nível Bloom | Atividades Sugeridas |
|-------------|---------------------|
| **Lembrar** | Quiz, flashcards, matching |
| **Entender** | Mapa mental, resumo, parafrasear |
| **Aplicar** | Simulação, exercício prático, roleplay |
| **Analisar** | Estudo de caso, comparação, debate |
| **Avaliar** | Peer review, julgamento, priorização |
| **Criar** | Projeto, protótipo, proposta, design |

## Blended Learning Patterns

### Modelo Flipped Classroom

```
ANTES (Assíncrono)
└── Conteúdo teórico (vídeo, leitura)

DURANTE (Síncrono)
└── Aplicação, discussão, dúvidas

DEPOIS (Assíncrono)
└── Prática individual, reflexão
```

### Modelo 70-20-10 Aplicado

| % | Tipo | Exemplo |
|---|------|---------|
| 70% | On-the-job | Projetos, assignments, rotação |
| 20% | Social | Mentoria, coaching, peer learning |
| 10% | Formal | Workshops, e-learning, cursos |
