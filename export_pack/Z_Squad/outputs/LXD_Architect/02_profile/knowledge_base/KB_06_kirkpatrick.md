# KB_06: Avaliação de Treinamento — Kirkpatrick

## Visão Geral

O modelo **Kirkpatrick** é o framework mais utilizado globalmente para avaliar a eficácia de programas de treinamento. Desenvolvido por **Donald Kirkpatrick** em 1959.

---

## Os 4 Níveis

```
┌─────────────────────────────────────────────────────────────────┐
│                    KIRKPATRICK 4 NÍVEIS                          │
│                                                                  │
│   NÍVEL 4: RESULTADOS                                           │
│   "Qual o impacto no negócio?"                                  │
│   └── ROI, KPIs, métricas organizacionais                       │
│                     ▲                                            │
│   NÍVEL 3: COMPORTAMENTO                                        │
│   "Estão aplicando no trabalho?"                                │
│   └── Observação, feedback de gestores                          │
│                     ▲                                            │
│   NÍVEL 2: APRENDIZAGEM                                         │
│   "O que aprenderam?"                                           │
│   └── Testes, demonstrações, avaliações                         │
│                     ▲                                            │
│   NÍVEL 1: REAÇÃO                                               │
│   "Gostaram do treinamento?"                                    │
│   └── Pesquisa de satisfação                                    │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

---

## Nível 1: Reação

### O que mede

Satisfação e percepção imediata dos participantes sobre o treinamento.

### Métodos de Avaliação

| Método | Quando | Exemplo |
|--------|--------|---------|
| Pesquisa post-training | Imediatamente após | Formulário NPS |
| Smile sheets | Final de cada dia | Escala 1-5 |
| Feedback verbal | Durante pausas | Conversa estruturada |

### Perguntas Típicas

1. O treinamento atendeu suas expectativas?
2. O conteúdo foi relevante para seu trabalho?
3. O instrutor foi eficaz?
4. Você recomendaria para colegas?

### Target

**≥ 4.0/5.0** ou **NPS ≥ 50**

### Limitações

- Satisfação ≠ Aprendizagem
- Instrutor carismático pode mascarar conteúdo ruim
- Necessário, mas não suficiente

[Kirkpatrick, D.L., Evaluating Training Programs, 1959]

---

## Nível 2: Aprendizagem

### O que mede

Aquisição de conhecimentos, habilidades e atitudes pretendidos.

### Métodos de Avaliação

| Método | Tipo | Melhor para |
|--------|------|-------------|
| Pré e pós-teste | Cognitivo | Conhecimento |
| Demonstração prática | Habilidades | Skills técnicos |
| Simulação/roleplay | Comportamental | Soft skills |
| Autoavaliação | Confiança | Atitudes |

### Modelo de Delta

```
Aprendizagem = Pós-teste - Pré-teste
                    Pré-teste

Melhoria de 70% = (85 - 50) / 50 = 70%
```

### Target

**≥ 70% de melhoria** ou **≥ 80% aprovação no pós-teste**

### Perguntas-chave

1. Conseguem explicar os conceitos principais?
2. Demonstram a habilidade em simulação?
3. Identificam situações de aplicação?

---

## Nível 3: Comportamento

### O que mede

Transferência do aprendizado para o trabalho real — **aplicação on-the-job**.

### Por que é difícil

- Depende de fatores além do treinamento (gestão, cultura, ferramentas)
- Leva tempo para ser observável
- Requer colaboração de gestores

### Métodos de Avaliação

| Método | Timing | Quem avalia |
|--------|--------|-------------|
| Observação direta | 30-90 dias | Gestor |
| Feedback 360 | 60-90 dias | Múltiplas fontes |
| KPIs de desempenho | Contínuo | Sistema |
| Self-report | 30-60 dias | Participante |
| Entrevistas | 30-90 dias | L&D |

### Barreiras à Transferência

| Barreira | Causa | Solução |
|----------|-------|---------|
| Sem oportunidade | Trabalho não permite | Redesign de função |
| Gestor não apoia | Cultura | Engajar gestores |
| Medo de errar | Ambiente | Segurança psicológica |
| Esquecimento | Tempo | Reforço espaçado |

### Target

**≥ 60% de aplicação relatada** ou **Melhoria observável em KPIs**

---

## Nível 4: Resultados

### O que mede

Impacto nos indicadores de negócio — resultados organizacionais mensuráveis.

### Exemplos de Métricas

| Área | Métricas |
|------|----------|
| Vendas | Receita, conversão, ticket médio |
| Operações | Produtividade, qualidade, erros |
| RH | Turnover, engajamento, absenteísmo |
| Clientes | NPS, CSAT, retenção |
| Custos | Redução de desperdício, retrabalho |

### Desafios

1. **Atribuição:** Como isolar efeito do treinamento de outras variáveis?
2. **Tempo:** Resultados podem demorar meses/anos
3. **Dados:** Nem sempre disponíveis ou acessíveis

### Métodos de Isolamento

- Grupo controle (com vs sem treinamento)
- Análise de tendência (antes/depois)
- Estimativa de especialistas (% atribuído ao treinamento)

## New World Kirkpatrick Model

### Evolução (Jim Kirkpatrick, 2016)

O modelo atualizado inverte a ordem de planejamento:

```
PLANEJAMENTO: Nível 4 → 3 → 2 → 1
(Começar pelo resultado desejado)

EXECUÇÃO: Nível 1 → 2 → 3 → 4
(Medir progressivamente)
```

### Adições do New Model

| Conceito | Descrição |
|----------|-----------|
| **Leading Indicators** | Sinais precoces de sucesso antes do Nível 4 |
| **Required Drivers** | Elementos que impulsionam ou inibem Nível 3 |
| **Kirkpatrick Chain** | Causalidade entre níveis |

[Kirkpatrick, J.D., & Kirkpatrick, W.K., Four Levels of Training Evaluation, 2016]

### Quando Avaliar em Qual Nível

| Tipo de Treinamento | L1 | L2 | L3 | L4 |
|---------------------|:--:|:--:|:--:|:--:|
| Compliance básico | ✅ | ✅ | ⚪ | ⚪ |
| Soft skills | ✅ | ✅ | ✅ | ⚪ |
| Técnico/processos | ✅ | ✅ | ✅ | ⚪ |
| Liderança | ✅ | ✅ | ✅ | ✅ |
| Estratégico/alto custo | ✅ | ✅ | ✅ | ✅ |

**Legenda:** ✅ = Obrigatório | ⚪ = Opcional
