# KB_13: Microlearning & Mobile Learning

## Visão Geral

**Microlearning** entrega conteúdo em pequenas doses (2-7 min) para consumo rápido. **Mobile learning** otimiza para dispositivos móveis.

---

## Princípios de Microlearning

### Características

| Aspecto | Microlearning | Tradicional |
|---------|---------------|-------------|
| Duração | 2-7 minutos | 30-90 minutos |
| Foco | Um objetivo/skill | Múltiplos |
| Formato | Vídeo curto, quiz, infográfico | Curso completo |
| Acesso | On-demand, any device | Sessão agendada |
| Frequência | Diário/semanal | Eventual |

---

### Quando Usar

| Ideal para | Não ideal para |
|------------|----------------|
| Reforço de aprendizado | Habilidades complexas iniciais |
| Just-in-time performance support | Desenvolvimento profundo |
| Onboarding fragmentado | Certificações formais |
| Updates e compliance | Mudança de mindset |
| Preparação/follow-up de workshops | Primeiro contato com tema |

---

## Formatos de Microlearning

| Formato | Duração | Engajamento | Produção |
|---------|---------|-------------|----------|
| **Vídeo** | 2-6 min | Alto | Médio |
| **Infográfico** | 1-2 min | Médio | Baixo |
| **Quiz** | 2-5 min | Alto | Baixo |
| **Cenário interativo** | 3-7 min | Alto | Alto |
| **Podcast/áudio** | 5-15 min | Médio | Baixo |
| **Flashcards** | 1-5 min | Médio | Baixo |
| **Checklist/job aid** | 1 min | Baixo | Baixo |

---

## Estrutura de Micro-Vídeo

### Template 5 min

```
0:00 - 0:15 | HOOK (atenção)
0:15 - 0:30 | CONTEXTO (por que isso importa)
0:30 - 3:30 | CONTEÚDO (3-5 pontos chave)
3:30 - 4:30 | EXEMPLO (demonstração)
4:30 - 5:00 | CALL TO ACTION (o que fazer agora)
```

---

## Mobile Learning

### Princípios Mobile-First

1. **Thumb-friendly:** Botões grandes, scrolling vertical
2. **Offline-capable:** Conteúdo baixável
3. **Bite-sized:** Módulos de ≤5 min
4. **Context-aware:** Notificações inteligentes
5. **Visual-heavy:** Menos texto, mais imagem

### O que Funciona em Mobile

| Funciona ✅ | Não funciona ❌ |
|-------------|----------------|
| Vídeos curtos verticais | Textos longos |
| Quiz simples | Formulários complexos |
| Flashcards | Planilhas |
| Checklists | Downloads pesados |
| Áudio/podcast | Navegação complexa |
