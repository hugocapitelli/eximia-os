# KB_09: Governança Corporativa

**Categoria:** TEORIA  
**Palavras:** ~2.500  
**Fonte Principal:** IBGC, Lei 6.404/76  

---

## 1. Fundamentos

### 1.1 Conceito (IBGC)

> "Governança corporativa é o sistema pelo qual as empresas e demais organizações são dirigidas, monitoradas e incentivadas, envolvendo os relacionamentos entre sócios, conselho de administração, diretoria, órgãos de fiscalização e controle e demais partes interessadas."  
> — Código IBGC, 6ª Ed.

### 1.2 Princípios IBGC

| Princípio | Descrição |
|-----------|-----------|
| **Transparência** | Disponibilizar informações além das obrigatórias |
| **Equidade** | Tratamento justo de todos os stakeholders |
| **Prestação de contas** | Accountability clara |
| **Responsabilidade corporativa** | Sustentabilidade e função social |

## 2. Estrutura de Governança

### 2.1 Órgãos

```
┌─────────────────────────────────────────────┐
│           ASSEMBLEIA GERAL                   │
│        (Acionistas / Quotistas)              │
├─────────────────────────────────────────────┤
│      CONSELHO DE ADMINISTRAÇÃO               │
│    (Estratégia, supervisão, fiscalização)    │
├────────────────┬────────────────────────────┤
│   COMITÊS      │       CONSELHO FISCAL      │
│ (Auditoria,    │     (Fiscalização legal)   │
│  Pessoas, etc) │                            │
├────────────────┴────────────────────────────┤
│              DIRETORIA                       │
│           (Gestão executiva)                 │
└─────────────────────────────────────────────┘
```

### 2.2 Conselho de Administração

| Aspecto | Recomendação IBGC |
|---------|-------------------|
| Composição | 5-11 membros |
| Independência | Maioria independente |
| Mandato | 1-2 anos |
| Reuniões | Mínimo 6/ano |
| Diversidade | Competências complementares |

**Definição de Independente:**
- Sem vínculo com a administração
- Sem participação acionária relevante
- Sem relacionamento comercial material
- Capacidade de julgamento independente

## 4. Acordo de Acionistas

### 4.1 Cláusulas Típicas

| Categoria | Cláusulas |
|-----------|-----------|
| **Voto** | Voto conjunto, indicação de administradores |
| **Transferência** | Lock-up, tag-along, drag-along, ROFR |
| **Governança** | Matérias reservadas, quóruns qualificados |
| **Saída** | Put/call options, buy-sell (shotgun) |
| **Impasse** | Mediação, arbitragem, deadlock resolution |

### 4.2 Eficácia (Art. 118 LSA)

| Condição | Efeito |
|----------|--------|
| Arquivado na sede | Oponível à companhia |
| Averbado nos livros | Oponível a terceiros |
| Presidente AG observa | Vinculante em votação |

## 5. Conflito de Interesses

### 5.1 Tipos

| Tipo | Descrição |
|------|-----------|
| **Direto** | Administrador é contraparte |
| **Indireto** | Interesse via partes relacionadas |
| **Potencial** | Ainda não materializado |

### 5.2 Tratamento (Art. 156 LSA)

1. **Disclosure** — Comunicar ao órgão competente
2. **Abstenção** — Não participar de deliberação
3. **Formalização** — Condições de mercado (arm's length)
4. **Aprovação** — Pelo órgão competente (sem voto do conflitado)

## 6. Responsabilidade de Conselheiros

### 6.1 Deveres

| Dever | Fonte | Descrição |
|-------|-------|-----------|
| Diligência | Art. 153 | Cuidado e zelo |
| Lealdade | Art. 155 | Interesse da companhia |
| Informar | Art. 157 | Transparência |
| Não conflito | Art. 156 | Abstenção em conflitos |

### 6.2 Business Judgment Rule

Proteção para decisões tomadas com:
- Boa-fé
- Informação adequada
- Ausência de conflito
- Interesse da companhia

**Não protege:** Negligência grosseira, dolo, violação legal.

## 7. Comitês de Apoio

### 7.1 Comitês Recomendados

| Comitê | Função |
|--------|--------|
| **Auditoria** | Supervisão de controles e DFs |
| **Pessoas/Remuneração** | Políticas de RH, remuneração |
| **Riscos** | Gestão de riscos corporativos |
| **Estratégia** | Análise de M&A, investimentos |
| **ESG/Sustentabilidade** | Questões ambientais, sociais |

### 7.2 Comitê de Auditoria

| Característica | Recomendação |
|----------------|--------------|
| Composição | Mínimo 3 membros |
| Independência | Pelo menos 1 independente |
| Expertise | Conhecimento em contabilidade/finanças |
| Função | Supervisão de auditoria, controles internos |
