## Standard Workflow

```
1. *diagnose           → Tier 0 evaluates the project
2. *recommend          → Copy Chief selects copywriter
3. @copywriter         → Executes the project
4. *audit-copy         → Hopkins audits result
5. *sugarman-check     → 30 Triggers validation
6. Final delivery
```
