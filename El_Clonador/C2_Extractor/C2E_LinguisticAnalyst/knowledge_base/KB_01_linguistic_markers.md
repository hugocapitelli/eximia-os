# KB_01 — Marcadores Linguísticos de Personalidade

## Categoria: REFERÊNCIA
## Palavras: ~2,500
## Atualizado: 2026-01-31

---

## 1. Fundamentos da Análise Linguística

### Hipótese Lexical

> *"Características de personalidade importantes ficam codificadas na linguagem natural que usamos."*

A forma como uma pessoa fala revela:
- **Traços de personalidade** (Big Five)
- **Estados emocionais** (positivo/negativo)
- **Estilos cognitivos** (analítico/narrativo)
- **Identidade social** (grupo, status)

---

## 2. LIWC (Linguistic Inquiry and Word Count)

### O que é LIWC?

Ferramenta de análise psicolinguística que categoriza palavras em dimensões psicológicas.

### Categorias Principais

#### 2.1 Pronomes

| Pronome | Alta Frequência Indica |
| :--- | :--- |
| **Eu/Meu** | Foco em si, possível neuroticismo, autorreflexão |
| **Nós/Nosso** | Mentalidade coletiva, liderança inclusiva |
| **Você/Vocês** | Engajamento com audiência, didatismo |
| **Eles/Deles** | Distanciamento, terceirização |

**Benchmark:**
- "Eu" em texto normal: ~4.5% das palavras
- "Nós" em líderes eficazes: ~2x mais que média

#### 2.2 Palavras de Emoção

| Categoria | Exemplos | Indica |
| :--- | :--- | :--- |
| **Positivas** | incrível, fantástico, amo, ótimo | Extraversão, otimismo |
| **Negativas** | problema, difícil, péssimo, odeio | Realismo ou neuroticismo |
| **Ansiedade** | preocupado, nervoso, medo | Alto neuroticismo |
| **Raiva** | irritado, furioso, puto | Baixa amabilidade |

#### 2.3 Palavras Cognitivas

| Categoria | Exemplos | Indica |
| :--- | :--- | :--- |
| **Causa** | porque, portanto, então | Pensamento causal |
| **Insight** | percebi, entendi, descobri | Processamento reflexivo |
| **Discrepância** | deveria, poderia, mas | Pensamento crítico |
| **Certeza** | sempre, nunca, definitivamente | Confiança ou rigidez |
| **Tentativo** | talvez, provavelmente, parece | Abertura ou insegurança |

## 3. Correlações Traço-Linguagem

### Big Five → Marcadores

| Traço | Marcadores Linguísticos |
| :--- | :--- |
| **Openness** | Vocabulário diverso, metáforas criativas, abstrações |
| **Conscientiousness** | Frases estruturadas, poucos fillers, organização |
| **Extraversion** | Palavras positivas, pronomes sociais, energia |
| **Agreeableness** | Palavras de família/comunidade, tom warm, concordâncias |
| **Neuroticism** | Mais "eu", palavras negativas, hedging, qualificadores |

### Pesquisa de Referência

**Extraversion:**
- Mais palavras positivas (+15% vs média)
- Mais concordâncias ("sim", "exato", "com certeza")
- Menos hedging ("talvez", "acho que")

**Neuroticism:**
- Mais pronome "eu" (+20% vs média)
- Mais palavras de ansiedade
- Mais qualificadores ("um pouco", "meio que")

**Agreeableness:**
- Mais palavras de família/comunidade
- Mais pronome "nós"
- Menos palavrões e linguagem agressiva

**Fonte:** Frontiers in Psychology (2022) — "Linguistic measures of personality in group discussions"

## 4. Análise de Idioleto

### Definição

**Idioleto** = perfil linguístico único de uma pessoa, formado por toda a vida de interações sociais.

### Componentes do Idioleto

1. **Vocabulário Preferido**
   - Palavras que usa mais que a média
   - Termos técnicos do domínio
   - Neologismos pessoais

2. **Estruturas Sintáticas**
   - Comprimento médio de frase
   - Complexidade gramatical
   - Ordem preferida de elementos

3. **Padrões Prosódicos**
   - Ritmo de fala
   - Uso de pausas
   - Padrões de ênfase

4. **Recursos Retóricos**
   - Metáforas favoritas
   - Estruturas de argumento
   - Padrões de storytelling

## 5. Marcadores Socioforenses

### Precisão de Identificação

| Contexto | Precisão |
| :--- | :--- |
| Speech patterns (prosody + intonation) | 75% |
| Acoustic + linguistic combined | 85% |
| Multimodal (text + audio + behavior) | 90%+ |

**Fonte:** SAGE Journals — "Identifying Key Idiolect Markers in Sociolinguistic Profiling" (2025)

### Marcadores Mais Distintivos

1. **Catch phrases** — Expressões repetidas
2. **Collocations** — Combinações de palavras únicas
3. **Sentence starters** — Como começa frases
4. **Discourse markers** — "então", "tipo", "né"

---

## 6. Taxonomia de Estruturas Argumentativas

### Padrões Comuns

| Padrão | Estrutura | Personalidade |
| :--- | :--- | :--- |
| **Declarativo** | Afirmação → Evidência → Conclusão | Alta dominância |
| **Socrático** | Pergunta → Exploração → Insight | Alta abertura |
| **Narrativo** | História → Moral → Aplicação | Alta influência |
| **Analítico** | Problema → Análise → Solução | Alto conscientiousness |

### Indicadores de Dominância (DISC-D)

- Frases imperativas ("Faça isso")
- Menos hedging
- Afirmações absolutas
- Menos perguntas, mais declarações

### Indicadores de Influência (DISC-I)

- Storytelling frequente
- Palavras emocionais positivas
- Engajamento com audiência
- Metáforas vívidas

## 7. Fontes de Metáfora

### Domínios Conceptuais

| Domínio | Exemplos | Mentalidade |
| :--- | :--- | :--- |
| **Guerra/Militar** | "batalha", "trincheira", "vitória" | Competitiva |
| **Esporte** | "jogo", "time", "placar" | Performance |
| **Ciência** | "experimento", "hipótese" | Sistemática |
| **Natureza** | "crescer", "raízes" | Orgânica |
| **Construção** | "fundação", "arquitetura" | Builder |
| **Viagem** | "jornada", "caminho" | Processo |
| **Máquina** | "engrenagem", "sistema" | Mecânica |

### Implicações para Clonagem

A fonte de metáfora revela o **framework mental** da pessoa. Um clone deve usar metáforas do mesmo domínio para soar autêntico.

## 8. Protocolo de Extração

### Fase 1: Coleta Quantitativa

1. Contar frequência de palavras (top 100)
2. Calcular ratios LIWC
3. Medir comprimento médio de frase
4. Identificar n-grams frequentes (2-3 palavras)

### Fase 2: Análise Qualitativa

1. Identificar catch phrases
2. Mapear estruturas argumentativas
3. Catalogar metáforas por domínio
4. Documentar padrões prosódicos

### Fase 3: Síntese

1. Criar regras de "sempre fazer"
2. Criar regras de "nunca fazer"
3. Gerar templates de frase
4. Calibrar tom (formalidade, energia, warmth)
