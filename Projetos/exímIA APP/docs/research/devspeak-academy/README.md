## Resumo por Módulo

### 1. UI & Glossário
Terminologias visuais, componentes, layouts e interações essenciais para desenvolvimento Pixel Perfect.

**Categorias:**
- Terminologia de UI (viewport, breakpoint, pixel density, DPR, retina)
- Tipos de componentes (50+ componentes: buttons, modals, cards, etc.)
- Sistemas de layout (Grid, Flexbox, posicionamento CSS)
- Design visual (tipografia, cores, sombras, gradientes)
- Padrões de interação (estados, transições, gestos, scroll)
- Acessibilidade (ARIA roles, landmarks, foco, semântica)
- Design System (tokens, atomic design, tematização)
- Unidades CSS e Media Queries

---

### 2. Sistema de Arquivos
Estrutura de pastas, imports, exports e organização de projetos React/Next.js.

**Categorias:**
- Extensões de arquivo (.js, .jsx, .ts, .tsx, .module.css, etc.)
- Import/Export (named, default, re-exports, barrel files, dynamic imports)
- Sistemas de módulos (CommonJS vs ES Modules)
- Estrutura React (components/, hooks/, services/, etc.)
- Next.js específico (App Router, arquivos especiais, convenções de rotas)
- Arquivos de configuração (package.json, tsconfig, next.config, etc.)
- Path Aliases (@/, configuração)
- Padrões de organização (Feature-Based, Atomic Design, DDD, Colocation)

---

### 3. Dados & Formatos
JSON, XML, APIs, extensões de imagem e otimização.

**Categorias:**
- Formatos de dados (JSON, XML, YAML, TOML, CSV, Protocol Buffers)
- Conceitos JSON (estruturas, métodos, JSON Schema, JSON5/JSONC)
- APIs (REST, GraphQL, gRPC, métodos HTTP, status codes, autenticação)
- Formatos de imagem (PNG, JPG, WebP, AVIF, SVG, ICO)
- Otimização de imagens (lazy loading, srcset, CDN, Next.js Image)
- Outros formatos (fontes, vídeo, áudio)
- Data fetching (fetch, Axios, SWR, TanStack Query)
- Serialização (Base64, URL encoding, FormData)

---

### 4. Terminal & CLI
Comandos essenciais do Git, NPM/Yarn e navegação via linha de comando.

**Categorias:**
- Fundamentos (shell, bash, zsh, stdin/stdout/stderr, pipes)
- Navegação e arquivos (cd, ls, mkdir, rm, grep, find)
- Fundamentos Git (working directory, staging, commits, branches)
- Comandos Git (50+ comandos essenciais documentados)
- Workflows Git (GitFlow, Trunk-Based, GitHub Flow, Conventional Commits)
- NPM/Yarn/PNPM (comandos, semver, package.json, workspaces)
- Configuração de ambiente (PATH, dotfiles, nvm/fnm/volta)
- Ferramentas CLI úteis (curl, ssh, tar, fzf, ripgrep, gh)
- Gerenciamento de processos (Ctrl+C/Z, jobs, kill, ps)
- Atalhos de teclado essenciais
