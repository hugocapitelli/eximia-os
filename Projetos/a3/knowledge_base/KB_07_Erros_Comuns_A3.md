# KB_07: Erros Comuns no A3 e Como Evitar

## Citação Importante

> "Mesmo que o A3 às vezes seja desfilado como relíquia sagrada, é apenas uma ferramenta menor. O trabalho principal ainda é identificar e resolver o problema. Se tenho escolha entre uma análise de causa raiz desleixada num A3 e uma boa num envelope usado, fico com o envelope a qualquer momento. Usar um relatório A3 não oferecerá vantagem alguma se o conteúdo for lixo!"
>
> — Michel Baudin

## Erros de Processo

### 1. Não Atualizar Frequentemente

| Problema | Consequência | Como Evitar |
|----------|--------------|-------------|
| A3 fica estático após criação | Não reflete conhecimento atual | Revisar semanalmente durante execução |
| Informações desatualizadas | Confusão e desalinhamento | Definir responsável por atualização |
| Perda de credibilidade | Equipe para de consultar | Incluir "última atualização" visível |

### 2. Falta de Envolvimento da Liderança

| Problema | Consequência | Como Evitar |
|----------|--------------|-------------|
| Sponsor ausente | Sinaliza que problema não é prioridade | Sponsor participa de revisões-chave |
| Liderança não faz coaching | Menos buy-in e input | Definir papel claro do sponsor |
| Decisões travadas | A3 não avança | Escalar bloqueios rapidamente |

### 3. Parar Análise de Causa Raiz Cedo Demais

| Problema | Consequência | Como Evitar |
|----------|--------------|-------------|
| "Operador errou" como causa | Trata sintoma, não causa | Sempre chegar a fator sistêmico |
| "Falta treinamento" como final | Problema vai retornar | Perguntar "por que falta?" |
| 5 Porquês superficial | Contramedidas ineficazes | Validar reversamente |

**Red Flag**: Se seu 5º Porquê responde "pessoa cometeu erro", você NÃO chegou à causa raiz.

### 4. Pular Etapas

| Etapa Pulada | Consequência | Como Evitar |
|--------------|--------------|-------------|
| Condições Atuais | Contramedidas sem base | Exigir mínimo 3 evidências |
| Causa Raiz | Trata sintomas | Não avançar sem Ishikawa completo |
| Follow-up | Quebra ciclo PDCA | Definir ritual de monitoramento |

### 5. Correr para Mostrar "A Resposta"

| Problema | Consequência | Como Evitar |
|----------|--------------|-------------|
| Necessidade de parecer competente | Análise apressada | Cultura de aprendizado |
| Pular para soluções | Perder causas raiz | Completar análise antes |
| Ego do autor | Resistência a feedback | Coaching através de perguntas |

## Erros de Conteúdo

### 6. Priorizar Forma sobre Conteúdo

| Problema | Consequência | Como Evitar |
|----------|--------------|-------------|
| A3 "bonito" com análise fraca | Falsa sensação de qualidade | Conteúdo > Estética |
| Foco em formatação | Tempo perdido | Usar templates simples |
| Gráficos decorativos | Não comunicam | Cada visual deve ter propósito |

### 7. Expandir Além de Uma Página

| Problema | Consequência | Como Evitar |
|----------|--------------|-------------|
| Transbordar para 2+ páginas | História não focada | Refinar, não expandir |
| Incluir tudo | Perda de clareza | Apêndice para detalhes |
| Resistência à síntese | Comunicação ineficaz | Disciplina de uma página |

**Princípio**: Se não cabe em uma página, você não entendeu bem o suficiente.

### 8. Metas Vagas

| VAGO ❌ | ESPECÍFICO ✅ |
|--------|--------------|
| "Melhorar qualidade" | "Reduzir defeitos de 500 para 250 PPM até Jun/26" |
| "Reduzir custos" | "OPEX ≤ 100% do orçamento aprovado" |
| "Aumentar produtividade" | "OEE ≥ 85% em todas as linhas até Dez/26" |

### 9. Contramedidas que são Ações

| AÇÃO (Errado) ❌ | CONTRAMEDIDA (Certo) ✅ |
|-----------------|----------------------|
| Treinar equipe | Implementar trabalho padronizado visual |
| Criar procedimento | Automatizar validação no sistema |
| Aumentar supervisão | Instalar poka-yoke no processo |
| Fazer auditoria | Criar sistema de alerta automático |

### 10. Monitoramento sem Ritual

| Problema | Consequência | Como Evitar |
|----------|--------------|-------------|
| Indicadores sem reunião | Não geram ação | Definir frequência fixa |
| Reunião sem foco | Tempo perdido | Foco em decisão, não explicação |
| Participantes variáveis | Perda de continuidade | Lista fixa de participantes |

## Erros de Mentalidade

### 11. Tratar A3 como Formulário

| Problema | Consequência | Como Evitar |
|----------|--------------|-------------|
| "Preencher os campos" | Exercício burocrático | Entender como processo de pensamento |
| Check-the-box | Sem aprendizado real | Coaching contínuo |
| Copiar de outros A3s | Não resolve problema específico | Sempre do zero para cada problema |

### 12. Culpar Pessoas

| Problema | Consequência | Como Evitar |
|----------|--------------|-------------|
| "Foi erro do João" | Não muda sistema | Perguntar "qual sistema permitiu?" |
| Ambiente de culpa | Pessoas escondem problemas | Cultura de segurança psicológica |
| Foco em indivíduo | Problema retorna | Sempre buscar causa sistêmica |

### 13. Assumir que "Resolveu"

| Problema | Consequência | Como Evitar |
|----------|--------------|-------------|
| Fechar A3 cedo demais | Problema retorna | Monitorar por período definido |
| Não verificar eficácia | Falsa confiança | Check obrigatório no PDCA |
| Parar de aprender | Estagnação | Usar "contramedida", não "solução" |

## Checklist Anti-Erros

Antes de finalizar qualquer A3, verifique:

### Processo
- [ ] A3 foi atualizado na última semana?
- [ ] Sponsor está engajado?
- [ ] Análise de causa chegou a fator sistêmico?
- [ ] Todas as etapas foram completadas?
- [ ] Feedback foi incorporado (Nemawashi)?

### Conteúdo
- [ ] Conteúdo é substantivo (não só forma)?
- [ ] Cabe em uma página?
- [ ] Metas são SMART?
- [ ] Contramedidas atacam causas (não são só ações)?
- [ ] Monitoramento tem ritual definido?

### Mentalidade
- [ ] A3 foi tratado como processo de pensamento?
- [ ] Foco foi no sistema, não em pessoas?
- [ ] Há humildade para evoluir a contramedida?

## Matriz de Gravidade dos Erros

| Gravidade | Erros | Impacto |
|-----------|-------|---------|
| **CRÍTICO** | Parar análise cedo, Culpar pessoas, Metas vagas | Problema não resolvido, retorna pior |
| **ALTO** | Pular etapas, Contramedida=Ação, Sem monitoramento | Desperdício de esforço |
| **MÉDIO** | Não atualizar, Forma > Conteúdo, Expandir páginas | Perda de eficácia |
| **BAIXO** | Falta envolvimento, Correr para resposta | Oportunidade perdida |
