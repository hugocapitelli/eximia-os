# KB_01: Gestão Ágil de Agentes (Agentic Agile)

## Princípios
1. **Entregas incrementais:** Não espere o projeto todo ficar pronto. Peça ao agente um MVP (Minimum Viable Prompt/Product) primeiro.
2. **Feedback Loop Rápido:** Se o Output do Agente X estiver ruim, corrija imediatamente, não passe para o Agente Y.
3. **Contexto é Rei:** Ao delegar, sempre forneça:
   - ROLE (Quem ele é)
   - TASK (O que fazer)
   - CONTEXT (Por que e para quem)
   - CONSTRAINT (Limites)
   - FORMAT (Como entregar)

## Workflow Padrão (The CEO Flow)

1. **Sprint Planning (Fase 1 CEO):** Definir o backlog do que precisa ser feito.
2. **Dispatch (Fase 5 CEO):** Enviar tarefas para os agentes.
3. **Daily/Check (Fase 5 CEO):** Verificar outputs intermediários.
4. **Review (Fase 3 CEO):** Validar se atende aos critérios de aceitação.

## Resolução de Conflitos
- Se Agente A diz "X" e Agente B diz "Y": O CEO decide baseado no objetivo estratégico.
- Se um Agente trava: O CEO deve tentar reformular o pedido ou consultar o "The Recruiter" para criar um agente melhor.
