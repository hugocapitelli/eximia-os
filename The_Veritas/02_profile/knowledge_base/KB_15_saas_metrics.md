## 2. Métricas Chave e Benchmarks 2024

### LTV:CAC Ratio

**Definição:** Lifetime Value / Customer Acquisition Cost

| Benchmark | Contexto | Fonte |
| :---: | :--- | :--- |
| **3:1** | Mínimo saudável | Industry standard |
| **5:1** | Bom | Wundertalent (2024) |
| **6:1** | Média atual | FirstPageSage (2024) |
| **>5:1** | Pode indicar subinvestimento | Klipfolio |

**Nota:** Early-stage pode aceitar 1.5-2x para crescimento rápido.

### CAC (Customer Acquisition Cost)

| Benchmark | Contexto | Fonte |
| :---: | :--- | :--- |
| **$200-600** | Média B2B SaaS | Klipfolio (2024) |
| **$1.78** | Median New Customer CAC Ratio | SaaS.wtf (2024) |
| **$1.00** | Expansion CAC Ratio (2024) | SaaS.wtf |
| **$1.61** | Blended CAC Ratio | SaaS.wtf |

### NRR (Net Revenue Retention)

**Definição:** ARR retido + expansão - churn - downsell

| Benchmark | Contexto | Fonte |
| :---: | :--- | :--- |
| **101%** | Mediana private SaaS | BenchmarkIT (2024) |
| **106%** | Mediana venture-backed | Optif.ai (2024) |
| **110%** | Public SaaS 2023 | BenchmarkIT |
| **>120%** | Best-in-class | Industry |
| **115-125%** | Enterprise típico | Optif.ai |
| **90-105%** | SMB típico | Optif.ai |

**Insight:** NRR ≥100% → crescimento 2x mais rápido (Chartmogul 2024)

### Rule of 40

**Definição:** Growth Rate (%) + Profit Margin (%) ≥ 40%

| Benchmark | Contexto | Fonte |
| :---: | :--- | :--- |
| **34%** | Mediana public SaaS (Aug 2024) | Drivetrain |
| **40%+** | Target saudável | Industry |
| **34** | PLG median | BenchmarkIT |
| **20** | Sales-led median | BenchmarkIT |

## 3. Outras Métricas Importantes

### Burn Multiple

**Definição:** Net Burn / Net New ARR

| Benchmark | Interpretação |
| :---: | :--- |
| **<1x** | Excelente |
| **1-2x** | Bom |
| **>2x** | Atenção |

### Magic Number

**Definição:** Net New ARR / S&M Spend (quarter anterior)

| Benchmark | Interpretação |
| :---: | :--- |
| **>0.75** | Acelerar investimento |
| **0.5-0.75** | Saudável |
| **<0.5** | Otimizar antes de escalar |

### Churn Rate

| Tipo | Benchmark | Contexto |
| :--- | :---: | :--- |
| **Logo Churn** | <5% annual | SMB ok com mais alto |
| **Revenue Churn** | <10% annual | Target |
| **Net Churn** | Negative | Best-in-class |
