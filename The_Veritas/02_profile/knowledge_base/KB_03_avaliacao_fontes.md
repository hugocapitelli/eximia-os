## 1. Por Que Avaliar Fontes?

> *"Na era da desinformação, a avaliação crítica de fontes é a competência mais importante do pesquisador."*
> — Mike Caulfield (SIFT Method, 2017)

### O Problema

| Tipo de Fonte | Risco |
| :--- | :--- |
| SEO farms | Conteúdo gerado para ranking, sem verificação |
| Blogs pessoais | Opinião sem corroboração |
| Press releases | Viés comercial/promocional |
| Redes sociais | Desinformação viral |
| Wikipedia | Editável por qualquer um (mas útil para iniciar) |

## 2. SIFT Method (Recomendado para Web)

### Origem
Criado por Mike Caulfield em 2017 para literacia digital.

### As 4 Etapas

```
┌─────────────────────────────────────────────────────────────┐
│                      SIFT METHOD                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  S — STOP                                                   │
│      Pare antes de ler. Você conhece a fonte?               │
│                                                             │
│  I — INVESTIGATE THE SOURCE                                 │
│      Saia da página. O que outros dizem sobre ela?          │
│                                                             │
│  F — FIND BETTER COVERAGE                                   │
│      Busque outras fontes que cubram o mesmo fato.          │
│                                                             │
│  T — TRACE CLAIMS TO ORIGINAL                               │
│      Rastreie afirmações até a fonte primária.              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**Fonte:** Caulfield, M. (2017). Web Literacy for Student Fact-Checkers.

### Aplicação Prática

**Cenário:** Encontrei um artigo dizendo que "o mercado de AgTech vai crescer 30% em 2025".

| Etapa | Ação | Resultado |
| :--- | :--- | :--- |
| **S** | Parar | Nunca ouvi falar deste site |
| **I** | Investigar | Google: "site X" reputation → SEO farm |
| **F** | Find Better | McKinsey, Gartner dizem 15-20% |
| **T** | Trace | O 30% vem de um press release de startup |

**Conclusão:** Descartar fonte original, usar McKinsey/Gartner.

## 3. CRAAP Test (Recomendado para Acadêmico)

### Origem
Criado por Sarah Blakeslee, CSU Chico Library, 2004.

### Os 5 Critérios

| Critério | Pergunta | Score 1-5 |
| :--- | :--- | :---: |
| **C**urrency | Quando foi publicado? É atual? | __ |
| **R**elevance | É relevante para minha pergunta? | __ |
| **A**uthority | Quem é o autor? É qualificado? | __ |
| **A**ccuracy | É verificável? Tem citações? | __ |
| **P**urpose | Qual a intenção? Informar, vender, persuadir? | __ |
| **TOTAL** | | __ /25 |

**Fonte:** Blakeslee, S. (2004). The CRAAP Test.

### Interpretação

| Score | Avaliação | Ação |
| :---: | :--- | :--- |
| 20-25 | Excelente | Usar como Tier 1-2 |
| 15-19 | Boa | Usar com corroboração |
| 10-14 | Questionável | Buscar fonte melhor |
| <10 | Ruim | Descartar |

### Checklist Detalhado

#### Currency (Atualidade)
- [ ] Data de publicação está visível
- [ ] Informação é atual para o tema (temas dinâmicos: <2 anos)
- [ ] Links funcionam
- [ ] Não há sinais de abandono

#### Relevance (Relevância)
- [ ] Responde minha pergunta de pesquisa
- [ ] Nível de profundidade adequado
- [ ] Público-alvo compatível

#### Authority (Autoridade)
- [ ] Autor é identificável
- [ ] Autor tem credenciais verificáveis
- [ ] Organização é reconhecida
- [ ] Contato disponível

#### Accuracy (Precisão)
- [ ] Afirmações têm evidência/citação
- [ ] Pode ser verificado em outras fontes
- [ ] Sem erros factuais óbvios
- [ ] Linguagem neutra (sem sensacionalismo)

#### Purpose (Propósito)
- [ ] Intenção é clara (informar, não vender)
- [ ] Viés é explicitado se existir
- [ ] Não há conflito de interesse oculto
- [ ] Não é clickbait

## 4. RADAR Framework

### Origem
Mandalios, J. (2013). Cinco critérios para avaliação de website.

### Os 5 Critérios

| Critério | Foco |
| :--- | :--- |
| **R**ationale | Por que este recurso existe? |
| **A**uthority | Quem criou? Qual expertise? |
| **D**ate | Quando foi criado/atualizado? |
| **A**ccuracy | Quão preciso e verificável? |
| **R**elevance | Quão útil para minha necessidade? |

## 5. Source Tier System (The_Veritas)

### Classificação por Autoridade

```
TIER 1: MÁXIMA AUTORIDADE
├── Governamental: NASA, USDA, CVM, SEC, IBGE
├── Acadêmico: PubMed, IEEE, Nature, Science
└── Internacional: World Bank, IMF, FAO, OECD

TIER 2: ALTA AUTORIDADE
├── Consultoria: McKinsey, BCG, Deloitte, EY
├── Mídia premium: Bloomberg, Reuters, WSJ, FT
└── Indústria: Gartner, Forrester, a16z

TIER 3: MÉDIA AUTORIDADE
├── Mídia nacional: Valor, Exame, InfoMoney
├── Associações: CNA, Abiove, Abranet
└── Empresas: Relatórios anuais, investor relations

TIER 4: BAIXA AUTORIDADE
├── Blogs: Apenas se autor verificável
├── Wikipedia: Apenas para contexto inicial
└── Social: Apenas para citações diretas
```

### Regra de Uso

| Tier | Uso Permitido |
| :--- | :--- |
| Tier 1 | Pode ser fonte única |
| Tier 2 | Requer 1 corroboração |
| Tier 3 | Requer 2 corroborações |
| Tier 4 | Apenas citação, nunca como fato |

## 6. Lateral Reading

### Conceito

> *"Não leia verticalmente (dentro do site). Leia lateralmente (o que outros dizem sobre ele)."*
> — Stanford History Education Group

### Processo

```
ENCONTRA ARTIGO NO SITE "AgTech Weekly"
       │
       ▼
ABRE NOVA ABA: "AgTech Weekly" reputation
       │
       ▼
VERIFICA:
├── Wikipedia: Não existe entrada
├── MediaBias: Não listado
├── LinkedIn: Autor não encontrado
└── Whois: Domínio registrado há 6 meses
       │
       ▼
CONCLUSÃO: Fonte não verificável, Tier 4
```

## 7. Red Flags (Sinais de Alerta)

### Descarte Imediato Se:

| Red Flag | Exemplo |
| :--- | :--- |
| Autor anônimo | "Equipe Editorial" sem nomes |
| Sem data | Artigo sem timestamp |
| Sem citações | Afirmações sem fontes |
| Clickbait | "VOCÊ NÃO VAI ACREDITAR!" |
| Erros gramaticais | Sinais de conteúdo gerado |
| Domínio suspeito | .xyz, .info, subdomínios estranhos |
| Popup excessivo | Sinais de SEO farm |
