# KB_07: ROI de Treinamento — Phillips Methodology

## Visão Geral

A metodologia **Phillips ROI** expande o modelo Kirkpatrick adicionando um 5º nível para calcular o Retorno sobre Investimento financeiro de programas de treinamento.

---

## Os 5 Níveis Phillips

```
┌─────────────────────────────────────────────────────────────────┐
│                    PHILLIPS ROI - 5 NÍVEIS                       │
│                                                                  │
│   NÍVEL 5: ROI                              ← NOVO              │
│   "Qual o retorno financeiro?"                                  │
│   └── Benefícios monetários vs Custos                           │
│                     ▲                                            │
│   NÍVEL 4: IMPACTO NO NEGÓCIO                                   │
│   "Qual a mudança em métricas de negócio?"                      │
│   └── + Isolamento do efeito do treinamento                     │
│                     ▲                                            │
│   NÍVEL 3: APLICAÇÃO                                            │
│   "Estão usando no trabalho?"                                   │
│   └── + Barreiras e facilitadores                               │
│                     ▲                                            │
│   NÍVEL 2: APRENDIZAGEM                                         │
│   "O que aprenderam?"                                           │
│                     ▲                                            │

│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

[Phillips, J.J., Return on Investment in Training, 1996]

## Fórmula do ROI

### Cálculo Básico

```
ROI (%) = (Benefícios Líquidos - Custos do Programa) × 100
           ──────────────────────────────────────────
                    Custos do Programa

         = (Benefícios - Custos) × 100
           ────────────────────────
                  Custos
```

### Exemplo Prático

| Item | Valor |
|------|-------|
| Benefício monetário anual | R$ 500.000 |
| Custo total do programa | R$ 100.000 |
| Benefício líquido | R$ 400.000 |
| **ROI** | **400%** |

> *"Para cada R$1 investido, retornou R$4 adicionais."*

---

## Processo de Cálculo

### Passo 1: Coletar Dados de Nível 4

Identificar métricas impactadas:
- Produtividade
- Qualidade
- Custos
- Tempo
- Satisfação do cliente

### Passo 2: Isolar Efeitos do Treinamento

**Técnicas de Isolamento:**

| Técnica | Descrição | Precisão |
|---------|-----------|----------|
| Grupo controle | Comparar treinados vs não-treinados | Alta |
| Análise de tendência | Projetar linha base vs realizado | Média |
| Estimativa de participantes | Perguntar % atribuído ao treino | Média |
| Estimativa de gestores | Gestores estimam impacto | Média |
| Fatores independentes | Controlar outras variáveis | Alta |

### Passo 3: Converter para Valores Monetários

| Métrica | Conversão |
|---------|-----------|
| Tempo economizado | Horas × Custo/hora |
| Erros reduzidos | Custo médio por erro × Redução |
| Vendas aumentadas | Receita adicional ou margem |
| Turnover reduzido | Custo de reposição × Redução |

### Passo 4: Calcular Custos Totais

**Categorias de Custo:**
- Desenvolvimento (design, materiais)
- Facilitação (instrutores, viagens)
- Participantes (salários durante training)
- Infraestrutura (salas, tecnologia)
- Avaliação
- Overhead

### Passo 5: Calcular ROI e BCR

```
BCR (Benefit-Cost Ratio) = Benefícios Totais
                           ─────────────────
                           Custos Totais

Se BCR = 3:1, significa que para cada R$1 gasto, retornou R$3
```

## Benefícios Intangíveis

### O que não converter em dinheiro

Alguns benefícios são difíceis de monetizar, mas ainda são valiosos:

| Intangível | Por que importa |
|------------|-----------------|
| Engajamento | Correlaciona com produtividade |
| Clima organizacional | Impacta retenção |
| Imagem da empresa | Atração de talentos |
| Satisfação do cliente | Impacto de longo prazo |
| Inovação | Difícil de quantificar imediatamente |

**Regra:** Não forçar conversão. Listar como benefícios adicionais.

## Níveis de Avaliação: Quando usar

| Situação | Níveis recomendados |
|----------|---------------------|
| Treinamento de baixo custo | 1, 2 |
| Treinamento de médio custo | 1, 2, 3 |
| Programa estratégico | 1, 2, 3, 4 |
| Alto investimento (>R$100K) | 1, 2, 3, 4, 5 |
| Piloto para escalar | 1, 2, 3, 4, 5 |

## Benchmarks de ROI

### O que esperar

| Tipo de Programa | ROI Típico |
|------------------|------------|
| Soft skills | 100-200% |
| Técnico/processos | 200-400% |
| Vendas | 300-700% |
| Liderança | 200-500% |
| Safety/compliance | 500-1000% |

**Nota:** ROI médio de programas bem avaliados: 150-200%

[Phillips & Phillips, Show Me the Money, 2007]
