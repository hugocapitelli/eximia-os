## Análise Geral do Carrossel

### Contexto de Uso

Este carrossel será consumido em **ambiente de scroll rápido** (redes sociais). O usuário tem:
- **Atenção fragmentada** (múltiplos estímulos competindo)
- **Tempo limitado** (decisão de parar ou continuar em < 2 segundos)
- **Expectativa de valor** (troca atenção por insight)

### O Design Deve:

1. **Parar o scroll** (Visceral)
2. **Recompensar a leitura** (Behavioral)
3. **Criar identificação** (Reflective)

### SLIDE 1 — O HOOK

**Copy:**
```
"Dê-me uma alavanca e moverei o mundo."
— Arquimedes

E se a IA fosse a sua?
```

#### Princípios Aplicados

| Princípio | Análise | Recomendação |
|-----------|---------|--------------|
| **Discoverability** | ✅ Usuário descobre imediatamente: é uma citação + pergunta | Manter hierarquia clara |
| **Signifiers** | ⚠️ A pergunta final precisa de destaque visual para sinalizar "isto é para você" | Usar cor/tamanho diferente para "E se a IA fosse a sua?" |
| **Mapping** | ✅ Fluxo de leitura natural: citação → autor → pergunta | — |
| **Conceptual Model** | ✅ Metáfora da alavanca é universalmente compreensível | — |

#### Design Emocional

| Nível | Análise | Nota |
|-------|---------|------|
| **Visceral** | Citação icônica cria autoridade imediata | 9/10 |
| **Behavioral** | Leitura fluida, sem obstáculos cognitivos | 9/10 |
| **Reflective** | "E se a IA fosse a sua?" provoca reflexão pessoal | 10/10 |

#### Recomendações Específicas

1. **Hierarquia tipográfica:**
   - Citação: 60% do slide, fonte serif para elegância histórica
   - Atribuição: 15%, discreta
   - Pergunta: 25%, sans-serif bold, cor de destaque

2. **Contraste:**
   - Fundo escuro para autoridade
   - Texto claro para legibilidade
   - Pergunta em cor de marca para diferenciação

3. **Espaço negativo:**
   - Mínimo 15% de margem em cada lado
   - Respiração entre citação e pergunta

4. **Evitar:**
   - Imagens de fundo que competem com texto
   - Mais de 2 famílias tipográficas
   - Decorações desnecessárias

### SLIDE 2 — A VIRADA

**Copy:**
```
A IA não veio para substituir você.

Veio para amplificar.

Sua clareza.
Sua decisão.
Seu tempo para o que importa.
```

#### Princípios Aplicados

| Princípio | Análise | Recomendação |
|-----------|---------|--------------|
| **Feedback** | ✅ Cada linha reforça a anterior (feedback conceitual) | — |
| **Constraints** | ⚠️ Lista de 3 pode virar "parede de texto" se mal formatada | Espaçamento vertical generoso |
| **Signifiers** | ⚠️ "Amplificar" é a palavra-chave, precisa de destaque máximo | Tamanho maior, cor diferente |
| **Affordances** | ✅ Estrutura convida à leitura sequencial | — |

#### Design Emocional

| Nível | Análise | Nota |
|-------|---------|------|
| **Visceral** | Contraste "substituir/amplificar" cria tensão visual interessante | 8/10 |
| **Behavioral** | Ritmo de leitura agradável com lista de 3 | 9/10 |
| **Reflective** | Resolve o medo profundo de ser substituído | 10/10 |

#### Recomendações Específicas

1. **Técnica de contraste:**
   - "Não veio para substituir" em cor neutra/apagada
   - "Veio para AMPLIFICAR" em cor vibrante, tamanho 150%
   - Considerar riscar "substituir" visualmente (efeito strikethrough sutil)

2. **Lista de 3:**
   - Alinhamento à esquerda (mais natural para leitura)
   - Espaçamento vertical de 1.5-2x entre itens
   - Ícones opcionais (✓ ou •) para ritmo visual

3. **Gulf of Evaluation:**
   - O usuário deve sair sabendo: "exímIA amplifica, não substitui"
   - Este é o take-away principal do carrossel

### SLIDE 3 — O MANIFESTO

**Copy:**
```
A primeira Revolução libertou músculos.
A segunda, liberta mentes.

Quando Leonardo teve tempo para pensar,
o mundo mudou.

Nós criamos esse tempo.
Para você.
```

#### Princípios Aplicados

| Princípio | Análise | Recomendação |
|-----------|---------|--------------|
| **Conceptual Model** | ✅ Metáfora Revolução Industrial → IA é poderosa e compreensível | — |
| **Mapping** | ✅ Músculos → Mentes cria paralelo visual claro | Usar mesma estrutura tipográfica |
| **Signifiers** | ⚠️ "Para você" precisa de destaque como fechamento pessoal | Isolar visualmente, cor de destaque |
| **Constraints** | ⚠️ Slide mais denso, risco de sobrecarga | Dividir em 3 blocos visuais distintos |

#### Design Emocional

| Nível | Análise | Nota |
|-------|---------|------|
| **Visceral** | Referência a Leonardo/Renascimento evoca grandeza | 9/10 |
| **Behavioral** | Mais texto = exige mais commitment | 7/10 |
| **Reflective** | "Para você" personaliza a promessa | 10/10 |

#### Recomendações Específicas

1. **Divisão em 3 blocos:**
   ```
   BLOCO 1 (40%): Revolução/Músculos/Mentes
   BLOCO 2 (35%): Leonardo/Mundo mudou
   BLOCO 3 (25%): "Nós criamos... Para você."
   ```

2. **Contraste músculos/mentes:**
   - "Músculos" em cor mais fria/neutra
   - "Mentes" em cor mais quente/destaque
   - Mesmo tamanho para manter paralelo

3. **Referência a Leonardo:**
   - Pode usar itálico para tom reflexivo
   - NÃO usar imagem do Vitruviano (clichê)
   - Deixar a referência textual trabalhar

4. **"Para você" — O fechamento:**
   - Separação visual clara (linha, espaço ou cor)
   - Fonte menor mas bold
   - Cor primária da marca

### SLIDE 4 — CTA + ASSINATURA

**Copy:**
```
exímIA

Inteligência que amplifica a sua.

→ Siga para ver o futuro que já começou.
```

#### Princípios Aplicados

| Princípio | Análise | Recomendação |
|-----------|---------|--------------|
| **Discoverability** | ✅ CTA claro com seta indicando ação | — |
| **Signifiers** | ✅ Seta (→) é signifier universal de "próximo passo" | — |
| **Affordances** | ⚠️ Em carrossel, ação é seguir, não clicar | Texto do CTA deve ser "Siga" não "Clique" |
| **Feedback** | ⚠️ Usuário não sabe o que ganha ao seguir | Adicionar valor implícito ("ver o futuro") |

#### Design Emocional

| Nível | Análise | Nota |
|-------|---------|------|
| **Visceral** | Logo como elemento hero = identidade forte | 9/10 |
| **Behavioral** | CTA simples e direto | 9/10 |
| **Reflective** | "Futuro que já começou" = aspiracional | 9/10 |

#### Recomendações Específicas

1. **Hierarquia:**
   ```
   LOGO: 40% do slide, centro-superior
   TAGLINE: 30%, centro
   CTA: 20%, inferior
   ESPAÇO: 10% margens
   ```

2. **Logo como elemento principal:**
   - Tamanho hero (dominante)
   - Se tiver símbolo, pode usar só símbolo grande
   - Cor sólida, sem gradientes complexos

3. **Tagline:**
   - Tracking (espaçamento entre letras) amplo para elegância
   - Mesmo peso visual que headlines anteriores

4. **CTA:**
   - Seta (→) antes do texto
   - Cor diferente das demais linhas (mais suave ou mais vibrante)
   - Não parecer "botão" — é convite, não ordem

## Consistência Visual (Cross-Slide)

### Princípio de Continuidade

> *"Bom design cria um modelo conceitual coerente. O usuário deve entender que está em uma experiência única, não em 4 peças separadas."*

| Elemento | Recomendação |
|----------|--------------|
| **Fundo** | Consistente OU gradiente sutil entre slides |
| **Tipografia** | Máximo 2 famílias (serif para citações, sans para corpo) |
| **Cores** | Paleta limitada (3-4 cores no total) |
| **Margens** | Idênticas em todos os slides |
| **Posição do texto** | Mesma zona visual (centro ou centro-esquerdo) |

### Transições

Para plataformas que suportam animação (Stories, Reels):
- **Fade suave** entre slides (não cortes bruscos)
- **Texto pode entrar linha a linha** (revelação progressiva)
- **Evitar** rotações, zooms excessivos, efeitos datados

## Nota Final

> *"A pergunta não é se o design é bom ou ruim. A pergunta é: ele serve as pessoas que vão usá-lo?"*

Este carrossel serve seu público porque:

1. **Para o scroll** com autoridade (Arquimedes)
2. **Resolve o medo** de substituição (amplifica, não substitui)
3. **Eleva o espírito** com referência histórica (Leonardo, Revolução)
4. **Convida sem pressionar** (CTA suave)

O código **AMPLIFICAÇÃO** está presente em cada slide. O design deve amplificar essa mensagem, não competir com ela.
