# KB_03: Templates de Cenarios para Perguntas Socraticas

## Por que Usar Cenarios?

Cenarios praticos:
- Tornam perguntas mais engajantes e relevantes
- Exigem aplicacao de conhecimento, nao apenas recall
- Revelam compreensao genuina vs superficial
- Preparam o aluno para situacoes reais

## Template 1: O Consultor

### Estrutura
```
Imagine que voce e um consultor de [AREA].
Um cliente te procura com o seguinte problema: [PROBLEMA].
[CONTEXTO ADICIONAL - restricoes, recursos, stakeholders]
[PERGUNTA que pede analise/recomendacao]
```

### Exemplo
```
Imagine que voce e um consultor agricola.
Um pequeno produtor te procura dizendo que quer "modernizar" sua fazenda, mas tem orcamento limitado de R$50 mil.
Ele pode investir em sensores IoT para monitoramento OU em capacitacao da equipe, mas nao em ambos.
Que perguntas voce faria a ele antes de recomendar uma das opcoes?
```

## Template 2: O Dilema

### Estrutura
```
[STAKEHOLDER] enfrenta um dilema:
Por um lado, [OPCAO A com beneficios].
Por outro, [OPCAO B com beneficios diferentes].
[CONTEXTO que complica a decisao]
Que criterios voce usaria para ajuda-lo a decidir?
```

### Exemplo
```
Um gestor de cooperativa enfrenta um dilema:
Por um lado, pode manter fornecedores locais (mais caro, mas fortalece a comunidade).
Por outro, pode trocar por fornecedores de fora (mais barato, mas enfraquece vinculos locais).
A cooperativa tem margem apertada mas se posiciona como "socialmente responsavel".
Que criterios voce usaria para ajuda-lo a decidir?
```

### Skill: Avaliacao/Reflexao

---

## Template 3: O Contraditorio

### Estrutura
```
[AFIRMACAO comum ou do texto].
Porem, [CONTRA-ARGUMENTO ou situacao onde nao se aplica].
Como voce conciliaria essas duas perspectivas?
OU
Em que situacoes [afirmacao] seria verdade e em quais nao?
```

### Exemplo
```
E comum ouvir que "tecnologia sempre aumenta produtividade".
Porem, algumas fazendas que investiram pesado em tecnologia nao viram retorno significativo.
O que diferenciaria uma adocao de tecnologia bem-sucedida de uma fracassada?
```

### Skill: Analise

## Template 4: O Stakeholder Multiplo

### Estrutura
```
[SITUACAO que afeta diferentes grupos].
[STAKEHOLDER A] veria isso como [PERSPECTIVA A].
[STAKEHOLDER B] veria isso como [PERSPECTIVA B].
[STAKEHOLDER C] veria isso como [PERSPECTIVA C].
Como voce encontraria uma solucao que [CRITERIO]?
```

### Exemplo
```
Uma nova regulamentacao ambiental vai restringir o uso de agua em uma regiao agricola.
Produtores veem isso como ameaca a sobrevivencia economica.
Ambientalistas veem como medida insuficiente.
O governo local quer evitar conflito e perda de empregos.
Se voce mediasse essa situacao, que solucao proporia que atendesse ao maximo de interesses?
```

## Template 5: O E Se...?

### Estrutura
```
[CONTEXTO baseado no conteudo].
E se [VARIAVEL] fosse diferente: [NOVO VALOR]?
Como isso mudaria [RESULTADO/ESTRATEGIA/ANALISE]?
```

### Exemplo
```
O texto discute estrategias de gestao para fazendas de medio porte.
E se o contexto fosse uma agricultura familiar com menos de 5 hectares?
Quais estrategias ainda se aplicariam e quais precisariam ser completamente repensadas?
```

### Skill: Analise/Aplicacao

---

## Template 6: O Critico

### Estrutura
```
[POSICAO ou recomendacao do texto].
Se voce fosse argumentar CONTRA essa posicao, que pontos levantaria?
Que evidencias enfraqueceriam esse argumento?
```

### Exemplo
```
O texto recomenda que produtores rurais diversifiquem suas culturas para reduzir riscos.
Se voce fosse argumentar CONTRA a diversificacao, que pontos levantaria?
Em que situacoes a especializacao poderia ser mais vantajosa?
```

## Template 7: O Designer

### Estrutura
```
Voce foi contratado para [OBJETIVO].
Considerando [RESTRICOES], [RECURSOS], [CONTEXTO].
Que [SOLUCAO/ESTRATEGIA/PLANO] voce proporia?
Explique suas escolhas.
```

### Exemplo
```
Voce foi contratado para criar um programa de capacitacao em gestao para pequenos produtores rurais.
Considerando que eles tem pouca disponibilidade de tempo, baixa conectividade e resistencia a mudancas.
Que abordagem voce proporia para o programa?
Explique por que essa abordagem seria mais eficaz que um curso tradicional.
```

### Skill: Criar/Sintese

---

## Template 8: O Investigador

### Estrutura
```
[ORGANIZACAO/PESSOA] enfrenta [PROBLEMA/RESULTADO NEGATIVO].
Ja tentaram [SOLUCOES QUE NAO FUNCIONARAM].
Se voce fosse investigar a causa raiz, por onde comecaria?
Que hipoteses testaria?
```

### Exemplo
```
Uma fazenda de soja esta com produtividade 20% abaixo da media regional.
Ja trocaram de semente, ajustaram irrigacao e aplicaram mais fertilizante, sem resultado.
Se voce fosse investigar a causa raiz, por onde comecaria?
Que hipoteses voce levantaria alem das obvias?
```

### Skill: Analise

## Regras para Uso de Cenarios

### FAZER:
- Usar numeros especificos (R$50 mil, 20%, 5 hectares)
- Incluir restricoes realistas (tempo, dinheiro, conhecimento)
- Nomear stakeholders concretos (produtor, gestor, consultor)
- Deixar espaco para multiplas solucoes validas
- Conectar ao conteudo do capitulo

### NAO FAZER:
- Criar cenarios com resposta unica "certa"
- Usar situacoes muito fantasiosas ou impossiveis
- Incluir detalhes irrelevantes que confundem
- Criar cenarios que nao tem relacao com o conteudo
- Pedir apenas descricao do cenario (deve pedir acao)

## Distribuicao Recomendada

Para 3 perguntas, usar pelo menos 2 com cenarios:
- Pergunta 1: Com cenario (Template 1, 2 ou 7)
- Pergunta 2: Com cenario (Template 3, 4 ou 5)
- Pergunta 3: Pode ser mais conceitual (Template 6 ou analise direta)
