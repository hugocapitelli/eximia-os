# KB_03: Preservacao de Intencao Pedagogica

## Principio Fundamental

O Editor NUNCA deve mudar o significado do que o ORIENTADOR quis comunicar. A edicao e sobre FORMA, nao CONTEUDO.

---

## O que PRESERVAR

### 1. Pontos de Feedback
Se o ORIENTADOR reconheceu algo especifico na resposta do aluno, isso deve permanecer.

**Original:**
```
Voce identificou corretamente a relacao entre clima e producao agricola.
```

**Editado (CORRETO):**
```
Voce identificou corretamente a relacao entre clima e producao agricola, o que demonstra boa compreensao do tema.
```

**Editado (ERRADO):**
```
Sua resposta foi interessante. (PERDEU a especificidade)
```

---

### 2. Nuances e Ressalvas
Se o ORIENTADOR adicionou uma nuance ou ressalva, ela deve permanecer.

**Original:**
```
Sua observacao e valida, embora existam casos em que isso nao se aplica.
```

**Editado (CORRETO):**
```
Sua observacao e valida e toca em um ponto importante, embora existam casos em que essa regra nao se aplica diretamente.
```

**Editado (ERRADO):**
```
Sua observacao e valida. (PERDEU a ressalva)
```

---

### 3. A Pergunta Especifica
A pergunta deve ser a mesma em essencia. Pode ser levemente refraseada para fluidez, mas o FOCO deve ser identico.

**Original:**
```
Como voce aplicaria esse conceito em uma fazenda de pequeno porte?
```

**Editado (CORRETO):**
```
Como voce aplicaria esse conceito no contexto de uma fazenda de pequeno porte?
```

**Editado (ERRADO):**
```
O que voce acha sobre fazendas pequenas? (MUDOU o foco da pergunta)
```

---

### 4. Tom e Postura
Se o ORIENTADOR foi mais provocativo ou mais acolhedor, manter.

**Original (provocativo):**
```
Voce parece assumir que tecnologia sempre e a solucao. Mas e se nao fosse?
```

**Editado (CORRETO):**
```
Voce parece assumir que tecnologia sempre e a solucao, o que e uma visao comum. Mas e se nao fosse o caso em todas as situacoes?
```

**Editado (ERRADO):**
```
Interessante ponto sobre tecnologia. O que mais voce considera? (PERDEU o tom provocativo)
```

---

## O que PODE Mudar

### 1. Palavras Excessivas
Remover palavras que nao adicionam significado.

**Original:**
```
E muito importante ressaltar que sua resposta demonstra uma compreensao bastante adequada do tema em questao.
```

**Editado:**
```
Sua resposta demonstra compreensao adequada do tema.
```

---

### 2. Estruturas Roboticas
Substituir por linguagem natural.

**Original:**
```
Em primeiro lugar, cabe destacar que voce abordou o tema. Em segundo lugar, e necessario pontuar que...
```

**Editado:**
```
Voce abordou o tema de forma relevante. Alem disso...
```

---

### 3. Repeticoes
Eliminar redundancias.

**Original:**
```
A gestao de riscos e importante. De fato, a importancia da gestao de riscos nao pode ser subestimada.
```

**Editado:**
```
A gestao de riscos e fundamental e sua importancia nao pode ser subestimada.
```

---

## Teste de Preservacao

Antes de entregar, responda:

| Pergunta | Deve ser SIM |
|----------|--------------|
| O feedback reconhece os mesmos pontos? | SIM |
| As nuances/ressalvas estao presentes? | SIM |
| A pergunta tem o mesmo foco? | SIM |
| O tom geral e similar? | SIM |
| Um aluno entenderia a mesma mensagem? | SIM |

Se qualquer resposta for NAO, revisar a edicao.

---

## Hierarquia de Prioridades

Quando houver conflito entre objetivos:

1. **PRESERVAR SIGNIFICADO** (mais importante)
2. Manter estrutura de 2 paragrafos
3. Remover rotulos
4. Melhorar fluidez
5. **Impor estilo proprio** (menos importante - EVITAR)

---

## Exemplos de Preservacao

### Exemplo 1: Preservando Especificidade

**Input do ORIENTADOR:**
```
[Feedback]
Voce menciona os tres pilares da sustentabilidade, mas foca apenas no ambiental. Os pilares economico e social tambem sao cruciais.

[Pergunta]
Como os tres pilares se equilibram em uma decisao real de uma empresa agricola?
```

**Output do Editor (CORRETO):**
```
Voce menciona os tres pilares da sustentabilidade, o que mostra familiaridade com o conceito. Porem, seu foco no pilar ambiental deixa de lado os pilares economico e social, que sao igualmente cruciais.

Como os tres pilares se equilibram em uma decisao real de uma empresa agricola?
```

### Exemplo 2: Preservando Provocacao

**Input do ORIENTADOR:**
```
[Feedback]
Interessante, mas voce parece confiar muito na tecnologia como solucao universal.

[Provocacao]
E se a tecnologia criasse mais problemas do que resolvesse em certos contextos?
```

**Output do Editor (CORRETO):**
```
Interessante observacao, embora voce parececa confiar bastante na tecnologia como solucao universal. Nem sempre essa premissa se sustenta em todos os contextos.

E se a tecnologia criasse mais problemas do que resolvesse em certas situacoes?
```
