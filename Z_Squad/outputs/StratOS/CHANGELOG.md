# Changelog - StratOS

## [1.2.0] - 2026-01-11
### Added
- **System Rules V2 (Pack):**
    - **Rule 10 (SWOT):** Drivers must be explicitly derived from SWOT finding ("Com base na SWOT...").
    - **Rule 11 (Driver Model):** Enforced rich description format (Intention + Scope).
    - **Rule 12 (Logic Flow):** Changed hierarchy to `Dimension -> Driver -> Initiative -> Meta`.
    - **Rule 13 (Manual Entry):** Initiatives are never auto-created; user writes, agent refines.
    - **Rule 14 (Tactical/Operational):** Strict separation and ordering (Tactical first).
    - **Rule 15 (User defined quantity):** Loop for initiative inputs.
    - **Rule 16 (Metas Last):** Metas are created at the end to measure initiative success.
    - **Rule 17 (Agent Persona):** Updated motto to emphasize user decision/ownership.

## [1.1.0] - 2026-01-11
### Changed
- **Mental DNA**: Added "Provocateur" archetype for strategic questioning.
- **Workflow**:
    - Added **Step 0: Strategic Context (SWOT)**.
    - Added strict breakdown for **Step 1: North Star** (Vision -> Purpose -> Values).
    - Refined **Step 2: Strategic Dimensions** to adopt strict Balanced Scorecard logic (Finance -> Customer -> Processes -> Learning).
    - Refined **Step 6: Initiatives** to distinguish between Tactical (Capabilities) and Operational (Routines) initiatives.
    - Updated terminology: Changed "KPIs" to "Metas (Targets)" and "Objectives" to "Dimensions".
- **Rules**: Added "Iron Laws" regarding BSC hierarchy and Target vs. Effort distinction.
