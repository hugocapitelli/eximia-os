## As 12 Fórmulas Essenciais

### 1. AIDA (Attention-Interest-Desire-Action)
**A mais clássica de todas.**

```
A — ATTENTION: Headline que para o scroll
    "Você está perdendo dinheiro toda semana sem saber"

I — INTEREST: Desenvolva o problema
    "90% dos empreendedores cometem esse erro"

D — DESIRE: Mostre a solução/benefício
    "Imagina economizar R$500/mês com uma mudança simples"

A — ACTION: CTA claro
    "Salve esse post e aplique hoje"
```

### 2. PAS (Problem-Agitate-Solve)
**Para quando a dor é o driver principal.**

```
P — PROBLEM: Identifique a dor
    "Você trabalha 12h por dia mas não vê resultado"

A — AGITATE: Amplifique
    "Enquanto isso, seu concorrente trabalha menos e fatura mais"

S — SOLVE: Apresente a solução
    "O segredo está na priorização..."
```

### 3. BAB (Before-After-Bridge)
**Para transformações claras.**

```
BEFORE: Estado atual (dor)
    "Antes: Acordando cansado, sem energia"

AFTER: Estado desejado (prazer)
    "Depois: Acordando com disposição, produtivo"

BRIDGE: Como ir de um ao outro
    "O que mudou: Uma rotina matinal de 20 minutos"
```

### 4. 4 U's (Useful-Urgent-Unique-Ultra-specific)
**Checklist para headlines.**

```
USEFUL: Resolve problema real
URGENT: Razão para agir agora
UNIQUE: Diferente do comum
ULTRA-SPECIFIC: Números e detalhes

Exemplo:
"7 técnicas [ÚTIL] pouco conhecidas [ÚNICO]
que podem dobrar suas vendas [ESPECÍFICO]
essa semana [URGENTE]"
```

### 5. FAB (Features-Advantages-Benefits)
**Para converter features em copy persuasivo.**

```
FEATURE: O que é
    "10GB de armazenamento"

ADVANTAGE: O que faz de diferente
    "5x mais que a concorrência"

BENEFIT: O que significa para você
    "Nunca mais se preocupe com espaço"

→ No copy, lidere com BENEFIT
```

### 6. The 1-2-3-4 Formula
**Para listas e listicles.**

```
1 — O que eu tenho para você
2 — O que isso vai fazer por você
3 — Quem sou eu
4 — O que você precisa fazer agora
```

### 7. Star-Chain-Hook
**Para storytelling + conversão.**

```
STAR: Apresente o protagonista/problema
    "Maria estava falida..."

CHAIN: Sequência de eventos/fatos
    "Primeiro ela tentou X, depois Y..."

HOOK: O gancho que prende
    "Até descobrir o método que mudou tudo"
```

### 8. QUEST (Qualify-Understand-Educate-Stimulate-Transition)
**Para carrosséis educacionais.**

```
Q — QUALIFY: Isso é para você?
U — UNDERSTAND: Mostrar que entende a dor
E — EDUCATE: Ensinar algo valioso
S — STIMULATE: Criar desejo por mais
T — TRANSITION: Levar para próximo passo
```

### 9. The 5 Basic Objections
**Antecipe e resolva objeções.**

```
1. Não tenho tempo → "Em apenas 5 minutos..."
2. Não tenho dinheiro → "Grátis / Sem investimento..."
3. Não vai funcionar → "Funcionou para 10.000 pessoas..."
4. Não acredito em você → "Comprovado por / Usado por..."
5. Não preciso agora → "Cada dia que passa..."
```

### 10. The So-What Test
**Valide cada frase.**

```
AFIRMAÇÃO: "Nosso produto tem 10GB de espaço"
SO WHAT: E daí?
RESPOSTA: "Então você nunca mais vai perder uma foto importante"

Sempre responda o "so what" antes de publicar
```

### 11. The Rule of One
**Uma ideia, um leitor, uma promessa.**

```
ONE READER: Escreva para uma pessoa específica
ONE IDEA: Um conceito por slide
ONE PROMISE: Uma transformação principal
ONE CTA: Uma ação desejada
```

### 12. Empathy First
**Comece sempre pela perspectiva do leitor.**

```
❌ ERRADO: "Nós criamos um produto incrível"
✅ CERTO: "Você já se sentiu assim..."

❌ ERRADO: "Nossa metodologia é revolucionária"
✅ CERTO: "Imagine se você pudesse..."
```

## Headlines por Categoria

### Curiosidade
- "O que [experts] não querem que você saiba sobre..."
- "A verdade sobre [tema] que ninguém fala"
- "Por que [coisa comum] está completamente errada"

### Medo/Urgência
- "Você está cometendo esse erro agora?"
- "O que acontece se você não [ação]"
- "[N] sinais de que você precisa agir agora"

### Benefício Direto
- "Como [resultado] em [tempo]"
- "[N] formas de [benefício] sem [dor]"
- "O método de [tempo] para [resultado]"

### Social Proof
- "Como [pessoa] fez [resultado]"
- "O segredo de [grupo de sucesso]"
- "[N] pessoas já fizeram isso"

### Contrário
- "Por que você deveria PARAR de [consenso]"
- "[Coisa positiva] está te prejudicando"
- "Esquece tudo que você sabe sobre [tema]"
