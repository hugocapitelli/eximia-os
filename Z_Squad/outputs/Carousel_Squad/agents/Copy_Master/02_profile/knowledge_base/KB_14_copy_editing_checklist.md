# KB_14 — Copy Editing Checklist

**Agente:** Copy Master
**Categoria:** REVISAO
**Versao:** 1.0

---

## O Poder da Edicao

> "O primeiro rascunho de qualquer coisa e uma porcaria." — Ernest Hemingway

Escrever e reescrever. Os melhores copywriters gastam mais tempo editando do que escrevendo o primeiro rascunho.

---

## O Processo de Edicao em 3 Passos

### PASSO 1: MACRO EDIT (Estrutura)
Olhe para o TODO. A estrutura faz sentido?

### PASSO 2: MICRO EDIT (Frases)
Cada frase serve um proposito?

### PASSO 3: POLISH (Palavras)
Cada palavra e necessaria?

---

## CHECKLIST COMPLETO DE EDICAO

### NIVEL 1: ESTRATEGIA

```
[ ] OBJETIVO CLARO
    Qual a UNICA acao que quero que o leitor tome?
    Se houver mais de uma, priorize.

[ ] PUBLICO DEFINIDO
    Para quem estou escrevendo?
    Essa pessoa se reconhece no texto?

[ ] PROMESSA FORTE
    A promessa principal esta clara?
    E especifica o suficiente?
    E crivel?

[ ] ANGULO DEFINIDO
    Qual a perspectiva unica?
    O que diferencia de outros conteudos?
```

---

### NIVEL 2: ESTRUTURA

```
[ ] HOOK PODEROSO
    Os primeiros 125 caracteres param o scroll?
    Criaria curiosidade suficiente para continuar?
    Teste: mostre so o hook para alguem. Ele quer ler mais?

[ ] FLUXO LOGICO
    Um slide leva naturalmente ao proximo?
    A ordem dos argumentos faz sentido?
    Existe uma progressao clara?

[ ] TRANSICOES SUAVES
    As passagens entre ideias sao fluidas?
    O leitor sabe para onde esta indo?

[ ] CLIMAX ANTES DO CTA
    O ponto mais forte vem no momento certo?
    Cria momentum para a acao?

[ ] CTA CLARO
    A acao desejada e obvia?
    Existe apenas UM CTA principal?
    O beneficio de agir esta claro?
```

---

### NIVEL 3: CONTEUDO

```
[ ] PROMESSA CUMPRIDA
    O conteudo entrega o que a headline prometeu?
    O leitor ficaria satisfeito ao final?

[ ] VALOR REAL
    O leitor aprendeu/ganhou algo?
    Vale o tempo que ele investiu?

[ ] SEM ENROLACAO
    Cada slide tem uma razao de existir?
    Poderia cortar algum sem perder valor?

[ ] OBJECOES ANTECIPADAS
    As duvidas principais foram respondidas?
    As resistencias foram tratadas?

[ ] PROVA ADEQUADA
    Existe evidencia para as afirmacoes?
    A prova social e relevante e crivel?
```

---

### NIVEL 4: FRASES

```
[ ] UMA IDEIA POR FRASE
    Cada frase comunica uma unica coisa?
    Posso quebrar frases longas em duas?

[ ] VOZ ATIVA
    "Nos criamos X" > "X foi criado por nos"
    "Voce vai conseguir" > "Sera conseguido"

[ ] CURTAS E DIRETAS
    Frases tem menos de 20 palavras?
    Paragrafos tem menos de 3 linhas?

[ ] RITMO VARIADO
    Mistura de frases curtas e medias?
    Nao fica monotono?

[ ] INICIO FORTE
    As primeiras palavras de cada frase sao fortes?
    Evita comecar com "E", "Mas", "Entao" em excesso?
```

---

### NIVEL 5: PALAVRAS

```
[ ] SEM PALAVRAS VAZIAS
    Cortar: muito, realmente, basicamente, literalmente
    Cortar: praticamente, mais ou menos, talvez
    Cortar: que, de, para (quando possiveis)

[ ] VERBOS FORTES
    "Ele fez a criacao" → "Ele criou"
    "Vai fazer voce ter" → "Da a voce"
    "E importante para" → "[Beneficio direto]"

[ ] ESPECIFICIDADE
    "Muitos" → "2.847"
    "Bons resultados" → "47% mais vendas"
    "Em pouco tempo" → "Em 21 dias"

[ ] POWER WORDS
    Usou palavras de impacto onde apropriado?
    Sem excesso (parece spam)?

[ ] LINGUAGEM DO PUBLICO
    Usou as palavras que SEU PUBLICO usa?
    Evitou jargao que eles nao conhecem?
```

---

### NIVEL 6: TOM E VOZ

```
[ ] CONSISTENCIA
    O tom e o mesmo do inicio ao fim?
    Corresponde a sua marca/personalidade?

[ ] SEGUNDA PESSOA
    Usa "voce" mais do que "eu/nos"?
    O leitor se sente falado diretamente?

[ ] CONVERSA, NAO PALESTRA
    Soa como uma pessoa falando?
    Leia em voz alta — soa natural?

[ ] EMOCAO APROPRIADA
    A emocao corresponde ao objetivo?
    Nao e exagerada nem apatica?
```

---

### NIVEL 7: FORMATACAO

```
[ ] ESCANEABILIDADE
    Da pra entender a mensagem so escaneando?
    Tem espaco em branco suficiente?

[ ] HIERARQUIA VISUAL
    O mais importante se destaca?
    Headlines maiores que corpo?

[ ] MOBILE-FIRST
    Fica bom no celular?
    Texto legivel sem zoom?

[ ] EMOJIS ESTRATEGICOS
    Usados para chamar atencao, nao decorar?
    Sem excesso (max 2-3 por slide)?

[ ] CAPS COM MODERACAO
    Apenas para ENFASE real?
    Nao parece SPAM ou GRITARIA?
```

---

### NIVEL 8: VERIFICACAO FINAL

```
[ ] ORTOGRAFIA E GRAMATICA
    Passou por corretor ortografico?
    Nomes proprios estao corretos?

[ ] LINKS E CTAs
    Links funcionam?
    CTAs estao visiveis e clicaveis?

[ ] NUMEROS E DADOS
    Numeros estao corretos?
    Fontes podem ser comprovadas se questionadas?

[ ] LEGAL E ETICO
    Promessas sao verdadeiras?
    Disclaimers necessarios estao presentes?

[ ] LEITURA EM VOZ ALTA
    Li todo o texto em voz alta?
    Tropecei em alguma parte?
    Soou natural e fluido?
```

## O TESTE DO CORTE DE 50%

### O Exercicio

```
1. Conte o numero de palavras do seu copy
2. Desafie-se a cortar 50%
3. Mantenha APENAS o essencial
4. Compare as duas versoes

Voce vai descobrir que a versao curta
quase sempre e mais forte.
```

### Perguntas para Cada Frase

```
1. "Essa frase e necessaria?"
2. "Posso dizer isso com menos palavras?"
3. "Essa frase move o leitor para a acao?"
4. "Se eu cortar, o texto perde algo?"
```

## ANTES vs DEPOIS: Exemplos de Edicao

### EXEMPLO 1: Corte de Palavras Vazias

```
ANTES (23 palavras):
"Na verdade, o que realmente acontece e que
basicamente a maioria das pessoas acaba por
nao conseguir ter bons resultados."

DEPOIS (9 palavras):
"A maioria das pessoas nao consegue resultados."
```

### EXEMPLO 2: Voz Ativa

```
ANTES:
"O curso foi criado por especialistas que
sao reconhecidos no mercado."

DEPOIS:
"Especialistas reconhecidos criaram o curso."
```

### EXEMPLO 3: Especificidade

```
ANTES:
"Muitas pessoas ja tiveram otimos resultados
usando nosso metodo em pouco tempo."

DEPOIS:
"2.847 pessoas dobraram suas vendas
em menos de 30 dias com o metodo."
```

### EXEMPLO 4: Beneficio vs Feature

```
ANTES:
"Nosso curso tem 47 aulas em video
com mais de 30 horas de conteudo."

DEPOIS:
"Domine [habilidade] do zero ao avancado
no seu ritmo, com suporte individual."
```

## CHECKLIST RAPIDO (Pre-Publicacao)

Use este checklist de 2 minutos antes de publicar:

```
[ ] Hook para o scroll?
[ ] Promessa cumprida?
[ ] Uma ideia por slide?
[ ] CTA claro e unico?
[ ] Leitura em voz alta OK?
[ ] Ortografia verificada?
[ ] Mobile testado?
```

## A Regra de Ouro

```
PRIMEIRO RASCUNHO: Escreva livremente
EDICAO 1: Corte 30%
EDICAO 2: Refine cada frase
EDICAO 3: Polish cada palavra
TESTE: Leia em voz alta
PUBLICAR: So quando passar em TUDO
```
