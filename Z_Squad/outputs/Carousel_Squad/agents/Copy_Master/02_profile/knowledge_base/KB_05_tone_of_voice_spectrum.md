## O Que e Tom de Voz?

Tom de voz e a PERSONALIDADE da marca expressa em palavras. Nao e O QUE voce diz, mas COMO voce diz.

> "As pessoas esquecem o que voce disse, esquecem o que voce fez, mas nunca esquecem como voce as fez sentir." — Maya Angelou

## O Espectro de 5 Dimensoes

### DIMENSAO 1: FORMALIDADE

```
FORMAL ←――――――――――――――――→ CASUAL

1. Ultra-formal (Juridico/Institucional)
   "Prezado(a) cliente, informamos que..."

2. Formal (Corporativo B2B)
   "Gostaríamos de apresentar nossa solucao..."

3. Neutro (Jornalístico)
   "O estudo revela que 70% das empresas..."

4. Casual (Marca moderna)
   "Olha so o que a gente descobriu..."

5. Ultra-casual (Entre amigos)
   "Mano, tu nao vai acreditar nisso..."
```

### DIMENSAO 2: EMOCIONALIDADE

```
RACIONAL ←――――――――――――――――→ EMOCIONAL

1. Altamente racional (Dados)
   "Os dados mostram um aumento de 47%..."

2. Racional (Logico)
   "Considerando os fatores X, Y e Z..."

3. Equilibrado
   "Isso funciona porque... e voce vai sentir..."

4. Emocional (Storytelling)
   "Imagine acordar todo dia com essa sensacao..."

5. Altamente emocional (Inspiracional)
   "Voce MERECE isso. Voce PODE. Voce VAI."
```

### DIMENSAO 3: AUTORIDADE

```
HUMILDE ←――――――――――――――――→ AUTORITARIO

1. Humilde (Aprendiz)
   "Eu ainda estou aprendendo, mas descobri..."

2. Colaborativo (Par)
   "Vamos juntos nessa jornada..."

3. Guia (Mentor)
   "Deixa eu te mostrar o caminho..."

4. Expert (Autoridade)
   "Baseado em 15 anos de experiencia..."

5. Guru (Absoluto)
   "Isso e o que funciona. Ponto final."
```

### DIMENSAO 4: HUMOR

```
SERIO ←――――――――――――――――→ DIVERTIDO

1. Ultra-serio (Crise/Sensivel)
   "Este e um momento delicado..."

2. Serio (Profissional)
   "Precisamos abordar esse tema com cuidado..."

3. Neutro (Informativo)
   "Veja como funciona este processo..."

4. Leve (Com toques de humor)
   "Spoiler: vai dar certo (mas continua lendo)"

5. Divertido (Comedia)
   "Plot twist: seu concorrente tambem nao sabe o que ta fazendo"
```

### DIMENSAO 5: ENERGIA

```
CALMO ←――――――――――――――――→ ENERGETICO

1. Muito calmo (Meditativo)
   "Respire fundo. Vamos com calma..."

2. Tranquilo (Reflexivo)
   "Pare um momento para considerar..."

3. Equilibrado (Normal)
   "Vamos ver como isso funciona..."

4. Animado (Entusiasmado)
   "Isso aqui e MUITO bom! Olha so..."

5. Explosivo (Hype)
   "PARA TUDO! Voce precisa ver isso AGORA!"
```

## Perfis de Tom Pre-Definidos

### PERFIL 1: EXECUTIVO B2B
```
Formalidade: 4/5 (Formal)
Emocionalidade: 2/5 (Racional)
Autoridade: 4/5 (Expert)
Humor: 2/5 (Serio)
Energia: 2/5 (Tranquilo)

EXEMPLO:
"Nossa solucao foi desenvolvida para otimizar
processos corporativos, reduzindo custos em ate
40%. Mais de 500 empresas ja implementaram
com sucesso comprovado."
```

### PERFIL 2: COACH/MENTOR
```
Formalidade: 2/5 (Casual)
Emocionalidade: 4/5 (Emocional)
Autoridade: 3/5 (Guia)
Humor: 3/5 (Neutro)
Energia: 4/5 (Animado)

EXEMPLO:
"Ei, eu sei que voce ta cansado de tentar.
Mas deixa eu te contar uma coisa: eu ja
estive exatamente onde voce esta. E encontrei
a saida. Vem comigo que eu te mostro."
```

### PERFIL 3: INFLUENCER JOVEM
```
Formalidade: 1/5 (Ultra-casual)
Emocionalidade: 4/5 (Emocional)
Autoridade: 2/5 (Colaborativo)
Humor: 4/5 (Leve)
Energia: 5/5 (Explosivo)

EXEMPLO:
"GENTE! Voces nao vao acreditar no que eu
descobri! Tipo, eu to CHOCADA! Vem ver
isso aqui que vai mudar tua vida serio
mesmo nao to exagerando!"
```

### PERFIL 4: EDUCADOR TECNICO
```
Formalidade: 3/5 (Neutro)
Emocionalidade: 2/5 (Racional)
Autoridade: 4/5 (Expert)
Humor: 3/5 (Neutro com toques)
Energia: 3/5 (Equilibrado)

EXEMPLO:
"Vamos entender como esse algoritmo funciona.
Basicamente, ele analisa tres variaveis...
(nao se preocupa, nao vai ter matematica
complicada). O resultado e bem interessante."
```

### PERFIL 5: MARCA PREMIUM
```
Formalidade: 4/5 (Formal)
Emocionalidade: 3/5 (Equilibrado)
Autoridade: 5/5 (Absoluto)
Humor: 2/5 (Serio)
Energia: 2/5 (Tranquilo)

EXEMPLO:
"Excelencia nao e negociavel. Cada detalhe
foi meticulosamente pensado para entregar
uma experiencia incomparavel. Porque voce
merece nada menos que o melhor."
```

## Como Manter Consistencia de Tom

### 1. DOCUMENTO DE VOZ
```
Crie um guia que defina:
- Palavras que USAMOS
- Palavras que NAO usamos
- Frases de exemplo para cada contexto
- Do's and Don'ts
```

### 2. PERSONA DO LEITOR
```
Sempre escreva para UMA pessoa especifica:
- Nome (ficticio)
- Idade
- Dores e desejos
- Como ela fala
- O que ela le/assiste
```

### 3. LEITURA EM VOZ ALTA
```
Teste: Leia o copy em voz alta
- Soa natural?
- Parece "voce" falando?
- Seu publico falaria assim?
```

## Transicoes de Tom (Dentro do Mesmo Conteudo)

### PADRAO: GANCHO → CORPO → CTA

```
GANCHO (Alta energia, curiosidade)
"PARA TUDO! Isso vai mudar como voce ve vendas."

CORPO (Educacional, autoridade media)
"Existem 3 fatores que determinam se uma venda
acontece ou nao. O primeiro e..."

CTA (Emocional, direto)
"Voce merece vender mais. Comece agora.
Siga para mais dicas."
```

### PADRAO: EMPATIA → SOLUCAO → ACAO

```
EMPATIA (Emocional, acolhedor)
"Eu sei que ta dificil. Todo mundo fala que
e so fazer X, mas ninguem entende sua realidade."

SOLUCAO (Autoridade, confiavel)
"Depois de testar 47 metodos, encontrei o que
realmente funciona para pessoas como voce."

ACAO (Energia, urgencia)
"Chega de sofrer. DM 'QUERO' e eu te mando
o passo a passo. Vamos resolver isso HOJE."
```

## Sinais de Tom Inconsistente

### Red Flags

```
X Comecar formal e terminar casual (ou vice-versa)
X Usar girias e depois linguagem tecnica
X Tom alegre para assuntos serios
X Tom serio para conteudo leve
X Mudar de "voce" para "vocês" no mesmo texto
X Energia alta no inicio e monotona no fim
```

## Exercicio: Calibracao de Tom

### Pegue qualquer frase e reescreva em 5 tons:

**Original:** "Nosso produto ajuda voce a vender mais"

```
1. ULTRA-FORMAL:
"Nossa solucao foi desenvolvida para potencializar
os resultados comerciais da sua organizacao."

seu processo de vendas."

"Esse produto vai te ajudar a vender mais."

"Olha, isso aqui realmente faz voce vender mais."

"Mano, esse negocio e brabo pra vender!"
```
