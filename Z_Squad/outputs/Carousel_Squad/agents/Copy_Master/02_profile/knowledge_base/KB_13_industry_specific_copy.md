# KB_13 — Industry-Specific Copy

**Agente:** Copy Master
**Categoria:** SEGMENTACAO
**Versao:** 1.0

---

## Por Que Copy Varia por Industria

O mesmo produto pode precisar de copy completamente diferente dependendo de quem compra e como compra.

> "Voce nao escreve copy. Voce escreve copy para ALGUEM." — Gary Halbert

---

## SETOR 1: B2B (Business-to-Business)

### Caracteristicas do Publico

```
DECISOR:
- Geralmente nao e usuario final
- Precisa justificar para stakeholders
- Busca ROI mensuravel
- Ciclo de decisao longo
- Multiplos influenciadores

MOTIVACOES:
- Reduzir custos
- Aumentar eficiencia
- Diminuir riscos
- Impressionar superiores
- Resolver problema de time
```

### Tom e Linguagem B2B

```
TOM:
- Mais formal (mas nao robotico)
- Autoridade e credibilidade
- Dados e evidencias
- Profissional mas humano

LINGUAGEM:
✓ "Solucao", "Otimizacao", "Escalabilidade"
✓ "ROI", "KPIs", "Metricas"
✓ "Integracao", "Automacao", "Eficiencia"
✗ Girias, excesso de emojis, informalidade extrema
```

### Estrutura de Copy B2B

```
1. HOOK: Problema de negocio
   "Sua equipe perde 15h/semana em tarefas manuais?"

2. AGITACAO: Impacto nos resultados
   "Isso custa R$X/ano em produtividade perdida"

3. SOLUCAO: Beneficio de negocio
   "Automatize e recupere 60% desse tempo"

4. PROVA: Cases e numeros
   "A empresa X aumentou produtividade em 40%"

5. CTA: Proximo passo de baixo atrito
   "Agende uma demonstracao de 15 minutos"
```

### Templates B2B

```
HEADLINE:
"Como [tipo de empresa] esta [resultado mensuravel]
sem [aumento de custo/esforco]"

DEPOIMENTO:
"Antes: [problema de negocio].
Depois de implementar: [resultado com numero].
ROI: [X] meses."
— [Nome], [Cargo], [Empresa conhecida]

CTA:
"Fale com nosso especialista — sem compromisso"
"Receba um diagnostico gratuito"
"Veja como funciona na pratica — 15 min"
```

## SETOR 2: B2C (Business-to-Consumer)

### Caracteristicas do Publico

```
DECISOR:
- E o usuario final
- Decisao mais emocional
- Ciclo curto (minutos a dias)
- Um ou poucos influenciadores

MOTIVACOES:
- Resolver dor pessoal
- Satisfazer desejo
- Status/identidade
- Conveniencia
- Economia
```

### Tom e Linguagem B2C

```
TOM:
- Mais casual e conversacional
- Emocional e aspiracional
- Entretido e engajante
- Proximo e acessivel

LINGUAGEM:
✓ "Voce", "Imagina", "Finalmente"
✓ Girias moderadas do nicho
✓ Emojis estrategicos
✓ Perguntas diretas
```

### Estrutura de Copy B2C

```
1. HOOK: Dor ou desejo pessoal
   "Cansado de acordar sem energia?"

2. IDENTIFICACAO: "Eu sei como e"
   "Eu tambem passava por isso..."

3. SOLUCAO: Beneficio pessoal
   "Descubra o metodo que mudou minhas manhas"

4. PROVA: Transformacao de pessoas reais
   "A Maria saiu de [antes] para [depois]"

5. CTA: Acao imediata
   "Comece agora — e gratis"
```

### Templates B2C

```
HEADLINE:
"O [metodo/produto] para quem quer [resultado desejado]
sem [obstaculo comum]"

DEPOIMENTO:
"Eu era igual a voce. [Dor].
Ate que descobri isso. Agora [resultado].
Mudou minha vida."
— [Nome], [Caracteristica identificavel]

CTA:
"Quero isso pra mim"
"Comece sua transformacao"
"Eu quero [beneficio]"
```

## SETOR 3: PERSONAL BRAND (Marca Pessoal)

### Caracteristicas do Publico

```
DECISOR:
- Compra VOCE, nao so o produto
- Valoriza autenticidade
- Busca conexao e identificacao
- Quer aprender com sua jornada

MOTIVACOES:
- Inspiracao e motivacao
- Atalhos e experiencia de alguem
- Comunidade e pertencimento
- Transformacao pessoal
```

### Tom e Linguagem Personal Brand

```
TOM:
- Muito pessoal e autentico
- Vulneravel (com moderacao)
- Opinionated (com posicao clara)
- Story-driven

LINGUAGEM:
✓ "Eu", "Minha jornada", "O que aprendi"
✓ Historias pessoais
✓ Opiniao clara ("Eu acredito que...")
✓ Bastidores e processo
```

### Estrutura de Copy Personal Brand

```
1. HOOK: Historia ou opiniao forte
   "Perdi R$50.000 aprendendo isso..."

2. JORNADA: Sua historia real
   "Por 2 anos eu tentei de tudo..."

3. DESCOBERTA: O insight
   "Ate que percebi que o problema era..."

4. PROVA: Seus resultados
   "Hoje, eu [resultado]. E ajudo [perfil]"

5. CTA: Convite para comunidade
   "Quer aprender comigo? [CTA]"
```

### Templates Personal Brand

```
HEADLINE:
"[Tempo] atras, eu era [situacao].
Hoje, [resultado]. Aqui esta o que mudou:"

STORY:
"Vou ser honesto com voce.
[Vulnerabilidade].
Mas isso me ensinou [licao].
E e por isso que eu [acao atual]."

CTA:
"Quer fazer parte dessa jornada?"
"Vem construir isso comigo"
"Siga e acompanhe os bastidores"
```

## Comparativo por Setor

### HEADLINES

| B2B | B2C | Personal Brand |
|-----|-----|----------------|
| "Aumente a produtividade do time em 40%" | "Ganhe 2 horas do seu dia de volta" | "Como ganhei 2h/dia e mudou minha vida" |
| "Solucao enterprise para [problema]" | "O jeito mais facil de [resultado]" | "O metodo que uso todo dia para [X]" |
| "ROI comprovado em 3 meses" | "Resultados em 7 dias" | "Meus resultados em 30 dias (real)" |

### CTAs

| B2B | B2C | Personal Brand |
|-----|-----|----------------|
| "Agende uma demonstracao" | "Comece gratis" | "Me siga pra mais" |
| "Fale com especialista" | "Quero isso" | "DM que eu te ajudo" |
| "Baixe o case completo" | "Garanta o seu" | "Entra na comunidade" |

| B2B | B2C | Personal Brand |
|-----|-----|----------------|

| Cases com ROI | Avaliacoes e estrelas | Resultados de alunos |
| Certificacoes | Depoimentos emotivos | Screenshots de DMs |

## Adaptacao por Nicho Especifico

### SAUDE/FITNESS

```
CUIDADOS:
- Nao prometer cura
- Evitar "garantia de resultado"
- Disclaimer quando necessario

TOM:
- Motivacional mas realista
- Cientifico quando possivel
- Empatico com a jornada

COPY:
"O metodo que me ajudou a [resultado]
(e pode funcionar pra voce tambem)"
```

### FINANCAS/INVESTIMENTOS

```
CUIDADOS:
- Disclaimer de risco
- Nao garantir retornos
- Cuidado com promessas de riqueza

TOM:
- Educacional e informativo
- Autoridade sem arrogancia
- Realista sobre riscos

COPY:
"Como eu organizo meus investimentos
(nao e recomendacao, e minha estrategia)"
```

### EDUCACAO/CURSOS

```
TOM:
- Professor acessivel
- Metodico mas nao chato
- Transformacional

COPY:
"De [iniciante] a [nivel desejado]
com um metodo passo a passo
que qualquer pessoa consegue seguir"
```

### ECOMMERCE/PRODUTOS

```
TOM:
- Direto sobre beneficios
- Sensorial (descreva a experiencia)
- Urgente quando apropriado

COPY:
"Sinta [beneficio sensorial].
Experimente [produto].
[Resultado] garantido ou dinheiro de volta."
```

## Framework de Adaptacao

### PERGUNTAS PARA CADA SETOR

```
1. QUEM DECIDE a compra?
   → Adapte o nivel de formalidade

2. QUAL O CICLO de decisao?
   → Curto = mais emocional / Longo = mais racional

3. QUAIS AS OBJECOES tipicas?
   → Enderece no copy

4. QUAL O NIVEL DE CONSCIENCIA?
   → Ajuste a abordagem

5. COMO ELES FALAM?
   → Use a linguagem do publico
```

## Erros ao Adaptar Copy

### O Que NAO Fazer

```
X Usar copy B2C para publico B2B
  (parece amador)

X Ser muito formal para personal brand
  (perde autenticidade)

X Copiar linguagem sem entender contexto
  (soa falso)

X Ignorar regulamentacoes do setor
  (pode dar problema legal)

X Assumir que "funciona pra todo mundo"
  (copy universal nao existe)
```
