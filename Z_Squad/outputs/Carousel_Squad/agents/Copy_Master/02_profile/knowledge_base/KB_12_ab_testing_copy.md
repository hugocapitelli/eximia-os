## O Framework Cientifico de Testes

### O METODO CIENTIFICO APLICADO

```
1. OBSERVACAO
   "Meus carousels nao estao convertendo em follows"

2. HIPOTESE
   "Se eu mudar a headline para incluir numeros,
   o engajamento vai aumentar"

3. EXPERIMENTO
   Publicar versao A (sem numero) e B (com numero)
   com mesmas condicoes

4. ANALISE
   Comparar metricas apos periodo estatisticamente valido

5. CONCLUSAO
   "Headlines com numeros performam 23% melhor"

6. ITERACAO
   Aplicar aprendizado e testar proximo elemento
```

## O Que Testar no Copy (Por Prioridade)

### TIER 1: ALTO IMPACTO (Teste Primeiro)

```
1. HEADLINE/HOOK
   - Tipo de headline (pergunta vs afirmacao)
   - Presenca de numeros
   - Nivel de especificidade
   - Emocao primaria (medo vs desejo)

2. CTA
   - Verbo usado (Baixe vs Acesse vs Garanta)
   - Posicao (final vs meio)
   - Numero de CTAs (1 vs 2)
   - Urgencia (com vs sem)

3. PROMESSA PRINCIPAL
   - Beneficio principal destacado
   - Angulo de abordagem
   - Resultado prometido
```

### TIER 2: MEDIO IMPACTO

```
4. ESTRUTURA
   - Numero de slides (7 vs 10)
   - Quantidade de texto por slide
   - Ordem dos argumentos

5. TOM DE VOZ
   - Formal vs casual
   - Primeira pessoa vs segunda pessoa
   - Com humor vs serio

6. PROVA SOCIAL
   - Tipo (numeros vs depoimento)
   - Posicao (inicio vs fim)
   - Quantidade
```

### TIER 3: AJUSTE FINO

```
7. PALAVRAS ESPECIFICAS
   - Power words
   - Conectivos
   - Adjetivos

8. FORMATACAO
   - Uso de emojis
   - Caps vs minusculas
   - Quebras de linha

9. CAPTION
   - Tamanho
   - Hashtags
   - Estrutura
```

## Tipos de Testes

### TESTE A/B CLASSICO

**O que e:** Duas versoes com UMA variavel diferente.

```
VERSAO A: Headline sem numero
"Dicas para aumentar suas vendas"

VERSAO B: Headline com numero
"7 dicas para aumentar suas vendas"

VARIAVEL: Presenca de numero
CONTROLE: Todo o resto igual
```

**Quando usar:**
- Quando quer isolar impacto de UM elemento
- Quando tem volume suficiente para validacao

---

### TESTE MULTIVARIADO

**O que e:** Multiplas variaveis testadas simultaneamente.

```
VERSAO A: Headline A + CTA A
VERSAO B: Headline A + CTA B
VERSAO C: Headline B + CTA A
VERSAO D: Headline B + CTA B
```

**Quando usar:**
- Quando precisa velocidade
- Quando elementos podem ter interacao

**Cuidado:** Precisa de MUITO mais volume para significancia.

---

### TESTE SEQUENCIAL

**O que e:** Testar versoes em sequencia no tempo.

```
SEMANA 1: Abordagem A (todos os posts)
SEMANA 2: Abordagem B (todos os posts)
COMPARAR: Performance media de cada semana
```

**Quando usar:**
- Quando nao e possivel A/B simultaneo
- Para testes de tom/estilo geral

**Cuidado:** Variaveis externas podem interferir.

## Templates de Testes de Copy

### TESTE 1: HEADLINE - ESPECIFICIDADE

```
CONTROLE:
"Como ganhar dinheiro online"

VARIACAO A:
"Como ganhar R$5.000/mes online"

VARIACAO B:
"Como ganhei R$5.000/mes online em 90 dias"

HIPOTESE: Maior especificidade = maior conversao
METRICA: Taxa de clique/engajamento
```

### TESTE 2: CTA - VERBO

```
CONTROLE:
"Clique e saiba mais"

VARIACAO A:
"Baixe o guia gratis"

VARIACAO B:
"Garanta seu acesso agora"

VARIACAO C:
"Descubra o metodo"

HIPOTESE: Verbos de beneficio > verbos de acao generica
METRICA: Taxa de conversao do CTA
```

### TESTE 3: ANGULO - EMOCAO

```
CONTROLE (DOR):
"Pare de perder dinheiro com erros de copy"

VARIACAO (DESEJO):
"Multiplique seu faturamento com copy que converte"

VARIACAO (CURIOSIDADE):
"O segredo do copy que fatura milhoes"

HIPOTESE: Angulo de dor performa melhor que desejo
METRICA: Engajamento + conversao
```

### TESTE 4: PROVA SOCIAL - TIPO

```
CONTROLE (NUMERO):
"+10.000 alunos formados"

VARIACAO (DEPOIMENTO):
"'Mudou minha vida' — Maria, aluna"

VARIACAO (AUTORIDADE):
"Metodo usado pela Forbes"

HIPOTESE: Depoimentos geram mais confianca
METRICA: Taxa de conversao em venda
```

### TESTE 5: URGENCIA

```
CONTROLE (SEM URGENCIA):
"Inscreva-se na mentoria"

VARIACAO (ESCASSEZ):
"Ultimas 10 vagas para a mentoria"

VARIACAO (TEMPO):
"Inscricoes ate sexta — depois fecha"

VARIACAO (PRECO):
"Preco sobe amanha — garanta agora"

HIPOTESE: Urgencia de tempo > urgencia de quantidade
METRICA: Taxa e velocidade de conversao
```

## Como Medir Resultados

### METRICAS POR OBJETIVO

| Objetivo | Metrica Primaria | Metricas Secundarias |
|----------|-----------------|---------------------|
| Awareness | Alcance, Impressoes | Novos seguidores |
| Engajamento | Taxa de engajamento | Saves, Shares |
| Trafico | Cliques no link | Tempo no site |
| Leads | Taxa de conversao | CPL |
| Vendas | Revenue | AOV, LTV |

### CALCULANDO SIGNIFICANCIA

```
REGRA GERAL:
- Minimo 100 conversoes por variacao
- Minimo 95% de confianca
- Minimo 7 dias de teste (ciclo semanal)

FORMULA SIMPLIFICADA:
Se a diferenca for > 10%, geralmente e significativa
Se for < 5%, provavelmente e ruido

FERRAMENTAS:
- Google Optimize
- AB Tasty
- Calculadoras online de significancia
```

## Erros Comuns em Testes de Copy

### O Que NAO Fazer

```
X TESTAR MUITAS VARIAVEIS
  Impossivel saber o que causou a diferenca
  → Teste UMA variavel por vez

X ENCERRAR TESTE CEDO DEMAIS
  "Ja deu pra ver que B ta ganhando"
  → Espere significancia estatistica

X NAO DOCUMENTAR
  "Ah, eu lembro que testei isso"
  → Registre TUDO em planilha/doc

X IGNORAR CONTEXTO
  Teste na Black Friday ≠ teste em janeiro
  → Considere sazonalidade e eventos

X COPIAR TESTES DE OUTROS
  O que funciona pra X pode nao funcionar pra voce
  → Use como inspiracao, teste no SEU contexto

X TESTAR SEM HIPOTESE
  "Vamos ver qual vai melhor"
  → Sempre tenha hipotese clara ANTES

X MUDAR COPY BASEADO EM OPINIAO
  "Acho que ficaria melhor assim"
  → Deixe os dados decidirem
```
