# KB_06 — Slide-by-Slide Copy Patterns

**Agente:** Copy Master
**Categoria:** ESTRUTURA DE CAROUSEL
**Versao:** 1.0

---

## Anatomia do Carousel Perfeito

Um carousel e uma jornada. Cada slide tem um papel especifico na conducao do leitor do ponto A (scroll) ao ponto B (acao).

> "Cada slide deve fazer o leitor PRECISAR ver o proximo." — Copy Master

---

## SLIDE 1: A CAPA (Hook Slide)

### Objetivo
**PARAR O SCROLL** — Voce tem 0.5 segundos.

### Regras de Ouro

```
1. HEADLINE PODEROSA
   - Maximo 6-8 palavras
   - Use power words
   - Quebre em 2 linhas para impacto

2. CONTRASTE VISUAL
   - Texto grande e legivel
   - Cores que se destacam no feed
   - Espaco negativo estrategico

3. PROMESSA CLARA
   - O que o leitor vai ganhar?
   - Por que vale o tempo dele?
```

### Templates de Capa por Categoria

```
CURIOSIDADE:
"A verdade sobre [tema] que ninguem conta"
"Por que [coisa comum] esta errada"
"O segredo de [resultado]"

LISTA:
"[N] [coisas] para [resultado]"
"[N] erros que [consequencia]"
"[N] licoes de [autoridade]"

HOW-TO:
"Como [resultado] sem [dor]"
"O metodo para [beneficio]"
"[Tempo] para [transformacao]"

CONTRARIAN:
"Pare de [conselho comum]"
"[Coisa boa] esta te prejudicando"
"Esqueca [crenca popular]"
```

### Elementos Visuais que Funcionam

```
✓ Rosto da pessoa (conexao humana)
✓ Numeros grandes (especificidade)
✓ Setas indicando "desliza" (orientacao)
✓ Emojis estrategicos (atencao)
✓ Antes/Depois (curiosidade)
```

## SLIDE 2: O GANCHO DE RETENCAO

### Objetivo
**PRENDER A ATENCAO** — Confirmar que vale continuar.

### Estrategias

```
ESTRATEGIA 1: EXPANDA O PROBLEMA
"Voce provavelmente ja tentou..."
"O que ninguem te conta e que..."
"A maioria das pessoas para aqui..."

ESTRATEGIA 2: PROMETA O CONTEUDO
"Nos proximos slides, voce vai ver..."
"Eu vou te mostrar exatamente..."
"Ate o final, voce vai saber..."

ESTRATEGIA 3: CRIE IDENTIFICACAO
"Se voce ja se sentiu assim..."
"Isso acontece porque..."
"Eu tambem passei por isso..."
```

### Templates

```
"Antes de comecar, entenda uma coisa:
[insight que muda a perspectiva]"

"A maioria das pessoas comete esse erro:
[erro comum]. Mas existe um jeito melhor..."

"O que voce vai aprender agora
levou [tempo] para eu descobrir..."
```

---

## SLIDES 3-7: O CORPO (Conteudo Principal)

### Objetivo
**ENTREGAR VALOR** — Cumprir a promessa da capa.

### Principio do "One Thing"

```
REGRA: Uma ideia por slide
- Facilita compreensao
- Aumenta salvamentos
- Melhora retencao
```

### Estruturas de Corpo por Tipo

#### TIPO 1: LISTA (Mais Comum)

```
SLIDE 3: Item 1 — [Titulo]
         [2-3 linhas de explicacao]

SLIDE 4: Item 2 — [Titulo]
         [2-3 linhas de explicacao]

SLIDE 5: Item 3 — [Titulo]
         [2-3 linhas de explicacao]

[Continue ate o item final]
```

#### TIPO 2: PASSO A PASSO

```
SLIDE 3: Passo 1 — [Acao]
         "Primeiro, voce precisa..."

SLIDE 4: Passo 2 — [Acao]
         "Com isso pronto, agora..."

SLIDE 5: Passo 3 — [Acao]
         "Finalmente, voce vai..."
```

#### TIPO 3: PROBLEMA-SOLUCAO

```
SLIDE 3: O Problema
         [Descreva a dor]

SLIDE 4: Por Que Acontece
         [Explique a causa]

SLIDE 5: A Solucao
         [Apresente o caminho]

SLIDE 6: Como Aplicar
         [De o passo a passo]
```

#### TIPO 4: ANTES/DEPOIS

```
SLIDE 3: ANTES — [Situacao ruim]
         "Era assim que eu fazia..."

SLIDE 4: O QUE MUDOU
         "Ate que descobri..."

SLIDE 5: DEPOIS — [Situacao boa]
         "Agora, o resultado e..."
```

#### TIPO 5: MITO VS REALIDADE

```
SLIDE 3: MITO #1
         "Todo mundo diz que..."

SLIDE 4: REALIDADE
         "Mas a verdade e..."

[Repita para cada mito]
```

### Tecnicas de Transicao Entre Slides

```
"Mas isso nao e tudo..."
"O proximo ponto e ainda mais importante..."
"Agora vem a parte que muda tudo..."
"Se voce so lembrar de uma coisa, seja esta:"
"Desliza que o melhor ta por vir..."
```

## SLIDE PENULTIMO: O CLIMAX

### Objetivo
**ENTREGAR O INSIGHT MAIS VALIOSO** — O "ah-ha moment".

### Estrategias

```
ESTRATEGIA 1: O GRANDE SEGREDO
"O que realmente faz diferenca e..."

ESTRATEGIA 2: A SINTESE
"Resumindo tudo em uma frase..."

ESTRATEGIA 3: O PLOT TWIST
"Mas aqui esta o que ninguem percebe..."

ESTRATEGIA 4: O RESULTADO
"Quando voce aplica tudo isso, acontece..."
```

### Template

```
"BONUS: A dica que vale o carousel inteiro

[Insight poderoso em 2-3 linhas]

Isso sozinho pode [beneficio transformador]"
```

---

## SLIDE FINAL: O FECHAMENTO (CTA Slide)

### Objetivo
**CONVERTER ATENCAO EM ACAO** — O momento da verdade.

### Estrutura de 3 Elementos

```
1. RESUMO DO VALOR
   "Agora voce sabe [o que aprendeu]"

2. PROXIMA ACAO
   "O proximo passo e [CTA principal]"

3. REFORCO
   "[CTA secundario] para mais"
```

### Templates de Slide Final

```
TEMPLATE 1: TRIPLO CTA
"Gostou? Entao:
1. Salva para consultar depois
2. Comenta qual dica mais curtiu
3. Siga para conteudos diarios"

TEMPLATE 2: OFERTA
"Quer ir mais fundo?
[Oferta/Isca digital]
DM '[palavra]' que eu te mando"

TEMPLATE 3: PROMESSA FUTURA
"Isso foi so o comeco.
Amanha tem a parte 2.
Siga e ative as notificacoes."

TEMPLATE 4: SOCIAL PROOF
"Mais de [N] pessoas ja aplicaram isso.
Sua vez. Comeca agora.
[CTA]"
```

## Padroes de Texto por Posicao

### Tamanho do Texto

| Slide | Titulo | Corpo |
|-------|--------|-------|
| Capa | 3-6 palavras | N/A |
| Retencao | 5-8 palavras | 20-40 palavras |
| Corpo | 3-5 palavras | 30-60 palavras |
| Climax | 4-6 palavras | 40-70 palavras |
| Final | 2-4 palavras | 20-50 palavras |

### Formatacao

```
CAPA: CAIXA ALTA para headline
CORPO: Capitalizado normal
DESTAQUES: Negrito ou CAPS para enfase
NUMEROS: Grande e destacado (1., 2., 3.)
```

## Carousel de 5 Slides (Formato Rapido)

```
SLIDE 1: Hook (Capa)
         "7 formas de dobrar suas vendas"

SLIDE 2-4: Conteudo (3 pontos principais)
         Item 1, 2, 3 com explicacao

SLIDE 5: CTA
         "Salve + Siga para mais"
```

## Carousel de 10 Slides (Formato Completo)

```
SLIDE 1: Hook
SLIDE 2: Contexto/Promessa
SLIDE 3-8: Conteudo (6 pontos)
SLIDE 9: Bonus/Climax
SLIDE 10: CTA completo
```

---

## Erros Comuns por Posicao

### Capa
```
X Headline muito longa (mais de 8 palavras)
X Texto pequeno demais
X Promessa vaga ("Dicas importantes")
X Sem contraste visual
```

### Corpo
```
X Muito texto por slide (muro de texto)
X Varias ideias no mesmo slide
X Transicoes abruptas
X Conteudo que nao cumpre a promessa
```

### Final
```
X Sem CTA
X CTA escondido no meio do texto
X Multiplos CTAs confusos
X Terminar sem fechar o loop
```

## Checklist Pre-Publicacao

```
[ ] Capa para o scroll em 0.5s?
[ ] Slide 2 mantem atencao?
[ ] Uma ideia por slide no corpo?
[ ] Transicoes fluidas?
[ ] Climax entrega valor maximo?
[ ] CTA claro e acionavel?
[ ] Texto legivel no mobile?
[ ] Consistencia visual?
[ ] Promessa da capa cumprida?
```
