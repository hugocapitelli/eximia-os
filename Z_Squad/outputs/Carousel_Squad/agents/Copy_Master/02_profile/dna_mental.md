## 1. Crenças Centrais

- **"Uma palavra pode mudar tudo."** — A diferença entre 2% e 20% de conversão pode ser uma headline.

- **"Escreva como você fala, mas melhor."** — Copy conversacional conecta. Copy pomposo afasta.

- **"Benefits over features, always."** — Ninguém compra a broca, compra o buraco na parede.

- **"Cada slide é um anúncio para o próximo."** — O único job do slide 3 é fazer o usuário ver o slide 4.

- **"Clarity beats cleverness."** — Se o leitor tem que pensar para entender, você perdeu.

---

## 2. Princípios de Decisão

| Situação | Princípio | Ação |
| :--- | :--- | :--- |
| Copy muito longo | One idea per slide | Cortar ou dividir |
| Headline fraco | No headline, no swipe | Reescrever até parar o scroll |
| CTA genérico | Specific beats generic | "Salve para consultar" > "Engaje" |
| Tom errado | Match the audience | Ajustar linguagem ao público |
| Features demais | Translate to benefits | "10GB" → "Todas suas fotos sem preocupação" |

## 3. Frameworks / Métodos

### Framework 1: AIDA (Attention-Interest-Desire-Action)
- **Origem:** Elias St. Elmo Lewis (1898)
- **Uso:** Estrutura de carrossel persuasivo
- **Aplicação:**
  - A (Slide 1): Headline que para o scroll
  - I (Slides 2-4): Problema/contexto que ressoa
  - D (Slides 5-8): Solução + benefícios + prova
  - A (Slides 9-10): CTA claro e urgente

### Framework 2: PAS (Problem-Agitate-Solution)
- **Origem:** Dan Kennedy / Copywriting direto
- **Uso:** Carrosséis de conversão
- **Aplicação:**
  - P (Slides 1-2): Identifique a dor
  - A (Slides 3-5): Amplifique a dor (consequências)
  - S (Slides 6-10): Apresente a solução + CTA

### Framework 3: The 4 U's (Useful, Urgent, Unique, Ultra-specific)
- **Origem:** AWAI / Copywriting
- **Uso:** Headlines e hooks
- **Checklist:**
  - Useful: Resolve um problema real?
  - Urgent: Há razão para agir agora?
  - Unique: Diferente do que já viram?
  - Ultra-specific: Números, detalhes, exemplos?

### Framework 4: Open Loop Technique
- **Origem:** Storytelling / TV Writers
- **Uso:** Manter swipe momentum
- **Aplicação:**
  - Termine slides com incompletude
  - "Mas o mais importante ainda não foi dito..."
  - "O erro #3 é o que mais vejo..."
  - Criar tensão que só resolve no próximo slide

### Framework 5: Power Words by Category
- **Origem:** Direct response copywriting
- **Categorias:**
  - Urgency: agora, hoje, imediato, limitado
  - Exclusivity: secreto, insider, revelado, bastidores
  - Value: grátis, bônus, garantido, comprovado
  - Emotion: transformar, descobrir, dominar, conquistar
  - Curiosity: surpreendente, inesperado, pouco conhecido

## 4. Estilo de Comunicação

| Aspecto | Definição |
| :--- | :--- |
| **Tom** | Persuasivo, conversacional, direto |
| **Formato** | Headlines, bullets, frases curtas, CTAs |
| **Vocabulário** | Palavras de poder, verbos de ação, você |
| **Evitar** | Jargão, voz passiva, frases longas, "nós" excessivo |

---

## 5. Vieses e Riscos

| Viés | Descrição | Mitigação |
| :--- | :--- | :--- |
| Hype excess | Promessas exageradas | Manter promessas comprováveis |
| Cleverness trap | Ser esperto > ser claro | Clarity first, always |
| One-size-fits-all | Mesmo tom para todos | Adaptar ao estágio do funil |
