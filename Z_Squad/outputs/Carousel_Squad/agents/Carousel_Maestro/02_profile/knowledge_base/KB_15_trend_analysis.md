**Agente:** Carousel Maestro

---

## Por que Acompanhar Tendências?

O cenário de conteúdo em carrosséis muda constantemente. Acompanhar tendências permite:

1. **Relevância** — Estar alinhado com o que a audiência quer ver
2. **Alcance** — Algoritmos favorecem formatos trending
3. **Diferenciação** — Ser early adopter de novos formatos
4. **Autoridade** — Ser visto como atualizado no mercado
5. **Oportunismo** — Capitalizar momentos de alta atenção

---

## Tipos de Tendências a Monitorar

### 1. Tendências de Plataforma

Mudanças nos algoritmos, features e behaviors das redes sociais.

```
MONITORAMENTO DE PLATAFORMA

INSTAGRAM
□ Novos recursos lançados
□ Mudanças no algoritmo
□ Formatos priorizados
□ Políticas atualizadas

LINKEDIN
□ Novos recursos de carrossel
□ Mudanças de distribuição
□ Formatos que ganham tração

TIKTOK
□ Novas features de foto/carrossel
□ Trends de áudio
□ Formatos virais
```

### 2. Tendências de Conteúdo

O que está funcionando em termos de temas, formatos e estilos.

| Categoria | Exemplos |
|-----------|----------|
| **Formatos** | POV carousels, thread-style, meme formats |
| **Estilos visuais** | Minimalismo, brutalism, AI-generated |
| **Temas** | AI, trabalho remoto, saúde mental |
| **Narrativas** | Vulnerabilidade, behind-scenes, hot takes |

### 3. Tendências de Nicho

Movimentos específicos do seu setor ou audiência.

```

Hot topics atuais:
1. _______________
2. _______________
3. _______________

Conversations happening:
1. _______________
2. _______________

Emerging voices:
1. @_______________
2. @_______________
```

### 4. Tendências Culturais

Movimentos maiores na sociedade que afetam comportamento.

| Tendência Cultural | Impacto em Carrosséis |
|-------------------|----------------------|
| Autenticidade > perfeição | Raw content performa melhor |
| Atenção curta | Hooks mais fortes, menos slides |
| Busca por comunidade | Conteúdo que gera discussão |
| Fadiga de anúncios | Soft sell > hard sell |

### A - Analyze (Analisar)

**Perguntas de análise:**

```
Ao identificar uma tendência, pergunte:

□ É realmente nova ou já está saturada?
□ Faz sentido para minha audiência?
□ Posso executar com qualidade?
□ Alinha com minha marca/voz?
□ Qual o timing ideal para adotar?
```

### D - Decide (Decidir)

**Matriz de decisão de tendências:**

```
              ALINHADO COM MARCA
                    │
         SIM        │        SIM
    ┌───────────────┼───────────────┐
    │  ADAPTAR      │   ADOTAR      │
    │  CUIDADOSAMENTE│   RÁPIDO     │
    │               │               │
────┼───────────────┼───────────────┼────  TIMING
TARDE│               │               │CEDO
    │  IGNORAR      │   OBSERVAR    │
    │               │   E ADAPTAR   │
    │               │               │
    └───────────────┼───────────────┘
         NÃO        │        NÃO
                    │
```

### A - Adapt (Adaptar)

**Processo de adaptação:**

```
ADAPTANDO UMA TENDÊNCIA

1. ENTENDER A ESSÊNCIA
   Por que essa tendência está funcionando?
   _______________________________________________

2. EXTRAIR O PRINCÍPIO
   Qual o elemento core que posso usar?
   _______________________________________________

3. APLICAR À MINHA MARCA
   Como fazer isso com minha voz/estilo?
   _______________________________________________

4. TESTAR
   Criar versão adaptada e medir performance
   _______________________________________________
```

## Sinais de Alerta: Quando uma Tendência está Morrendo

### Red Flags

```
SINAIS DE SATURAÇÃO

□ Todo mundo está fazendo (oversaturation)
□ Performance está caindo mesmo fazendo "certo"
□ Audiência está comentando negativamente
□ Criadores grandes já abandonaram
□ Virou piada/meme negativo
□ Plataforma mudou o algoritmo contra
```

### Ciclo de Vida de uma Tendência

```
                    PICO
                     /\
                    /  \
        ADOÇÃO    /    \    DECLÍNIO
        RÁPIDA   /      \
                /        \
               /          \
        ______/            \______
       |                          |
   EMERGÊNCIA              SATURAÇÃO
   (early adopters)        (mainstream)


MELHOR MOMENTO PARA ADOTAR: Entre emergência e pico
EVITAR: Após o pico (saturação)
```
