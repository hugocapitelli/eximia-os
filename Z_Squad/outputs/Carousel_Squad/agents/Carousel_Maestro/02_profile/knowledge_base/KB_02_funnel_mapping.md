# KB_02 — Funnel Mapping for Carousels

**Agente:** Carousel Maestro
**Categoria:** ESTRATÉGIA
**Versão:** 1.0

---

## O Funil de Carrosséis: TOFU-MOFU-BOFU

### Visão Geral

Todo carrossel deve ser classificado por estágio do funil de marketing. Isso determina:
- Tom de voz
- Profundidade do conteúdo
- Tipo de CTA
- Métricas de sucesso

```
┌─────────────────────────────────────────────────────────────┐
│                        TOFU                                  │
│                   (Top of Funnel)                            │
│              Awareness / Descoberta                          │
│                                                              │
│  Objetivo: Atrair novos seguidores                          │
│  Tom: Acessível, amplo, educacional básico                  │
│  CTA: "Siga para mais", "Ative o sininho"                   │
├─────────────────────────────────────────────────────────────┤
│                        MOFU                                  │
│                  (Middle of Funnel)                          │
│             Consideração / Confiança                         │
│                                                              │
│  Objetivo: Construir relacionamento e credibilidade         │
│  Tom: Mais profundo, específico, expert                     │
│  CTA: "Salve para consultar", "Comente sua dúvida"          │
├─────────────────────────────────────────────────────────────┤
│                        BOFU                                  │
│                 (Bottom of Funnel)                           │
│                Decisão / Conversão                           │
│                                                              │
│  Objetivo: Converter em lead, venda ou ação                 │
│  Tom: Urgente, direto, orientado a ação                     │
│  CTA: "Link na bio", "DM 'QUERO'", "Cadastre-se"            │
└─────────────────────────────────────────────────────────────┘
```

---

## TOFU: Carrosséis de Awareness

### Características
- **Tema:** Amplo, relevante para muitas pessoas
- **Linguagem:** Simples, sem jargão
- **Valor:** Imediato, prático, aplicável
- **Tamanho:** 5-8 slides (mais curto)

### Tipos de Carrossel Ideais
1. **Listicle básico** — "5 dicas para iniciantes"
2. **Trend commentary** — "A nova tendência de X"
3. **Myth-busting** — "3 mitos sobre Y"
4. **Quote carousel** — Inspiração ampla

### CTAs Típicos
- "Siga para mais dicas como essa"
- "Ative o sininho para não perder"
- "Marque alguém que precisa ver isso"
- "Comenta se concorda"

### Métricas de Sucesso
| Métrica | Meta TOFU |
|---------|-----------|
| Reach | Maximizar |
| Follows | >1% do reach |
| Shares | >2% |
| Saves | >3% |

### Exemplo de Estrutura TOFU
```
Slide 1: "5 erros que todo iniciante em [área] comete"
Slide 2: Erro #1 (comum, relatable)
Slide 3: Erro #2
Slide 4: Erro #3
Slide 5: Erro #4
Slide 6: Erro #5
Slide 7: "A boa notícia: todos são evitáveis"
Slide 8: "Siga para mais dicas de [área]"
```

---

## MOFU: Carrosséis de Consideração

### Características
- **Tema:** Mais específico e profundo
- **Linguagem:** Pode usar termos do nicho
- **Valor:** Expertise demonstrada
- **Tamanho:** 8-12 slides (mais longo)

### Tipos de Carrossel Ideais
1. **How-to detalhado** — Tutorial completo
2. **Case study** — Resultado real com processo
3. **Framework explanation** — Modelo mental
4. **Comparison** — "X vs Y: qual escolher?"
5. **Behind-the-scenes** — Processo real

### CTAs Típicos
- "Salve para consultar depois"
- "Qual dessas dicas você já usa?"
- "DM se quiser saber mais"
- "Comente sua maior dúvida"

### Métricas de Sucesso
| Métrica | Meta MOFU |
|---------|-----------|
| Saves | >8% |
| Comments | >1% |
| DMs | Qualitativo |
| Time on post | >30s |

### Exemplo de Estrutura MOFU
```
Slide 1: "O framework que usei para [resultado específico]"
Slide 2: Contexto - onde eu estava
Slide 3: O problema que enfrentava
Slide 4: A descoberta do framework
Slide 5: Passo 1 do framework
Slide 6: Passo 2 do framework
Slide 7: Passo 3 do framework
Slide 8: Resultados obtidos
Slide 9: Erros comuns ao aplicar
Slide 10: "Salve e aplique no seu negócio"
```

---

## BOFU: Carrosséis de Conversão

### Características
- **Tema:** Focado em solução/oferta específica
- **Linguagem:** Direta, orientada a ação
- **Valor:** Transformação prometida
- **Tamanho:** 6-10 slides (objetivo claro)

### Tipos de Carrossel Ideais
1. **Product showcase** — Features + benefícios
2. **Testimonial collection** — Social proof
3. **Before/After** — Transformação visual
4. **Offer announcement** — Promoção/lançamento
5. **FAQ** — Objeções respondidas

### CTAs Típicos
- "Link na bio para garantir"
- "DM 'QUERO' para mais detalhes"
- "Vagas limitadas - cadastre-se"
- "Últimas horas para [oferta]"

### Métricas de Sucesso
| Métrica | Meta BOFU |
|---------|-----------|
| Link clicks | >1% |
| DMs received | Qualitativo |
| Conversions | Tracking via link |
| ROI | Positivo |

### Exemplo de Estrutura BOFU
```
Slide 1: "Como [nome] foi de [antes] para [depois] em X dias"
Slide 2: A situação antes (dor)
Slide 3: O que tentou e não funcionou
Slide 4: A descoberta de [solução]
Slide 5: Passo 1 da transformação
Slide 6: Passo 2 da transformação
Slide 7: Resultado final (números)
Slide 8: "Você também pode"
Slide 9: O que está incluso na [oferta]
Slide 10: "Link na bio - vagas limitadas"
```

## Matriz de Decisão: Qual Funil Escolher?

| Contexto | Funil Recomendado |
|----------|-------------------|
| Conta nova / poucos seguidores | TOFU |
| Lançamento de produto | BOFU |
| Construir autoridade | MOFU |
| Aumentar base de seguidores | TOFU |
| Gerar leads qualificados | MOFU → BOFU |
| Campanha de vendas | BOFU |
| Educar sobre tema complexo | MOFU |
| Viralizar/alcance máximo | TOFU |

## Proporção Ideal de Conteúdo

Para uma conta saudável e sustentável:

```
TOFU: 40%  ████████░░░░░░░░░░░░
MOFU: 40%  ████████░░░░░░░░░░░░
BOFU: 20%  ████░░░░░░░░░░░░░░░░
```

**Justificativa:**
- Muito TOFU = muitos seguidores, pouca conversão
- Muito BOFU = cansativo, perde seguidores
- Balance = crescimento sustentável com monetização

---

## Transição Entre Funis

### TOFU → MOFU
Quando um carrossel TOFU performa bem:
- Crie um MOFU aprofundando o tema
- "Você pediu, aqui está: o guia completo de [tema do TOFU]"

### MOFU → BOFU
Quando um carrossel MOFU gera engajamento:
- Crie um BOFU com oferta relacionada
- "Para quem quer aplicar [tema MOFU] com acompanhamento..."
