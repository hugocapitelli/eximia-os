# KB_12 — Briefing Templates

**Agente:** Carousel Maestro
**Categoria:** DOCUMENTAÇÃO
**Versão:** 1.0

---

## A Importância do Briefing

Um briefing bem estruturado é a diferença entre:
- Carrossel medíocre → Carrossel de alta performance
- Retrabalho infinito → Aprovação na primeira versão
- Equipe confusa → Squad alinhado

O Carousel Maestro usa briefings padronizados para comunicar com clareza para cada agente especialista do Carousel Squad.

---

## Checklist de Briefing Completo

Antes de enviar qualquer brief, verificar:

```
CHECKLIST DE QUALIDADE DO BRIEF

□ Objetivo está claro e mensurável?
□ Audiência está definida?
□ Mensagem central está em uma frase?
□ Estrutura/outline está completa?
□ Diretrizes visuais estão específicas?
□ Referências foram incluídas?
□ Deadline é realista?
□ Formato de entrega está definido?
□ Critérios de aprovação estão claros?
```
