# KB_06 — Audience Psychology

**Agente:** Carousel Maestro
**Categoria:** PSICOLOGIA DE AUDIÊNCIA
**Versão:** 1.0

---

## Fundamentos da Psicologia de Audiência

Carrosséis de alto impacto começam com compreensão profunda de QUEM está consumindo. Antes de criar qualquer slide, o Carousel Maestro deve mapear:

1. **Persona principal** — Quem é seu avatar ideal?
2. **Dores latentes** — O que tira o sono dessa pessoa?
3. **Desejos profundos** — O que ela realmente quer conquistar?
4. **Objeções** — O que impede a ação?
5. **Nível de consciência** — Onde ela está na jornada?

## Mapeamento de Dores (Pain Points)

### Os 5 Níveis de Dor

| Nível | Tipo | Exemplo | Uso em Carrossel |
|-------|------|---------|------------------|
| 1 | Superficial | "Não tenho tempo" | Hook inicial |
| 2 | Técnica | "Não sei usar a ferramenta X" | Conteúdo educacional |
| 3 | Emocional | "Me sinto sobrecarregado" | Conexão e empatia |
| 4 | Identidade | "Não sou bom o suficiente" | Transformação |
| 5 | Existencial | "Meu trabalho não tem propósito" | Storytelling profundo |

### Framework PAS para Carrosséis

**Problem → Agitate → Solution**

```
Slide 1: PROBLEM
"Você trabalha 12h por dia e ainda sente que está para trás?"

Slides 2-4: AGITATE
"E o pior é que..."
"Enquanto isso, seus concorrentes..."
"Se continuar assim..."

Slides 5-9: SOLUTION
"A verdade é que existe outro caminho..."
[Apresentar framework/método/solução]

Slide 10: CTA
"Pronto para mudar isso?"
```

### Conectando Desejos a Tipos de Carrossel

| Desejo | Tipo de Carrossel | Hook Exemplo |
|--------|-------------------|--------------|
| Sobrevivência | How-to prático | "Como ganhar R$5k/mês com X" |
| Segurança | Data/estatísticas | "Os 7 investimentos mais seguros de 2026" |
| Pertencer | Behind-the-scenes | "Um dia na vida de quem trabalha com X" |
| Status | Listicle aspiracional | "10 hábitos de CEOs de sucesso" |
| Legado | Storytelling | "Como construí algo que vai durar" |

## Níveis de Consciência (Eugene Schwartz)

### Os 5 Estados de Consciência

| Nível | Estado | O que a pessoa sabe | Estratégia de Carrossel |
|-------|--------|---------------------|------------------------|
| 1 | Unaware | Nem sabe que tem problema | Educar sobre o problema |
| 2 | Problem-Aware | Sabe do problema, não da solução | Agitar a dor |
| 3 | Solution-Aware | Sabe que existem soluções | Diferenciar sua solução |
| 4 | Product-Aware | Conhece seu produto | Superar objeções |
| 5 | Most Aware | Pronto para comprar | Oferta direta |

### Matriz de Conteúdo por Consciência

```
┌──────────────────────────────────────────────────────────────┐
│ NÍVEL 1-2 (Unaware/Problem-Aware)                            │
│ → TOFU content                                               │
│ → Hooks de curiosidade                                       │
│ → "Você sabia que...?"                                       │
│ → "O erro que 90% cometem"                                   │
├──────────────────────────────────────────────────────────────┤
│ NÍVEL 3 (Solution-Aware)                                     │
│ → MOFU content                                               │
│ → Comparativos, frameworks                                   │
│ → "Por que [solução] funciona"                               │
│ → "X vs Y: qual escolher"                                    │
├──────────────────────────────────────────────────────────────┤
│ NÍVEL 4-5 (Product/Most Aware)                               │
│ → BOFU content                                               │
│ → Cases, depoimentos                                         │
│ → "Como [cliente] conquistou [resultado]"                    │
│ → Ofertas diretas                                            │
└──────────────────────────────────────────────────────────────┘
```

## Gatilhos Mentais para Carrosséis

### Os 12 Gatilhos Mais Eficazes

| Gatilho | Descrição | Uso em Slide |
|---------|-----------|--------------|
| **Escassez** | Recursos limitados | "Só mais 10 vagas" |
| **Urgência** | Tempo limitado | "Últimas 24h" |
| **Prova Social** | Outros fizeram | "1.247 alunos" |
| **Autoridade** | Expert disse | "Segundo Harvard..." |
| **Reciprocidade** | Dar antes de pedir | Conteúdo de valor gratuito |
| **Compromisso** | Consistência | "Se você concorda, comente" |
| **Afinidade** | Semelhança | "Eu também já passei por isso" |
| **Curiosidade** | Loop aberto | "O que descobri vai te surpreender" |
| **Antecipação** | Futuro positivo | "Imagine quando você..." |
| **Novidade** | Algo novo | "O método que ninguém está usando" |
| **Especificidade** | Números exatos | "Em 37 dias" (não "em 1 mês") |
| **Contraste** | Antes vs depois | "De R$0 para R$47k/mês" |

## Objeções Universais e Respostas

### As 7 Objeções Mais Comuns

| Objeção | Resposta em Carrossel |
|---------|----------------------|
| "Não tenho tempo" | "Em apenas 15 minutos por dia..." |
| "Não tenho dinheiro" | "Com investimento zero você pode..." |
| "Não vai funcionar para mim" | "Funciona mesmo se você é iniciante/ocupado/etc." |

| "Já tentei antes" | "A diferença deste método é..." |
| "Não confio" | [Depoimentos, casos reais] |

## Pesquisa de Audiência: Métodos Práticos

### 1. Análise de Comentários
- Leia os últimos 50 comentários dos seus posts
- Identifique perguntas recorrentes
- Note a linguagem exata usada

### 2. Stories com Enquetes
- "Qual seu maior desafio com X?"
- "O que te impede de Y?"
- Use as respostas como base para carrosséis

### 3. DMs e Conversas
- Salve prints de dúvidas frequentes
- Identifique padrões de objeções
- Use a linguagem do cliente

### 4. Análise de Concorrentes
- Quais posts dos concorrentes têm mais engajamento?
- O que os comentários desses posts revelam?
- Que dores estão sendo atendidas?
