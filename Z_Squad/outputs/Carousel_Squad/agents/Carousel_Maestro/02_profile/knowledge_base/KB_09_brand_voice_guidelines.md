## O que é Brand Voice?

Brand Voice é a personalidade consistente da marca expressa através de palavras. Em carrosséis, a voz deve ser:

1. **Reconhecível** — O seguidor sabe que é você sem ver o @
2. **Consistente** — Mesmo tom em todos os posts
3. **Autêntica** — Reflete valores reais da marca
4. **Diferenciada** — Destaca-se dos concorrentes
5. **Adaptável** — Funciona em diferentes contextos

## Pilar 1: Tom de Voz

### Espectros de Tom

Posicione sua marca em cada espectro:

```
FORMAL ────────────●──────────── CASUAL
           (onde você está?)

SÉRIO ────────────●──────────── DIVERTIDO
           (onde você está?)

RESPEITOSO ────────────●──────────── IRREVERENTE
           (onde você está?)

ENTUSIÁSTICO ────────────●──────────── MATTER-OF-FACT
           (onde você está?)
```

### Matriz de Tom por Contexto

| Contexto | Tom Recomendado |
|----------|-----------------|
| Conteúdo educacional | Confiante, acessível |
| Storytelling | Vulnerável, autêntico |
| Promocional | Entusiástico, direto |
| Resposta a crise | Sério, empático |
| Bastidores | Leve, informal |

### Exemplo de Definição

```
NOSSA VOZ É:

✅ Confiante, não arrogante
   "Descobri um método que funciona"
   ❌ "Eu sei mais que todo mundo"

✅ Acessível, não simplista
   "Vou te explicar de um jeito simples"
   ❌ "É óbvio, qualquer um entende"

✅ Direta, não rude
   "Vamos ao que interessa"
   ❌ "Para de enrolação"

✅ Inspiradora, não pregadora
   "Você pode conseguir também"
   ❌ "Você TEM que fazer isso"
```

## Pilar 4: Personalidade

### Arquétipo de Marca

Selecione 1-2 arquétipos que definem sua marca:

| Arquétipo | Características | Exemplo de Voz |
|-----------|-----------------|----------------|
| **Sábio** | Expert, confiável | "Os dados mostram que..." |
| **Herói** | Corajoso, inspirador | "Você pode vencer isso" |
| **Criador** | Inovador, original | "Vamos criar algo novo" |
| **Explorador** | Aventureiro, livre | "Descubra novos caminhos" |
| **Rebelde** | Disruptivo, ousado | "Esqueça o que te ensinaram" |
| **Mago** | Transformador, visionário | "Imagine as possibilidades" |
| **Cara Comum** | Autêntico, igual | "Eu também passo por isso" |
| **Amante** | Apaixonado, sensorial | "Sinta a diferença" |
| **Bobo** | Divertido, leve | "Vamos rir disso juntos" |
| **Cuidador** | Empático, protetor | "Estou aqui para ajudar" |
| **Governante** | Líder, organizado | "O caminho certo é..." |
| **Inocente** | Otimista, puro | "Tudo vai dar certo" |

## Guia de Aplicação em Carrosséis

### Slide de Abertura (Hook)

**Padrões de hook por voz:**

| Tipo de Voz | Exemplo de Hook |
|-------------|-----------------|
| Confiante | "O método que uso para [resultado]" |
| Empática | "Eu sei como é difícil [dor]" |
| Divertida | "POV: você descobriu que [situação]" |
| Provocadora | "Tudo que te ensinaram sobre [tema] está errado" |
| Educativa | "Aprenda em 7 slides o que levei anos para descobrir" |

### Slides de Desenvolvimento

**Manter consistência:**
```
□ Mesmo nível de formalidade
□ Mesmo padrão de formatação
□ Mesma extensão de texto
□ Mesmo uso de emojis
□ Mesma estrutura visual
```

### Slide de CTA

**Padrões de CTA por voz:**

| Tipo de Voz | Exemplo de CTA |
|-------------|----------------|
| Amigável | "Salva aí pra lembrar depois" |
| Profissional | "Salve este carrossel para referência futura" |
| Entusiástico | "Bora! Salva e coloca em prática HOJE!" |
| Provocador | "Se você chegou até aqui, já sabe o que fazer" |
| Empático | "Quando estiver pronto, estarei aqui" |

## Checklist de Consistência de Voz

Antes de publicar, verifique:

```
CHECKLIST DE BRAND VOICE

□ O tom está alinhado com nossa definição?
□ Usamos nosso vocabulário característico?
□ Evitamos palavras da "lista negra"?
□ A personalidade está evidente?
□ O CTA soa como nossa marca?
□ Alguém reconheceria que é nosso sem ver o @?
□ Está coerente com posts anteriores?
```

## Adaptação por Contexto

### Quando ajustar o tom (sem perder a voz):

| Contexto | Ajuste |
|----------|--------|
| Assunto sensível | Mais empático, menos casual |
| Promocional | Mais direto, menos educativo |
| Resposta a crise | Mais sério, mais formal |
| Bastidores | Mais leve, mais pessoal |
| Dados/pesquisa | Mais autoridade, menos coloquial |

### O que NUNCA muda:
- Valores da marca
- Vocabulário core
- Personalidade fundamental
- Respeito pela audiência

---

## Erros Comuns de Brand Voice

| Erro | Problema | Solução |
|------|----------|---------|
| Inconsistência | Confunde o público | Documentar e treinar |
| Imitar concorrentes | Perde autenticidade | Definir diferencial |
| Voz genérica | Não se destaca | Aprofundar personalidade |
| Forçar trending | Soa falso | Só usar se alinhado |
| Mudar demais | Perde identidade | Evoluir gradualmente |
