## Por que Adaptar para Múltiplas Plataformas?

Cada plataforma tem:

1. **Audiências diferentes** — Comportamentos e expectativas únicos
2. **Algoritmos diferentes** — O que funciona em uma pode falhar em outra
3. **Formatos técnicos diferentes** — Dimensões, limites, recursos
4. **Culturas diferentes** — Linguagem e tendências próprias

**Regra de ouro:** Adapte, não replique. O mesmo conteúdo precisa ser otimizado para cada plataforma.

## Comparativo de Plataformas para Carrosséis

### Especificações Técnicas

| Especificação | Instagram | LinkedIn | TikTok |
|---------------|-----------|----------|--------|
| **Formato** | 1:1 ou 4:5 | 1:1 ou 1.91:1 | 9:16 |
| **Dimensões ideais** | 1080x1350px | 1080x1080px | 1080x1920px |
| **Máx. slides** | 20 | 20 | 35 (fotos) |
| **Limite de texto** | 2.200 chars | 3.000 chars | 2.200 chars |
| **Links clicáveis** | Não (só stories/bio) | Sim (caption) | Não (só bio) |
| **Hashtags** | 3-5 relevantes | 3-5 profissionais | 3-5 trending |

### Perfil de Audiência

| Aspecto | Instagram | LinkedIn | TikTok |
|---------|-----------|----------|--------|
| **Idade predominante** | 18-34 | 25-54 | 16-30 |
| **Mindset** | Entretenimento + inspiração | Profissional + networking | Entretenimento + descoberta |
| **Momento de consumo** | Lazer, pausas | Trabalho, desenvolvimento | Scroll infinito |
| **Expectativa** | Visual bonito + valor | Insights profissionais | Autenticidade + rapidez |
| **Velocidade de consumo** | Médio | Lento (refletido) | Muito rápido |

## Instagram: O Padrão de Carrosséis

### Características do Instagram

```
┌─────────────────────────────────────────────────────────────┐
│ INSTAGRAM CAROUSEL BEST PRACTICES                           │
├─────────────────────────────────────────────────────────────┤
│ ✓ Visual-first (design é prioridade)                       │
│ ✓ Hooks visuais + textuais                                 │
│ ✓ Storytelling funciona muito bem                          │
│ ✓ Saves e shares são métricas-chave                        │
│ ✓ Caption complementa, não repete                          │
│ ✓ CTA para salvar/compartilhar                             │
├─────────────────────────────────────────────────────────────┤
│ ✗ Não coloque links na caption (não clicáveis)             │
│ ✗ Não use texto demais por slide                           │
│ ✗ Não ignore o slide 2 (segunda chance do algoritmo)       │
└─────────────────────────────────────────────────────────────┘
```

### Formato Ideal Instagram

```
DIMENSÕES: 1080 x 1350px (4:5)

ESTRUTURA TÍPICA:
• 7-10 slides
• Texto: 150-250 caracteres/slide
• Visual: limpo, branded
• Caption: 150-300 palavras
• Hashtags: 3-5 no final
```

### O que Funciona no Instagram

| Tipo de Conteúdo | Performance | Por que |
|------------------|-------------|---------|
| Educational how-to | Alta | Valor salvável |
| Listicles visuais | Muito alta | Referência rápida |
| Storytelling pessoal | Alta | Conexão emocional |
| Behind-the-scenes | Média-alta | Humanização |
| Data visualization | Média-alta | Shareability |

## LinkedIn: O Profissional

### Características do LinkedIn

```
┌─────────────────────────────────────────────────────────────┐
│ LINKEDIN CAROUSEL BEST PRACTICES                            │
├─────────────────────────────────────────────────────────────┤
│ ✓ Insights profissionais e acionáveis                      │
│ ✓ Tom mais sóbrio e reflexivo                              │
│ ✓ Dados e pesquisas aumentam credibilidade                 │
│ ✓ Storytelling de carreira/negócios funciona               │
│ ✓ Links na caption são clicáveis                           │
│ ✓ Comentários geram muito alcance                          │
├─────────────────────────────────────────────────────────────┤
│ ✗ Evite conteúdo muito casual/informal                     │
│ ✗ Não use design "instagramável" demais                    │
│ ✗ Não ignore o caption (leitores leem mais)                │
│ ✗ Evite self-promotion excessiva                           │
└─────────────────────────────────────────────────────────────┘
```

### Formato Ideal LinkedIn

```
DIMENSÕES: 1080 x 1080px (1:1) ou 1080 x 566px (1.91:1)

ESTRUTURA TÍPICA:
• 8-12 slides (aceita mais denso)
• Texto: 200-400 caracteres/slide
• Visual: profissional, clean
• Caption: 300-600 palavras (stories funcionam)
• Hashtags: 3-5 relevantes ao setor
```

### Adaptações Específicas para LinkedIn

| Elemento | Instagram | LinkedIn |
|----------|-----------|----------|
| Hook | Curiosidade/emoção | Insight/dado impactante |
| Tom | Casual-inspiracional | Profissional-reflexivo |
| Exemplos | Lifestyle, pessoal | Cases de negócio, carreiras |
| CTA | "Salve para depois" | "Comente sua experiência" |
| Caption | Complementar, curta | Expandida, com story |

### O que Funciona no LinkedIn

| Tipo de Conteúdo | Performance | Por que |
|------------------|-------------|---------|
| Industry insights | Muito alta | Valor profissional |
| Career advice | Alta | Interesse universal |
| Case studies | Alta | Prova de expertise |
| Leadership lessons | Média-alta | Aspiracional |
| Personal journey | Média-alta | Autenticidade |

## TikTok: O Disruptivo

### Características do TikTok

```
┌─────────────────────────────────────────────────────────────┐
│ TIKTOK CAROUSEL (Photo Mode) BEST PRACTICES                 │
├─────────────────────────────────────────────────────────────┤
│ ✓ Ultra-rápido e direto ao ponto                           │
│ ✓ Tendências e formatos trending                           │
│ ✓ Autenticidade > produção perfeita                        │
│ ✓ Humor e entretenimento funcionam                         │
│ ✓ Áudio trending aumenta alcance                           │
│ ✓ Formato vertical (9:16) obrigatório                      │
├─────────────────────────────────────────────────────────────┤
│ ✗ Não seja corporativo demais                              │
│ ✗ Não ignore o áudio (adicione música)                     │
│ ✗ Não use design muito "polido" (parece ad)                │
│ ✗ Não faça slides longos demais                            │
└─────────────────────────────────────────────────────────────┘
```

### Formato Ideal TikTok

```
DIMENSÕES: 1080 x 1920px (9:16)

ESTRUTURA TÍPICA:
• 5-10 slides (mais curto)
• Texto: 50-150 caracteres/slide
• Visual: raw, autêntico
• Áudio: trending sound
• Caption: curta, com hashtags
```

### Adaptações Específicas para TikTok

| Elemento | Instagram | TikTok |
|----------|-----------|--------|
| Velocidade | Médio | Ultra-rápido |
| Estética | Polido, branded | Raw, autêntico |
| Tom | Aspiracional | Relatable, divertido |
| Hook | Promessa de valor | Curiosidade/choque |
| Áudio | Opcional | Essencial (trending) |
| Formato | 4:5 ou 1:1 | 9:16 sempre |

### O que Funciona no TikTok

| Tipo de Conteúdo | Performance | Por que |
|------------------|-------------|---------|
| "POV" carousels | Muito alta | Formato nativo |
| Quick tips | Alta | Valor rápido |
| "Things that..." | Alta | Relatable |
| Storytelling rápido | Média-alta | Entretenimento |
| Ranking/comparativo | Alta | Engajamento (debates) |

## Framework de Adaptação em 5 Passos

### Passo 1: Conteúdo Master

Crie primeiro o conteúdo "master" (geralmente para Instagram):

```
CONTEÚDO MASTER:
Título: ___
Slides: 10
Dimensão: 1080x1350
Tom: ___
```

### Passo 2: Análise de Fit

| Plataforma | Fit | Adaptação Necessária |
|------------|-----|---------------------|
| Instagram | ✓ Original | Nenhuma |
| LinkedIn | ? | Tom: ___, Estrutura: ___ |
| TikTok | ? | Tom: ___, Formato: ___ |

### Passo 3: Adaptação de Formato

```
CONVERSÃO INSTAGRAM → LINKEDIN:
□ Redimensionar para 1:1
□ Ajustar tom (mais profissional)
□ Expandir caption com contexto
□ Trocar CTA para comentários
□ Ajustar exemplos para B2B

CONVERSÃO INSTAGRAM → TIKTOK:
□ Redimensionar para 9:16
□ Simplificar texto (menos por slide)
□ Adicionar áudio trending
□ Ajustar tom (mais casual)
□ Cortar para 5-7 slides
```

### Passo 4: Validação Cultural

Verifique se o conteúdo faz sentido na cultura da plataforma:

```
CHECKLIST CULTURAL:
□ O tom está adequado à plataforma?
□ Os exemplos ressoam com essa audiência?
□ O formato está nativo?
□ O CTA faz sentido aqui?
□ As hashtags são relevantes?
```

### Passo 5: Agendamento Estratégico

| Plataforma | Melhor Dia | Melhor Horário |
|------------|------------|----------------|
| Instagram | Ter-Qui | 9h, 12h, 19h |
| LinkedIn | Ter-Qua | 8h, 10h, 17h |
| TikTok | Todos | 19h-22h |

## Erros Comuns na Adaptação

| Erro | Consequência | Solução |
|------|--------------|---------|
| Copiar sem adaptar | Baixa performance | Otimizar para cada plataforma |
| Ignorar cultura local | Parece "de fora" | Estudar trends da plataforma |
| Mesmo CTA em todas | Perde conversões | Adaptar CTA ao comportamento |
| Não ajustar formato | Visual cortado/feio | Redimensionar corretamente |
| Postar no mesmo horário | Perde alcance | Ajustar ao pico de cada uma |
