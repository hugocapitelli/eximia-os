## Micro-Stories: Definicao

Micro-stories sao narrativas compactas otimizadas para consumo rapido em redes sociais. Cada slide deve funcionar como uma "cena" completa, mas a historia so faz sentido no conjunto.

---

## FORMATO 5 SLIDES — Lightning Story

### Quando Usar:
- Insights rapidos
- Uma licao unica
- Posts de alto impacto
- Teste de conceitos

### Estrutura Padrao

```
Slide 1: HOOK + PROMISE
         Captura atencao + promete valor

Slide 2: CONTEXT
         Situacao ou problema

Slide 3: TURNING POINT
         O que mudou tudo

Slide 4: REVELATION
         O insight/resultado

Slide 5: CTA + MORAL
         Acao + licao
```

### Template A: Quick Win

```
1: "Uma mudanca. 3x mais resultado."
2: "Eu fazia [X] errado por [tempo]"
3: "Ate descobrir que [insight simples]"
4: "O resultado foi imediato: [numero/resultado]"
5: "Teste hoje. Volte e me conta. Salve isso."
```

### Template B: Myth Buster

```
1: "Para de fazer [erro comum]. Nao funciona."
2: "A maioria acredita que [mito]"
3: "A verdade: [realidade]"
4: "Por que? [explicacao rapida]"
5: "Faca [alternativa correta] em vez disso."
```

### Template C: Daily Lesson

```
1: "Licao de hoje: [titulo impactante]"
2: "Ontem eu [situacao]"
3: "Percebi que [insight]"
4: "Isso significa que [implicacao]"
5: "Sua vez: [CTA com pergunta]"
```

## FORMATO 7 SLIDES — Complete Mini-Story

### Quando Usar:
- Historias pessoais
- Cases de sucesso
- Tutoriais rapidos
- Conteudo de nicho

### Estrutura Padrao

```
Slide 1: HOOK
         Resultado ou pergunta provocativa

Slide 2: BEFORE
         Estado anterior / problema

Slide 3: INCIDENT
         O que desencadeou mudanca

Slide 4: STRUGGLE
         Dificuldades enfrentadas

Slide 5: BREAKTHROUGH
         O momento de descoberta

Slide 6: AFTER
         Resultado / transformacao

Slide 7: LESSON + CTA
         Licao universal + acao
```

### Template A: Personal Story

```
1: "Quase desisti de [area]. Ainda bem que nao desisti."
2: "Ha [tempo], eu estava [situacao negativa]"
3: "Um dia, [pessoa/evento] disse/fez [algo]"
4: "Tentei [X], [Y], [Z]. Nada funcionava."
5: "Ate que entendi: o problema era [insight]"
6: "Hoje: [resultado concreto]"
7: "Se voce esta [na mesma situacao]: [conselho]. [CTA]"
```

### Template B: Case Study

```
1: "[Cliente] foi de [A] para [B] em [tempo]"
2: "Situacao inicial: [problema]"
3: "O que tentaram antes: [solucoes que falharam]"
4: "O diagnostico: [o real problema]"
5: "A solucao: [o que fizemos]"
6: "Resultado: [numeros]"
7: "O principio por tras: [licao generalizavel]"
```

### Template C: Framework Intro

```
1: "O metodo [NOME] em 5 minutos"
2: "Por que criar isso: [problema que resolve]"
3: "Passo 1: [primeiro elemento]"
4: "Passo 2: [segundo elemento]"
5: "Passo 3: [terceiro elemento]"
6: "Exemplo real: [aplicacao]"
7: "Comece por [primeiro passo concreto]. [CTA]"
```

## FORMATO 10 SLIDES — Full Narrative

### Quando Usar:
- Historias completas
- Conteudo aprofundado
- Branding e autoridade
- Transformacoes complexas

### Estrutura Padrao

```
Slide 1: HOOK
         Captura irresistivel

Slide 2: CONTEXT
         Quem, quando, onde

Slide 3: PROBLEM
         A dor principal

Slide 4: AMPLIFICATION
         Por que o problema importa

Slide 5: TURNING POINT
         O momento de mudanca

Slide 6: SOLUTION PART 1
         Primeiro insight/acao

Slide 7: SOLUTION PART 2
         Segundo insight/acao

Slide 8: CLIMAX
         O grande momento

Slide 9: PROOF
         Evidencia do resultado

Slide 10: CTA + MORAL
          Acao + licao maior
```

### Template A: Transformation Story

```
1: "De [estado ruim] para [estado bom] em [tempo]"
2: "[Tempo] atras, eu era [descricao]"
3: "O problema: [dor especifica]"
4: "Isso afetava [areas da vida]"
5: "Tudo mudou quando [incidente]"
6: "Primeiro, [mudanca 1]"
7: "Depois, [mudanca 2]"
8: "O momento que soube que funcionou: [cena especifica]"
9: "Numeros: [antes] → [depois]"
10: "A verdade: [licao]. Comece hoje. [CTA]"
```

### Template B: Educational Deep Dive

```
1: "[Numero] [coisas] que separam [grupo A] de [grupo B]"
2: "Por que isso importa: [contexto]"
3: "Coisa #1: [com explicacao]"
4: "Coisa #2: [com explicacao]"
5: "Coisa #3: [com explicacao]"
6: "Coisa #4: [com explicacao]"
7: "Coisa #5: [com explicacao]"
8: "A mais importante: [destaque uma]"
9: "Exemplo de quem aplica: [case]"
10: "Sua vez: [acao especifica]. [CTA]"
```

### Template C: Contrarian Take

```
1: "Tudo que te ensinaram sobre [X] esta errado"
2: "O conselho tradicional: [mito comum]"
3: "Por que parece fazer sentido: [logica superficial]"
4: "O problema: [falha na logica]"
5: "A realidade: [dado/exemplo contrario]"
6: "O que funciona DE VERDADE: [verdade]"
7: "Por que isso funciona: [explicacao]"
8: "Minha experiencia: [prova pessoal]"
9: "Resultados de quem mudou: [cases]"
10: "Teste voce mesmo: [desafio]. [CTA]"
```

## Matriz de Escolha de Formato

| Objetivo | 5 Slides | 7 Slides | 10 Slides |
|----------|----------|----------|-----------|
| Insight rapido | Ideal | - | - |
| Historia pessoal | - | Ideal | Bom |
| Case study | - | Ideal | Bom |
| Tutorial | - | Bom | Ideal |
| Conteudo educacional | - | - | Ideal |
| Branding | - | - | Ideal |
| Teste de conceito | Ideal | - | - |
| Alto impacto | Bom | Ideal | - |

## Regras por Formato

### Para 5 Slides:
- Cada palavra conta — seja brutal na edicao
- Uma ideia UNICA por carrossel
- Hook deve ser irresistivel
- Sem "gordura" narrativa
- CTA direto e claro

### Para 7 Slides:
- Equilibre historia e conteudo
- Mantenha um ritmo constante
- Slide 4 (meio) pode ter mais texto
- Cada slide deve ter um proposito claro
- Nao force conteudo para preencher slides

### Para 10 Slides:
- Varie o ritmo (rapido → lento → rapido)
- Use slides "de respiro" entre conteudo denso
- Construa para o climax no slide 8
- Slides 9-10 devem ter energia alta
- Mantenha consistencia visual

---

## Metricas por Formato

### 5 Slides
- Taxa de conclusao esperada: 70-85%
- Ideal para: Views e alcance
- Risco: Superficial demais

### 7 Slides
- Taxa de conclusao esperada: 55-70%
- Ideal para: Engajamento balanceado
- Risco: "Nem um nem outro"

### 10 Slides
- Taxa de conclusao esperada: 40-55%
- Ideal para: Salvamentos e autoridade
- Risco: Drop-off no meio

---

## Adaptando Entre Formatos

### Expandindo 5 → 7 Slides
- Separe o CONTEXT em problema (2) e agravamento (3)
- Adicione um slide de "tentativa que falhou" (4)

### Expandindo 7 → 10 Slides
- Adicione 2 slides de solucao (mais detalhes)
- Inclua slide de prova social (9)
- Separe climax (7) e resultado (8)

### Condensando 10 → 7 Slides
- Junte problema + amplificacao (2-3)
- Junte solucao parte 1 + parte 2 (5-6)
- Junte prova + CTA (9-10)

### Condensando 7 → 5 Slides
- Junte before + incident (2)
- Junte struggle + breakthrough (3)
- Junte after + lesson (4)

## Checklist Universal

- [ ] O formato escolhido e adequado ao conteudo?
- [ ] Cada slide tem um proposito claro?
- [ ] Ha slides "vazios" que podem ser cortados?
- [ ] O ritmo varia de forma intencional?
- [ ] O climax esta no lugar certo para o formato?
- [ ] O CTA e proporcional ao tamanho do carrossel?
- [ ] A historia PRECISA de todos esses slides?
