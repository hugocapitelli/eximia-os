## O Funil de Instagram

### Framework TOFU/MOFU/BOFU

```
┌─────────────────────────────────────────┐
│           TOFU (Top of Funnel)           │
│         AWARENESS / DESCOBERTA           │
│                                          │
│  Objetivo: Alcance maximo, visibilidade  │
│  Metricas: Reach, Impressions, Views     │
│  Conteudo: Reels, Carrosseis virais      │
└─────────────────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────┐
│          MOFU (Middle of Funnel)         │
│       CONSIDERACAO / ENGAJAMENTO         │
│                                          │
│  Objetivo: Construir relacao, confianca  │
│  Metricas: Engagement, Saves, Follows    │
│  Conteudo: Carrosseis educativos, Lives  │
└─────────────────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────┐
│         BOFU (Bottom of Funnel)          │
│         CONVERSAO / DECISAO              │
│                                          │
│  Objetivo: Converter em acao/compra      │
│  Metricas: Clicks, DMs, Conversoes       │
│  Conteudo: Cases, Ofertas, CTAs diretos  │
└─────────────────────────────────────────┘
```

## Metricas por Etapa do Funil

### TOFU Metrics

```
PRIMARIAS:
├── Reach (alcance total)
├── Impressions (visualizacoes totais)
├── Non-follower reach (% de descoberta)
├── Video views (para Reels)

SECUNDARIAS:
├── Profile visits from Explore
├── Hashtag reach
├── Shares (amplificacao)

BENCHMARKS TOFU:
├── Reach Rate: >40% dos followers
├── Non-follower %: >20% do reach
├── Share Rate: >0.5%
```

### MOFU Metrics

```
PRIMARIAS:
├── Engagement Rate
├── Save Rate
├── Comment Rate
├── Follow Rate (novos seguidores/reach)

SECUNDARIAS:
├── DMs recebidas
├── Story replies
├── Mentions

BENCHMARKS MOFU:
├── ER: >3% (para contas <100K)
├── Save Rate: >3%
├── Follow Rate: >1% do reach
```

### BOFU Metrics

```
PRIMARIAS:
├── Link Clicks (bio, Stories, DMs)
├── DM inquiries (perguntas sobre produto/servico)
├── Leads gerados
├── Vendas/Conversoes

SECUNDARIAS:
├── Click-through Rate
├── Cost per Lead (se ads)
├── Conversion Rate

BENCHMARKS BOFU:
├── CTR (link in bio): 1-3% dos profile visits
├── DM conversion: 5-15% dos DMs viram leads
├── Lead to sale: varia muito por produto/servico
```

## Conteudo por Etapa

### TOFU Content Strategy

**Objetivo:** Maximizar descoberta e alcance

```
TIPOS DE CONTEUDO:
├── Reels de entretenimento/educacao rapida
├── Carrosseis com hooks fortes
├── Memes/conteudo relatable
├── Hot takes e opinioes
├── Trending topics adaptados

CARACTERISTICAS:
├── Altamente shareavel
├── Facil de consumir
├── Apelo amplo dentro do nicho
├── Baixa barreira de entrada
└── Nao requer conhecimento previo

EXEMPLO DE CARROSSEL TOFU:
"5 erros de [nicho] que todo iniciante comete"
→ Apelo amplo, relatable, shareavel
```

### MOFU Content Strategy

**Objetivo:** Aprofundar relacao e construir autoridade

```
TIPOS DE CONTEUDO:
├── Carrosseis educativos profundos
├── Tutoriais passo-a-passo
├── Frameworks e metodologias
├── Behind the scenes
├── Stories interativos (Q&A, enquetes)
├── Lives e conteudo ao vivo

CARACTERISTICAS:
├── Alto valor de save
├── Conteudo de referencia
├── Demonstra expertise
├── Constroi confianca
└── Requer mais tempo de consumo

EXEMPLO DE CARROSSEL MOFU:
"Framework completo para [resultado especifico]"
→ Alto save rate, demonstra expertise
```

### BOFU Content Strategy

**Objetivo:** Converter interesse em acao

```
TIPOS DE CONTEUDO:
├── Case studies com resultados
├── Depoimentos de clientes
├── Comparacoes (antes/depois)
├── Ofertas e promocoes
├── CTAs diretos para produto/servico
├── FAQ sobre objecoes comuns

CARACTERISTICAS:
├── Prova social forte
├── CTA claro e direto
├── Urgencia/escassez (quando genuina)
├── Remove objecoes
└── Facilita proxima acao

EXEMPLO DE CARROSSEL BOFU:
"Como [cliente] conseguiu [resultado] em [tempo]"
→ Prova social + demonstracao de resultado
```

## Mix de Conteudo Ideal

### Regra 70-20-10

```
70% TOFU/MOFU: Conteudo de valor gratuito
├── Educativo
├── Entretenimento
├── Relacionamento

20% MOFU: Conteudo de autoridade
├── Cases leves
├── Frameworks proprios
├── Diferenciais

10% BOFU: Conteudo de conversao
├── Ofertas
├── CTAs diretos
├── Promocoes
```

### Calendario Semanal por Funil

```
EXEMPLO (7 posts/semana):

Segunda: TOFU - Reel de alcance
Terca:   MOFU - Carrossel educativo
Quarta:  TOFU - Carrossel viral/relatable
Quinta:  MOFU - Carrossel framework
Sexta:   TOFU - Reel entretenimento
Sabado:  MOFU - Case study leve
Domingo: BOFU - CTA para oferta/lista

RESULTADO: 4 TOFU, 3 MOFU/BOFU (aproximadamente 70-30)
```

## Otimizacao de Conversao

### Taxa de Conversao por Etapa

```
FUNIL TIPICO INSTAGRAM:

1000 Reach (TOFU)
  ↓ 5% engajam
50 Engajadores (MOFU)
  ↓ 20% visitam perfil
10 Profile Visits
  ↓ 30% clicam no link
3 Link Clicks (BOFU)
  ↓ 33% convertem
1 Conversao

TAXA TOTAL: 0.1% (1 em 1000)
```

### Identificando Gargalos

```
GARGALO NO TOFU:
├── Sintoma: Baixo reach
├── Causa: Hooks fracos, horarios errados, hashtags ruins
├── Solucao: Otimizar descoberta

GARGALO NO MOFU:
├── Sintoma: Alto reach, baixo engajamento
├── Causa: Conteudo nao ressoa, audiencia errada
├── Solucao: Melhorar relevancia, testar topicos

GARGALO NA TRANSICAO (MOFU→BOFU):
├── Sintoma: Alto engajamento, poucos profile visits
├── Causa: CTA fraco, bio confusa
├── Solucao: CTAs mais claros, otimizar bio

GARGALO NO BOFU:
├── Sintoma: Muitos visits, poucos cliques
├── Causa: Oferta fraca, landing page ruim, desalinhamento
├── Solucao: Melhorar oferta, alinhar expectativas
```

### Framework de Otimizacao

```
1. MEDIR: Calcule conversao em cada etapa
2. IDENTIFICAR: Qual etapa tem maior drop-off?
3. HIPOTETISAR: Por que esta acontecendo?
4. TESTAR: Uma mudanca por vez
5. MEDIR NOVAMENTE: Funcionou?
6. ITERAR: Proximo gargalo
```

## Bio como Ponto de Conversao

### Anatomia de Bio Otimizada

```
LINHA 1: O que voce faz / Proposta de valor
LINHA 2: Para quem / Resultado prometido
LINHA 3: Credencial / Prova social
LINHA 4: CTA + Link

EXEMPLO:
"Ajudo [audiencia] a [resultado]
[Credencial: X clientes, Y resultado]
[CTA] ↓"
```

### Link in Bio Strategies

```
OPCOES:

1. LINK DIRETO: Para oferta principal
   └── Simples, direto, menos fricao

2. LINKTREE/SIMILARES: Multiplos links
   └── Flexibilidade, mas adiciona clique

3. LANDING PAGE PROPRIA: Pagina dedicada
   └── Mais controle, tracking melhor

4. LINK ROTATIVO: Muda conforme conteudo
   └── Relevancia maxima, mais trabalho

BENCHMARK: CTR de link in bio = 1-3% dos profile visits
```

## Estrategias de Captura

### Do Instagram para Lista

```
FLUXO IDEAL:
Post → Profile → Link → Landing Page → Email capturado

LEAD MAGNETS QUE FUNCIONAM:
├── Ebook/Guia gratuito
├── Template/Planilha
├── Mini-curso por email
├── Checklist
├── Acesso a comunidade
├── Webinar/Live exclusiva
├── Desconto/Trial
```

### Do Instagram para DM

```
FLUXO:
Post com CTA "DM [palavra]" → DM automatizada → Conversa → Conversao

VANTAGENS:
├── Mais pessoal
├── Maior taxa de conversao
├── Relacionamento direto
├── Sem depender de link

FERRAMENTAS:
├── ManyChat (automacao)
├── Instagram native (manual)
├── Chatbots customizados
```

## Retargeting e Nurturing

### Warming Up Cold Audience

```
SEQUENCIA DE AQUECIMENTO:

SEMANA 1-2: TOFU pesado
├── Reels de alcance
├── Conteudo viral
├── Maximizar descoberta

SEMANA 3-4: TOFU + MOFU
├── Continuar alcance
├── Adicionar conteudo profundo
├── Comecar a construir autoridade

SEMANA 5-6: MOFU + BOFU leve
├── Carrosseis de valor
├── Cases estudies
├── Primeiros CTAs sutis

SEMANA 7+: Mix completo
├── Manter ritmo de valor
├── BOFU estrategico
├── Ofertas para audiencia aquecida
```

### Instagram → Remarketing

```
COM PIXEL/API (se usar ads):
├── Visitantes do perfil
├── Engajadores de posts
├── Viewers de videos
├── Pessoas que salvaram
└── Pessoas que clicaram no link

SEM ADS (organico):
├── Salvar engajadores frequentes
├── DM de nurturing
├── Mencoes em Stories
└── Conteudo direcionado para segmentos
```
