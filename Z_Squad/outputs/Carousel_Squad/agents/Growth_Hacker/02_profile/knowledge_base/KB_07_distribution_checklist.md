# KB_07 — Distribution Checklist

**Agente:** Growth Hacker
**Categoria:** DISTRIBUICAO
**Versao:** 1.0

---

## Filosofia de Distribuicao

**"Conteudo e rei, mas distribuicao e a rainha que faz tudo acontecer."** — Adaptado de Gary Vaynerchuk

Criar conteudo excelente e 50% do trabalho. Distribuir corretamente e os outros 50%.

---

## Pre-Publicacao Checklist

### 72h Antes: Planejamento

```
[ ] CONTEUDO FINALIZADO
    [ ] Design revisado (sem erros, alinhado)
    [ ] Copy revisada (gramatica, clareza)
    [ ] CTA definido e claro
    [ ] Numero de slides otimo (5-10)

[ ] ESTRATEGIA DEFINIDA
    [ ] Objetivo do post claro (awareness/engagement/conversao)
    [ ] Metrica principal definida
    [ ] Meta numerica estabelecida
    [ ] Tipo de conteudo categorizado

[ ] TIMING PLANEJADO
    [ ] Dia escolhido (baseado em dados)
    [ ] Horario definido (baseado em audiencia)
    [ ] Nao conflita com outros posts/eventos
    [ ] Calendario atualizado
```

### 24h Antes: Preparacao

```
[ ] HASHTAGS
    [ ] 5-10 hashtags selecionadas
    [ ] Mix de nicho/comunidade/ampla
    [ ] Verificadas (nenhuma banida)
    [ ] Salvas para copy/paste

[ ] CAPTION
    [ ] Hook forte (primeira linha)
    [ ] Valor claro
    [ ] CTA incluido
    [ ] Formatacao com espacos/quebras
    [ ] Emojis estrategicos (se aplicavel)

[ ] ELEMENTOS EXTRAS
    [ ] Alt text preparado (acessibilidade + SEO)
    [ ] Localizacao definida (se relevante)
    [ ] Tags de contas (se colaboracao)
```

### 2h Antes: Aquecimento

```
[ ] STORIES DE AQUECIMENTO
    [ ] Story 1: Enquete relacionada ao topico
    [ ] Story 2: Teaser do conteudo
    [ ] Story 3: Countdown/lembrete

[ ] NOTIFICACAO DA TRIBO
    [ ] DM para top 5-10 engajadores
    [ ] Mensagem personalizada
    [ ] Pedir opiniao/feedback (nao obrigar engajamento)

[ ] ENGAGEMENT PRE-POST
    [ ] Responder DMs pendentes
    [ ] Responder comentarios em posts anteriores
    [ ] Engajar em 5-10 posts do nicho
```

### 30min Antes: Checagem Final

```
[ ] REVIEW FINAL
    [ ] Slides na ordem correta
    [ ] Primeiro slide (hook) revisado
    [ ] Ultimo slide (CTA) revisado
    [ ] Texto legivel em mobile

[ ] PREPARACAO
    [ ] App Instagram atualizado
    [ ] Conexao estavel
    [ ] Primeiro comentario preparado
    [ ] Proxima hora livre para engajar
```

## Publicacao Checklist

### Momento T-0: Publicando

```
[ ] UPLOAD
    [ ] Imagens carregadas corretamente
    [ ] Caption colada (verificar formatacao)
    [ ] Alt text adicionado
    [ ] Localizacao adicionada (se aplicavel)
    [ ] Tags de contas (se aplicavel)

[ ] CONFIGURACOES
    [ ] Conta correta (se gerencia multiplas)
    [ ] "Publicar tambem no Facebook" (opcional)
    [ ] Agendamento desativado (se postando ao vivo)

[ ] POST PUBLICADO
    [ ] Verificar se apareceu no feed
    [ ] Verificar se slides estao corretos
    [ ] Verificar se caption esta formatada
```

### T+0 a T+2min: Primeiro Comentario

```
[ ] PRIMEIRO COMENTARIO PUBLICADO
    [ ] Conteudo: CTA adicional OU hashtags OU pergunta
    [ ] Timing: dentro de 2 minutos

OPCOES DE PRIMEIRO COMENTARIO:
├── Hashtags (se preferir caption limpa)
├── Pergunta de engajamento
├── Link relevante (use encurtador)
├── Contexto adicional
└── "Salva pra depois!"
```

## Pos-Publicacao Checklist

### T+0 a T+30min: Golden Window

```
[ ] ENGAJAMENTO INTENSIVO
    [ ] Responder TODOS os comentarios
    [ ] Respostas substantivas (nao so emoji)
    [ ] Perguntas de follow-up
    [ ] Curtir cada comentario

[ ] STORIES
    [ ] Compartilhar post nos Stories
    [ ] Adicionar sticker de pergunta/enquete
    [ ] CTA para ver o post

[ ] PROATIVO
    [ ] Visitar perfis de quem comentou
    [ ] Curtir 2-3 posts recentes deles
    [ ] Comentar genuinamente em 1-2 posts
```

### T+30min a T+1h: Manutencao

```
[ ] CONTINUAR RESPONDENDO
    [ ] Novos comentarios
    [ ] DMs relacionadas ao post
    [ ] Mencoes em Stories

[ ] ENGAJAMENTO EXTERNO
    [ ] Engajar em 10+ posts do nicho
    [ ] Comentarios substantivos
    [ ] Visibilidade para novos perfis

[ ] MONITORAMENTO
    [ ] Verificar alcance inicial (Insights)
    [ ] Comparar com media de posts anteriores
    [ ] Notar tendencia (acima/abaixo da media)
```

### T+1h a T+6h: Sustentacao

```
[ ] RESPOSTAS
    [ ] Responder novos comentarios (pode demorar mais)
    [ ] Responder DMs
    [ ] Agradecer shares/mencoes

[ ] STORIES DE REFORCO
    [ ] Story destacando ponto do carrossel
    [ ] Responder pergunta dos comentarios
    [ ] "Ainda nao viu? Link nos Stories"

[ ] CROSS-PLATFORM (opcional)
    [ ] Compartilhar no LinkedIn
    [ ] Tweetar insight do carrossel
    [ ] Post em comunidades relevantes
```

### T+6h a T+24h: Follow-up

```
[ ] ENGAJAMENTO CONTINUO
    [ ] Responder comentarios restantes
    [ ] Agradecer mencoes tardias
    [ ] Responder DMs pendentes

[ ] STORIES FINAL
    [ ] "Ultimo dia pra ver esse post"
    [ ] Destacar metrica ("Ja X saves!")
    [ ] Repost com comentario favorito

[ ] ANALISE PRELIMINAR
    [ ] Anotar metricas de 24h
    [ ] Comparar com benchmark
    [ ] Identificar padroes iniciais
```

## 48h e 7 Dias: Analise

### T+48h Checklist

```
[ ] METRICAS CONSOLIDADAS
    [ ] Reach total: ___
    [ ] Engagement rate: ___
    [ ] Saves: ___
    [ ] Shares: ___
    [ ] Comments: ___

[ ] COMPARACAO
    [ ] vs Meta definida: [atingiu/nao atingiu]
    [ ] vs Media dos ultimos 10 posts: [acima/abaixo/na media]
    [ ] vs Melhor post do mes: [comparacao]

[ ] QUALITATIVO
    [ ] Tipo de comentario predominante
    [ ] Perguntas frequentes
    [ ] Feedback geral
```

### T+7d Checklist

```
[ ] METRICAS FINAIS
    [ ] Anotar metricas finais na planilha
    [ ] Calcular performance relativa
    [ ] Atualizar benchmark se necessario

[ ] APRENDIZADOS
    [ ] O que funcionou?
    [ ] O que nao funcionou?
    [ ] O que testar no proximo?

[ ] DOCUMENTACAO
    [ ] Adicionar ao log de experimentos
    [ ] Atualizar banco de dados de performance
    [ ] Planejar iteracao/repost (se alta performance)
```

## Distribution Amplifiers

### Taticas de Amplificacao

```
IMEDIATAS (0-24h):
├── Cross-post para Stories (com interacao)
├── DM para engajadores-chave
├── Comentarios estrategicos em posts do nicho
├── Responder TODOS os comentarios
└── Share em grupos relevantes (WhatsApp, Telegram)

MEDIAS (24-72h):
├── Repurpose para Reels (se aplicavel)
├── Thread no Twitter/X
├── Post no LinkedIn
└── Email para lista (se tiver)

LONGO PRAZO (7-30d):
├── Adicionar aos Highlights
├── Referenciar em posts futuros
├── Reutilizar em email marketing
└── Atualizar e repostar (se evergreen)
```

### Checklist de Cross-Platform

```
[ ] LINKEDIN
    [ ] Adaptar conteudo para formato LinkedIn
    [ ] Postar como documento (carousel nativo)
    [ ] Mencionar post original do Instagram

[ ] TWITTER/X
    [ ] Thread com principais pontos
    [ ] Link para post do Instagram
    [ ] Imagens dos slides como tweets

[ ] EMAIL
    [ ] Newsletter semanal incluir
    [ ] Link direto para o post
    [ ] Contexto adicional

[ ] BLOG (se tiver)
    [ ] Expandir em artigo
    [ ] Embedar carrossel
    [ ] SEO keywords
```

## Emergency Checklist

### Se Performance Muito Baixa

```
[ ] DIAGNOSTICO (primeiras 2h)
    [ ] Verificar se post esta visivel
    [ ] Verificar se hashtags sao validas
    [ ] Comparar horario vs audiencia online
    [ ] Revisar hook (funciona sem contexto?)

[ ] ACOES IMEDIATAS
    [ ] Story apontando para o post
    [ ] DM para top engajadores pedindo feedback
    [ ] Engajar mais agressivamente em posts do nicho
    [ ] Comentario proprio com pergunta adicional

[ ] SE NAO MELHORAR (apos 24h)
    [ ] Documentar como aprendizado
    [ ] Nao deletar (dados sao uteis)
    [ ] Analisar o que poderia melhorar
    [ ] Planejar versao melhorada
```

### Se Performance Excepcional

```
[ ] MAXIMIZAR MOMENTUM
    [ ] Stories adicionais
    [ ] DMs agradecendo engajadores
    [ ] Responder TODOS os comentarios rapidamente
    [ ] Engajar mais para manter distribuicao

[ ] DOCUMENTAR
    [ ] Anotar TUDO que foi feito diferente
    [ ] Screenshots das metricas
    [ ] Salvar copy/design para referencia

[ ] REPLICAR
    [ ] Criar variacao do tema
    [ ] Testar elementos similares
    [ ] Planejar serie baseada no topico
```
