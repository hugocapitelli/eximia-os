## Fusos Horarios e Audiencia Global

### Mapeando sua Audiencia

**No Instagram Insights:**
1. Va em Insights > Audiencia
2. Veja distribuicao por cidade/pais
3. Identifique concentracoes

### Estrategia Multi-Fuso

```
SE 80%+ em um fuso:
→ Foque 100% nesse fuso

SE 60/40 split entre dois fusos:
→ Alterne posts entre horarios otimos de cada um

SE muito fragmentado:
→ Busque interseccao (ex: 12h Sao Paulo = 16h Lisboa)
```

### Calculadora de Interseccao

**Exemplo Brasil + Portugal:**
```
Brasil (SP): 12h = Portugal: 16h
Brasil (SP): 18h = Portugal: 22h (tarde demais)
Brasil (SP): 8h = Portugal: 12h

MELHOR INTERSECCAO: 12h Brasil (almoco) = 16h Portugal (fim tarde)
```

## Frequencia de Postagem

### Benchmarks por Objetivo

| Objetivo | Frequencia Minima | Frequencia Ideal | Frequencia Maxima |
|----------|-------------------|------------------|-------------------|
| Manutencao | 3x/semana | 4-5x/semana | 7x/semana |
| Crescimento moderado | 5x/semana | 7x/semana | 10x/semana |
| Crescimento agressivo | 7x/semana | 14x/semana | 21x/semana |

### Distribuicao por Formato

**Para 7 posts/semana:**
```
DISTRIBUICAO RECOMENDADA:
├── 3-4 Carrosseis (conteudo principal)
├── 2-3 Reels (alcance e descoberta)
└── 0-1 Single image (rapido/quote)

STORIES: 3-7 por dia (complementar)
```

### Espacamento Entre Posts

```
MINIMO: 4 horas entre posts
IDEAL: 6-12 horas entre posts
MAXIMO: 48 horas sem postar (antes de penalizacao)

REGRA: Nunca poste 2x no mesmo horario em dias consecutivos
```

## Instagram Insights: Leitura Correta

### Onde Encontrar Dados de Audiencia

```
CAMINHO:
Perfil > Insights > Total de Seguidores > Horarios de Maior Atividade

DADOS DISPONIVEIS:
├── Horas (barra por hora do dia)
└── Dias (barra por dia da semana)
```

### Interpretacao dos Graficos

```
PICO NO GRAFICO:
├── NAO significa "poste exatamente nesse horario"
├── SIGNIFICA "sua audiencia esta online nesse horario"
└── POSTE 30-60 minutos ANTES do pico

EXEMPLO:
├── Grafico mostra pico as 20h
├── Poste entre 19h-19h30
└── Post ganha momentum antes do pico
```

### Limitacoes do Instagram Insights

```
INSIGHTS MOSTRA:
├── Ultimos 7 dias apenas
├── Audiencia geral (nao segmentada)
└── Aproximacao (nao exato)

INSIGHTS NAO MOSTRA:
├── Diferenca por tipo de seguidor
├── Quando seguidores ENGAJAM (vs estao online)
└── Dados historicos longos
```

## Erros Comuns

### 1. Seguir Benchmarks Cegamente

**ERRADO:** "Later disse que melhor horario e 11h, entao sempre posto 11h"

**CERTO:** Use benchmarks como ponto de partida, depois otimize com SEUS dados

### 2. Nao Considerar o "Warm-Up Time"

**ERRADO:** Postar exatamente no horario de pico

**CERTO:** Postar 30-60 minutos ANTES para o post ganhar momentum

### 3. Inconsistencia

**ERRADO:** Postar aleatoriamente quando da tempo

**CERTO:** Cronograma consistente (audiencia cria expectativa)

### 4. Ignorar Sazonalidade

**ERRADO:** Mesmo horario o ano todo

**CERTO:** Ajustar para feriados, ferias, eventos
