## Instagram Insights (Nativo)

### Como Acessar

```
REQUISITO: Conta Profissional (Business ou Creator)

CAMINHO:
Perfil > Menu (3 linhas) > Insights
OU
Perfil > Botao "Insights" (se visivel)
```

### Secoes Disponiveis

**1. Overview (Visao Geral)**
```
METRICAS:
├── Contas alcancadas
├── Contas com engajamento
├── Total de seguidores
└── Conteudo compartilhado

PERIODO: Ultimos 7, 14, 30 ou 90 dias
```

**2. Contas Alcancadas**
```
METRICAS:
├── Alcance total
├── Alcance de seguidores vs nao-seguidores
├── Impressoes
├── Visitas ao perfil
├── Toques no site
├── Top posts/Stories/Reels por alcance
└── Demograficos de alcance
```

**3. Contas com Engajamento**
```
METRICAS:
├── Total de interacoes
├── Likes, comentarios, saves, shares
├── Demograficos de engajadores
└── Top posts por engajamento
```

**4. Total de Seguidores**
```
METRICAS:
├── Crescimento (novos - perdidos)
├── Top localizacoes
├── Faixa etaria
├── Genero
├── Horarios de maior atividade (por hora e dia)
```

**5. Conteudo Compartilhado**
```
DADOS POR POST:
├── Alcance
├── Impressoes
├── Interacoes (likes, comments, saves, shares)
├── Visitas ao perfil
├── Follows
└── Alcance de hashtags/Explore/Home
```

### Insights por Post Individual

```
COMO ACESSAR:
Post > "Ver Insights"

DADOS DISPONIVEIS:
├── Likes
├── Comentarios
├── Shares
├── Saves
├── Alcance
├── Impressoes
├── Visitas ao perfil
├── Follows
├── Interacoes
└── Distribuicao de alcance (Home, Hashtags, Explore, Other)

PARA CARROSSEIS:
├── Todas as metricas acima
└── NAO mostra swipe-through rate nativo (limitacao)
```

### Limitacoes do Instagram Insights

```
O QUE NAO MOSTRA:
├── Swipe-through rate por slide
├── Dados de mais de 90 dias
├── Export nativo (tem que anotar manual)
├── Comparacoes automaticas entre posts
├── Dados demograficos detalhados
├── Horario exato de publicacao retroativo
└── Performance por hashtag individual
```

## Meta Business Suite

### Vantagens sobre Instagram Insights

```
EXTRAS:
├── Agendamento nativo
├── Inbox unificado (Instagram + Facebook)
├── Dados de ate 2 anos
├── Export de dados (CSV)
├── Comparacao de periodos
├── Insights de competidores (limitado)
└── Interface desktop (mais facil de analisar)
```

### Como Acessar

```
OPCAO 1: business.facebook.com
OPCAO 2: App Meta Business Suite (mobile)
OPCAO 3: Via Facebook > Menu > Business Suite
```

### Secoes Relevantes

**1. Content (Conteudo)**
```
├── Todos os posts publicados
├── Filtros por tipo/data/performance
├── Metricas lado a lado
└── Export para planilha
```

**2. Insights**
```
├── Overview de todas as contas
├── Resultados por periodo
├── Benchmarks e comparacoes
└── Audience insights mais detalhados
```

**3. Planner**
```
├── Calendario de conteudo
├── Agendamento
├── Preview de posts
└── Rascunhos
```

## Ferramentas de Terceiros

### Tier 1: Scheduling + Analytics

**Later**
```
PRECO: $18-40/mes
PROS:
├── Visual planner excelente
├── Best Time to Post (AI)
├── Linkin.bio analytics
├── Hashtag suggestions
├── Instagram-focused

CONTRAS:
├── Analytics limitado no plano basico
└── Pode ser caro para iniciantes

METRICAS EXCLUSIVAS:
├── Swipe-through rate (estimado)
├── Comparacao de posts
└── Hashtag performance
```

**Buffer**
```
PRECO: $15-99/mes
PROS:
├── Interface simples
├── Multi-plataforma
├── Bom custo-beneficio
├── Team features

CONTRAS:
├── Analytics menos profundo
└── Instagram nao e foco principal

METRICAS EXCLUSIVAS:
├── Comparacao cross-platform
└── Sugestao de conteudo
```

**Hootsuite**
```
PRECO: $49-739/mes
PROS:

CONTRAS:
├── Caro

├── Social listening
```

## Construindo seu Stack de Analytics

### Para Iniciantes (Budget: $0-20/mes)

```
STACK RECOMENDADO:
├── Instagram Insights (gratis)
├── Meta Business Suite (gratis)
├── Google Sheets (gratis) - para tracking manual
└── Social Blade (gratis) - competitor tracking

WORKFLOW:
├── Semanal: Exportar dados para Sheets
├── Mensal: Analisar tendencias
└── Por post: Anotar metricas principais
```

### Para Intermediarios (Budget: $20-50/mes)

```
STACK RECOMENDADO:
├── Instagram Insights (gratis)
├── Later ou Metricool ($18-35/mes)
├── Google Sheets para custom tracking
└── Not Just Analytics para competitors

WORKFLOW:
├── Later: Scheduling + metricas principais
├── Sheets: Analises customizadas
└── NJA: Benchmark mensal vs competitors
```

### Para Avancados (Budget: $50-150/mes)

```
STACK RECOMENDADO:
├── Iconosquare ($49-79/mes)
├── Google Looker Studio (gratis) para dashboards
├── API do Instagram (se necessario)
└── HypeAuditor (se trabalha com influencers)

WORKFLOW:
├── Iconosquare: Metricas profundas
├── Looker: Dashboard executivo
└── Reports automatizados semanais
```

### Para Enterprise (Budget: $150+/mes)

```
STACK RECOMENDADO:
├── Sprout Social ($249+/mes)
├── BI tool (Tableau, Looker)
├── Custom API integrations
└── Data warehouse (BigQuery)

WORKFLOW:
├── Sprout: Operacao diaria
├── BI: Executive dashboards
└── Custom: Analises especificas
```

```
Colunas:
├── Data
├── Tipo (Carrossel/Reel/Image)
├── Topico/Tema
├── Hashtags usadas
├── Horario postado
├── Slides (se carrossel)
├── Reach
├── Impressions
├── Likes
├── Comments
├── Saves
├── Shares
├── Profile visits
├── Follows
```

```
Colunas:
├── Semana
├── Posts publicados
├── Reach total

├── ER medio

├── Pior post (link)
```

```
Colunas:
├── Mes

├── Reach total

├── Growth rate
```

**Aba 4: Competitor Tracking**

## Automacao de Reports

### Zapier/Make Integrations

```
AUTOMACOES UTEIS:
├── Later → Sheets: Log automatico de posts
├── Instagram (via API) → Sheets: Metricas diarias
├── Sheets → Email: Report semanal automatico
└── Sheets → Slack: Alerta de meta atingida
```

### Ferramentas de Automacao

| Ferramenta | Uso | Complexidade |

| Zapier | Integracoes simples | Baixa |
