## O Framework AARRR para Carrosseis

Sean Ellis popularizou o modelo Pirate Metrics. Adaptado para Instagram:

```
AWARENESS  → Alcance, Impressoes
ACQUISITION → Novos seguidores, Profile visits
ACTIVATION  → Saves, Shares, First interaction
RETENTION   → Comentarios recorrentes, DMs
REVENUE     → Link clicks, Conversoes
```

---

## Metricas Primarias (Tier 1)

### 1. Engagement Rate (ER)

**Formula Padrao:**
```
ER = (Likes + Comments + Saves + Shares) / Followers × 100
```

**Formula por Alcance (mais precisa):**
```
ER by Reach = (Likes + Comments + Saves + Shares) / Reach × 100
```

**Benchmarks por Tamanho de Conta (2025-2026):**

| Seguidores | ER Baixo | ER Medio | ER Bom | ER Excelente |
|------------|----------|----------|--------|--------------|
| <1K | <3% | 3-6% | 6-10% | >10% |
| 1K-10K | <2% | 2-4% | 4-7% | >7% |
| 10K-100K | <1.5% | 1.5-3% | 3-5% | >5% |
| 100K-1M | <1% | 1-2% | 2-3.5% | >3.5% |
| >1M | <0.5% | 0.5-1% | 1-2% | >2% |

**Carrosseis especificamente:**
- ER medio: 1.92% (vs 1.22% para imagens unicas)
- Top performers: 5-8% ER

### 2. Save Rate

**Formula:**
```
Save Rate = Saves / Reach × 100
```

**Benchmarks:**

| Tipo de Conteudo | Save Rate Medio | Save Rate Bom | Save Rate Excelente |
|------------------|-----------------|---------------|---------------------|
| Educativo | 2-4% | 4-7% | >7% |
| Entretenimento | 0.5-1% | 1-2% | >2% |
| Inspiracional | 1-2% | 2-4% | >4% |
| Tutorial/How-to | 3-5% | 5-10% | >10% |
| Template/Recurso | 5-8% | 8-15% | >15% |

**Por que e a metrica mais importante:**
- Indica valor de longo prazo
- Peso 4x maior que likes no algoritmo (estimativa)
- Correlaciona com reposicao no feed

### 3. Share Rate

**Formula:**
```
Share Rate = Shares / Reach × 100
```

**Benchmarks:**

| Tipo de Conteudo | Share Rate Medio | Share Rate Bom |
|------------------|------------------|----------------|
| Relatable/Memes | 1-2% | >3% |
| Educativo | 0.5-1% | >1.5% |
| Controverso | 1-3% | >4% |
| Util para outros | 0.8-1.5% | >2% |

**Neil Patel insight:** "Shares sao o multiplicador de alcance organico. Um share vale mais que 100 likes porque expoe seu conteudo a uma audiencia completamente nova."

## Metricas Secundarias (Tier 2)

### 4. Reach Rate

**Formula:**
```
Reach Rate = Reach / Followers × 100
```

**Benchmarks:**

| Tipo de Post | Reach Rate Medio | Reach Rate Bom |
|--------------|------------------|----------------|
| Carrossel | 30-45% | >50% |
| Single Image | 20-35% | >40% |
| Reels | 50-200%+ | >100% |
| Story | 5-15% | >20% |

**Alcance alem dos seguidores (Explore/Hashtags):**
- <10% do alcance de nao-seguidores: Baixo
- 10-30%: Medio
- 30-50%: Bom
- >50%: Viral

### 5. Swipe-Through Rate (STR)

**Formula:**
```
STR = Users que viram ultimo slide / Users que viram primeiro slide × 100
```

**Benchmarks:**

| Numero de Slides | STR Medio | STR Bom | STR Excelente |
|------------------|-----------|---------|---------------|
| 3 slides | 60-70% | 70-80% | >80% |
| 5 slides | 45-55% | 55-65% | >65% |
| 7 slides | 35-45% | 45-55% | >55% |
| 10 slides | 25-35% | 35-45% | >45% |

**Red flags:**
- Drop >30% entre slides 1 e 2: Hook fraco
- Drop consistente: Conteudo nao entrega valor
- Drop no penultimo slide: CTA mal posicionado

### 6. Comment Rate

**Formula:**
```
Comment Rate = Comments / Reach × 100
```

**Benchmarks:**

| Tipo | Comment Rate Medio | Comment Rate Bom |
|------|-------------------|------------------|
| Pergunta direta | 1-2% | >3% |
| Controverso | 0.5-1.5% | >2% |
| Educativo | 0.2-0.5% | >0.8% |
| Geral | 0.3-0.7% | >1% |

## Metricas de Crescimento (Tier 3)

### 7. Follower Growth Rate

**Formula:**
```
Monthly Growth Rate = (Novos - Perdidos) / Total inicial × 100
```

**Benchmarks:**

| Fase da Conta | Crescimento Saudavel | Crescimento Bom | Crescimento Excelente |
|---------------|---------------------|-----------------|----------------------|
| 0-1K | 10-20%/mes | 20-40%/mes | >40%/mes |
| 1K-10K | 5-10%/mes | 10-20%/mes | >20%/mes |
| 10K-100K | 3-5%/mes | 5-10%/mes | >10%/mes |
| 100K+ | 1-3%/mes | 3-5%/mes | >5%/mes |

### 8. Profile Visit Rate

**Formula:**
```
Profile Visit Rate = Profile Visits / Reach × 100
```

**Benchmarks:**
- Baixo: <2%
- Medio: 2-5%
- Bom: 5-10%
- Excelente: >10%

### 9. Follow Rate (de visitas)

**Formula:**
```
Follow Rate = New Follows / Profile Visits × 100
```

**Benchmarks:**
- Baixo: <5%
- Medio: 5-10%
- Bom: 10-20%
- Excelente: >20%

## Metricas de Retencao

### 10. Returning Engagers Rate

**Formula:**
```
Returning Rate = Engajadores repetidos / Total de engajadores unicos × 100
```

**Benchmarks:**
- Comunidade fraca: <20%
- Comunidade media: 20-35%
- Comunidade forte: 35-50%
- Comunidade excepcional: >50%

### 11. Story Retention

**Formula:**
```
Story Retention = Views no ultimo story / Views no primeiro story × 100
```

**Benchmarks por quantidade:**

| Stories | Retencao Media | Retencao Boa |
|---------|----------------|--------------|
| 3 | 70-80% | >85% |
| 5 | 50-60% | >70% |
| 7 | 35-45% | >55% |
| 10+ | 25-35% | >40% |
