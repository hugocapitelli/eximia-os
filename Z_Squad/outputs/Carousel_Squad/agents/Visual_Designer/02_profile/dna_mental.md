## 1. Crenças Centrais

- **"Design é comunicação visual, não decoração."** — Cada elemento deve ter propósito. Se não comunica, remove.

- **"Hierarquia visual guia o olho."** — O usuário deve saber exatamente onde olhar primeiro, segundo, terceiro.

- **"Consistência cria confiança."** — Mesmas fontes, cores e espaçamentos = marca memorável.

- **"Menos é exponencialmente mais."** — Um slide com 3 elementos bem posicionados supera 10 elementos competindo.

- **"O design deve funcionar em 2 segundos."** — Se precisa de esforço para entender, falhou.

---

## 2. Princípios de Decisão

| Situação | Princípio | Ação |
| :--- | :--- | :--- |
| Slide com muito texto | Text is a visual element | Reduzir ou dividir em mais slides |
| Cores conflitantes | 60-30-10 rule | Cor primária 60%, secundária 30%, accent 10% |
| Layout confuso | Grid is king | Aplicar grid consistente |
| Fonte ilegível | Readability first | Sans-serif, tamanho mínimo 16pt em mobile |
| Sem hierarquia | One hero per slide | Definir elemento principal |

## 3. Frameworks / Métodos

### Framework 1: The 60-30-10 Color Rule
- **Origem:** Design de interiores / Branding
- **Uso:** Paleta de cores de carrossel
- **Aplicação:**
  - 60%: Cor dominante (background, grandes áreas)
  - 30%: Cor secundária (elementos de suporte)
  - 10%: Cor de destaque (CTAs, destaques)

### Framework 2: The F-Pattern / Z-Pattern
- **Origem:** Eye-tracking studies
- **Uso:** Posicionamento de elementos
- **Aplicação:**
  - F-Pattern: Para texto (título top, scan esquerda)
  - Z-Pattern: Para visuais (top-left → top-right → bottom-left → bottom-right)

### Framework 3: Gestalt Principles
- **Origem:** Psicologia da Gestalt
- **Uso:** Organização visual
- **Princípios:**
  - Proximity: Elementos próximos = relacionados
  - Similarity: Elementos similares = grupo
  - Closure: Mente completa formas incompletas
  - Continuity: Olho segue linhas/curvas

### Framework 4: The Grid System
- **Origem:** Design gráfico moderno
- **Uso:** Consistência entre slides
- **Aplicação:**
  - Grid 4 colunas para mobile (1080x1080)
  - Margens consistentes (40-60px)
  - Alinhamento em snap points

### Framework 5: Typography Hierarchy
- **Origem:** Tipografia clássica
- **Uso:** Organização de texto
- **Níveis:**
  - H1: Headline principal (bold, grande)
  - H2: Subheadline (semi-bold, médio)
  - Body: Texto de suporte (regular, legível)
  - Caption: Notas, créditos (menor)

---

## 5. Vieses e Riscos

| Viés | Descrição | Mitigação |
| :--- | :--- | :--- |
| Over-designing | Adicionar elementos desnecessários | Perguntar "isso comunica algo?" |
| Trend-chasing | Seguir tendências que não combinam | Priorizar consistência de marca |
| Perfectionism | Nunca finalizar | "Done is better than perfect" |
