## Fundamentos Tipográficos

### A Importância da Tipografia

Typography is 95% of design. In carousel design, typography serves three critical functions:
1. **Communication** — Delivering the message clearly
2. **Hierarchy** — Guiding the viewer's reading order
3. **Branding** — Establishing visual identity and personality

### Anatomia Tipográfica Essencial
```
┌─────────────────────────────────────────────┐
│  HEADLINE          → Impact, emotion        │
│  Subheadline       → Context, support       │
│  Body text         → Information, detail    │
│  Caption/Label     → Metadata, attribution  │
└─────────────────────────────────────────────┘
```

## Font Pairing Strategies

### The 2+1 Rule

For carousels, limit to maximum 3 typefaces:
- **1 Display Font** — Headlines and hooks
- **1 Body Font** — Content and explanations
- **1 Accent Font (optional)** — Numbers, CTAs, special elements

### Pairing Techniques

**Contrast Pairing (Recommended)**
```
┌──────────────────────────────────────┐
│ Serif + Sans-Serif                   │
│ ─────────────────                    │
│ Playfair Display  │  Inter           │
│ Libre Baskerville │  Open Sans       │

```

## Typographic Hierarchy

### Size Scale System

Use a modular scale for consistent sizing:
```
Base Size: 24px (Body text)
Scale Ratio: 1.25 (Major Third)

H1: 58px   (24 × 1.25³ ≈ 58)  → Main hook
H2: 47px   (24 × 1.25² ≈ 47)  → Section headers
H3: 37px   (24 × 1.25¹ ≈ 37)  → Subheadings
Body: 24px (Base)             → Content
Small: 19px (24 ÷ 1.25)       → Captions
Micro: 15px (24 ÷ 1.25²)      → Labels
```

## Readability Optimization

### Line Length (Measure)

For carousel slides:
- **Ideal**: 35-50 characters per line
- **Maximum**: 60 characters
- Headlines: 15-25 characters for punch

```
TOO LONG:
┌─────────────────────────────────────────────────────┐
│ Este texto é muito longo para uma linha de          │
│ carrossel e vai causar fadiga visual                │
└─────────────────────────────────────────────────────┘

IDEAL:
┌──────────────────────────────────┐
│ Texto curto e                    │
│ impactante sempre                │
│ funciona melhor                  │
└──────────────────────────────────┘
```

### Line Height (Leading)

```
Heading: line-height: 1.0-1.1 (tight)
Body:    line-height: 1.4-1.5 (comfortable)
Lists:   line-height: 1.6-1.8 (generous)
```

### Letter Spacing (Tracking)

```
Headlines (All Caps): +2% to +5% tracking
Body Text: 0% (default)
Small Text: +1% (improved legibility)
```

## Mobile-First Typography

### Minimum Sizes for Instagram

```
┌────────────────────────────────────────┐
│ Element     │ Min Size │ Recommended   │
│─────────────│──────────│───────────────│
│ Headline    │ 40px     │ 48-72px       │
│ Subhead     │ 28px     │ 32-40px       │
│ Body        │ 20px     │ 24-28px       │
│ Caption     │ 16px     │ 18-20px       │
│ CTA Button  │ 24px     │ 28-32px       │
└────────────────────────────────────────┘
```

### Thumb-Scroll Consideration

Users scroll quickly. Typography must:
- Be scannable in 1-2 seconds
- Have clear visual entry points
- Use strategic bold/emphasis

## Text Treatment Techniques

### Emphasis Methods

```
1. SIZE CONTRAST
   ┌─────────────────────────┐
   │ HUGE word               │
   │ with small context      │
   └─────────────────────────┘

2. WEIGHT CONTRAST
   ┌─────────────────────────┐
   │ Light text with         │
   │ BOLD emphasis           │
   └─────────────────────────┘

3. COLOR CONTRAST
   ┌─────────────────────────┐
   │ Normal text and         │
   │ HIGHLIGHTED word        │
   └─────────────────────────┘

4. UNDERLINE/HIGHLIGHT
   ┌─────────────────────────┐
   │ Text with               │
   │ ━━━━━━━━━━━ highlight   │
   └─────────────────────────┘
```

### Text Effects to Use Sparingly

- Drop shadows (subtle, for contrast)
- Outlines (for text over images)
- Gradients (for modern hooks)
- Backgrounds/Pills (for emphasis)
