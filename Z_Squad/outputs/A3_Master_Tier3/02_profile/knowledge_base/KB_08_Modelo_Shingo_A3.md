# KB_08: Integração Modelo Shingo e A3

## O que é o Modelo Shingo

O Modelo Shingo é um framework globalmente reconhecido para alcançar excelência operacional, focando em criar uma cultura de melhoria contínua. É baseado em princípios, sistemas e ferramentas, priorizando comportamentos ideais sobre métricas de desempenho.

O modelo recebe o nome de Shigeo Shingo, um dos arquitetos do Sistema Toyota de Produção e pai do conceito de Poka-Yoke.

## O Diamante da Transformação

O Modelo Shingo é frequentemente representado na forma de um diamante, também conhecido como processo de transformação. Ele representa as relações entre quatro componentes:

```
                    RESULTADOS
                        ↑
                   COMPORTAMENTOS
                        ↑
                     SISTEMAS
                        ↑
                    PRINCÍPIOS
                        ↓
            ┌───────────────────────┐
            │       CULTURA         │
            │     (Centro de Tudo)  │
            └───────────────────────┘
```

### Fluxo Causal (CRÍTICO para A3)

```
PRINCÍPIOS informam → SISTEMAS direcionam → COMPORTAMENTOS produzem → RESULTADOS
```

Esta sequência é a base do princípio usado no A3: **Sistema → Comportamento → Resultado**

## Os Três Níveis de Princípios Shingo

### 1. Cultural Enablers (Habilitadores Culturais)

| Princípio | Descrição |
|-----------|-----------|
| **Respeito por cada indivíduo** | Valorizar pessoas como seres humanos completos |
| **Liderar com humildade** | Líderes servem, não comandam |

### 2. Continuous Improvement (Melhoria Contínua)

| Princípio | Descrição |
|-----------|-----------|
| **Buscar perfeição** | Nunca se contentar com o status quo |
| **Abraçar pensamento científico** | PDCA, experimentação, aprendizado |
| **Foco no processo** | Melhorar o processo, não culpar pessoas |
| **Assegurar qualidade na fonte** | Construir qualidade, não inspecionar |
| **Melhorar fluxo e puxar valor** | Eliminar desperdícios, criar pull |

### 3. Enterprise Alignment (Alinhamento Empresarial)

| Princípio | Descrição |
|-----------|-----------|
| **Pensar sistemicamente** | Ver conexões, não silos |
| **Criar constância de propósito** | Direção clara e consistente |
| **Criar valor para o cliente** | Foco no que importa |

## Aplicação no A3

### Princípio Central: Sistema → Comportamento → Resultado

Este princípio Shingo é **OBRIGATÓRIO** em todo A3:

1. **Nunca culpar pessoa** (comportamento individual)
2. **Sempre mudar sistema** (estrutura que gera comportamento)
3. **Sistema corrigido → Comportamento muda → Resultado melhora**

### Exemplo Prático

| Abordagem | Foco | Resultado |
|-----------|------|-----------|
| ❌ ERRADA | "Operador não seguiu procedimento" | Treinar operador (comportamento) |
| ✅ CORRETA | "Sistema permite que procedimento seja ignorado" | Implementar poka-yoke (sistema) |

### Por que Sistema Primeiro

- Pessoas geralmente querem fazer certo
- Se erram, o sistema permitiu ou induziu
- Mudar pessoa é temporário (ela ou outras vão errar novamente)
- Mudar sistema é duradouro (previne erro de qualquer pessoa)

## Papel da Liderança no Modelo Shingo

### O que Líderes Fazem
> "O trabalho dos líderes é colocar princípios orientadores na cultura organizacional."

### O que Gestores Fazem
> "O trabalho dos gestores é construir e manter sistemas de gestão em torno deles."

### O que Todos Fazem
> "A participação de cada membro da organização na melhoria das atividades é a chave para o sucesso."

## Integração com A3

| Componente Shingo | Aplicação no A3 |
|-------------------|-----------------|
| **Princípios** | Guiam a análise de causa raiz (sempre sistêmica) |
| **Sistemas** | São o foco das contramedidas |
| **Comportamentos** | Mudam como consequência dos sistemas |
| **Resultados** | São medidos no monitoramento |
| **Cultura** | É desenvolvida através do processo A3 (coaching) |

## KPIs e o Modelo Shingo

### Visão Shingo sobre Indicadores

> "Shingo acredita que KPIs atrasados frequentemente refletem a cultura e comportamentos da empresa, sublinhando a necessidade de líderes fomentarem um ambiente que direciona os resultados certos através de sistemas e princípios alinhados."

### Implicação para A3

- Indicadores são CONSEQUÊNCIA de sistemas
- Mudar indicador sem mudar sistema é inútil
- Foque contramedidas em sistemas, não em "melhorar o número"

## Erros Comuns vs. Abordagem Shingo

| Erro Comum | Por que é Erro (Shingo) | Abordagem Correta |
|------------|------------------------|-------------------|
| Culpar operador | Foco em comportamento, não sistema | Perguntar "qual sistema permitiu?" |
| Treinar como solução | Ação administrativa, mais fraca | Mudar sistema para tornar erro impossível |
| Criar mais regras | Adiciona complexidade sem mudar sistema | Simplificar e tornar padrão visual |
| Aumentar supervisão | Dependência de pessoas para detectar | Implementar detecção automática |
| Punir erros | Cria cultura de esconder problemas | Celebrar descoberta de problemas |

## Checklist de Alinhamento Shingo no A3

### Na Análise de Causa Raiz
- [ ] Causas são sistêmicas, não pessoais?
- [ ] Perguntamos "qual sistema permitiu?" para cada causa?
- [ ] Evitamos culpar indivíduos?

### Nas Contramedidas
- [ ] Contramedidas mudam SISTEMAS?
- [ ] Preferimos Prevenção > Detecção > Administrativo?
- [ ] O novo sistema torna o erro difícil ou impossível?

### No Monitoramento
- [ ] Indicadores medem eficácia do SISTEMA?
- [ ] Existe indicador de comportamento (leading)?
- [ ] Existe indicador de resultado (lagging)?

### Na Cultura
- [ ] O processo A3 está sendo usado para desenvolver pessoas?
- [ ] Liderança faz coaching através de perguntas?
- [ ] Há respeito por cada indivíduo envolvido?

## Resumo: Sistema → Comportamento → Resultado

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   "Quanto mais profundamente alguém entende os PRINCÍPIOS,     │
│    mais claramente entende o COMPORTAMENTO ideal.              │
│    Quanto mais claramente entende o comportamento ideal,       │
│    melhor pode projetar SISTEMAS para direcionar esse          │
│    comportamento e alcançar RESULTADOS ideais."                │
│                                                                 │
│                              — Modelo Shingo                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**No A3**: Sempre mude o SISTEMA. O comportamento mudará como consequência. O resultado será sustentável.
