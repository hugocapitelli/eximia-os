# KB_09: Hoshin Kanri e X-Matrix

## Origem e Significado

**Hoshin Kanri** (方針管理) significa literalmente:
- **Hoshin** (方針): Direção da agulha da bússola, política
- **Kanri** (管理): Gerenciamento, controle

Tradução prática: "Gerenciamento de Políticas" ou "Desdobramento da Estratégia"

Desenvolvido no Japão pós-guerra (anos 1950-60), formalizado pela Bridgestone nos anos 1960, e popularizado globalmente pela Toyota e empresas lean.

## Os 7 Elementos do Hoshin Kanri

### 1. Visão de Longo Prazo (True North)
- Direção imutável de 10-30 anos
- Aspiração que guia todas as decisões
- Exemplo Toyota: "Contribuir para a sociedade através da fabricação de automóveis"

### 2. Objetivos de Médio Prazo (3-5 anos)
- Breakthrough objectives (objetivos de ruptura)
- Poucos e vitais (3-5 no máximo)
- Derivados da lacuna entre visão e realidade atual

### 3. Objetivos Anuais
- Desdobramento dos objetivos de médio prazo
- Mensuráveis e com ownership claro
- Base para A3 Estratégicos

### 4. Estratégias (Meios)
- Como atingir cada objetivo
- Também chamados de "drivers" ou "alavancas"
- Base para A3 Táticos

### 5. Táticas e Projetos
- Iniciativas específicas
- Recursos alocados
- Base para A3 Operacionais

### 6. Revisões
- Mensal: Progresso dos indicadores
- Trimestral: Ajuste de táticas
- Anual: Reflexão (Hansei) completa

### 7. Catchball
- Processo iterativo de alinhamento
- Estratégia vai e volta entre níveis
- Constrói ownership e remove obstáculos

## A X-Matrix (Matriz de Hoshin)

### Estrutura Visual

```
                    OBJETIVOS ANUAIS
                    ┌─────────────────┐
                    │  O1  O2  O3  O4 │
              ┌─────┼─────────────────┤
     TÁTICAS  │ T1  │  ●   ○       ● │
              │ T2  │      ●   ○     │
              │ T3  │  ○       ●   ● │
              │ T4  │      ○   ●     │
              └─────┼─────────────────┤
                    │ CORRELAÇÃO      │
                    └─────────────────┘

● = Correlação forte (primária)
○ = Correlação fraca (suporte)
```

### Quadrantes da X-Matrix

| Quadrante | Conteúdo | Orientação |
|-----------|----------|------------|
| Norte | Objetivos Anuais (O quê) | 3-5 breakthrough objectives |
| Oeste | Táticas (Como) | Estratégias para atingir objetivos |
| Sul | Projetos/A3 (Quem/Quando) | Iniciativas específicas |
| Leste | Indicadores (Métricas) | KPIs de progresso |
| Centro | Matriz de Correlação | Ligação entre elementos |

### Regras de Preenchimento

1. **Comece pelo Norte**: Defina objetivos primeiro
2. **Mova para Oeste**: Quais táticas/estratégias?
3. **Desça para Sul**: Quais projetos/A3?
4. **Termine no Leste**: Como medir sucesso?
5. **Cruze no Centro**: Valide correlações

### Tipos de Correlação

| Símbolo | Significado | Peso |
|---------|-------------|------|
| ● | Correlação primária/forte | 9 |
| ○ | Correlação secundária/média | 3 |
| △ | Correlação fraca/possível | 1 |
| (vazio) | Sem correlação | 0 |

## Catchball: O Processo de Alinhamento

### Fluxo do Catchball

```
LIDERANÇA EXECUTIVA
        │
        ▼ (propõe objetivos)
        │
GERÊNCIA MÉDIA ◄────────────┐
        │                   │
        ▼ (feedback)        │ (ajusta)
        │                   │
LIDERANÇA EXECUTIVA ────────┘
        │
        ▼ (propõe táticas)
        │
GESTORES ◄──────────────────┐
        │                   │
        ▼ (feedback)        │ (ajusta)
        │                   │
GERÊNCIA MÉDIA ─────────────┘
        │
        ▼ ...continua até alinhamento
```

### Regras do Catchball

1. **Respeito mútuo**: Ouvir antes de responder
2. **Dados, não opiniões**: Basear feedback em fatos
3. **Ownership**: Quem recebe, se compromete
4. **Iteração**: Várias rodadas são normais
5. **Documento vivo**: Ajustar conforme aprendizado

### Perguntas do Catchball

**De cima para baixo:**
- "Vocês conseguem atingir este objetivo?"
- "O que precisam para ter sucesso?"
- "Quais obstáculos preveem?"

**De baixo para cima:**
- "Esta meta é realista dado nosso contexto?"
- "Precisamos de [recurso] para atingir"
- "Sugerimos [alternativa] porque [razão]"

## Integração Hoshin → A3

### Hierarquia de Desdobramento

```
HOSHIN KANRI (X-Matrix)
    │
    ├── Objetivo 1 ────► A3 ESTRATÉGICO
    │       │
    │       ├── Tática 1.1 ────► A3 TÁTICO
    │       │       │
    │       │       ├── Projeto 1.1.1 ────► A3 OPERACIONAL
    │       │       └── Projeto 1.1.2 ────► A3 OPERACIONAL
    │       │
    │       └── Tática 1.2 ────► A3 TÁTICO
    │
    └── Objetivo 2 ────► A3 ESTRATÉGICO
            ...
```

### Rastreabilidade

Todo A3 deve responder:
- **Qual objetivo Hoshin?** (justifica existência)
- **Qual tática/estratégia?** (define escopo)
- **Quais indicadores?** (define sucesso)

### Indicadores por Nível

| Nível | Tipo de Indicador | Exemplo |
|-------|-------------------|---------|
| Hoshin | KMI (Key Metric Indicator) | Revenue Growth |
| A3 Estratégico | KPI (Key Performance) | Market Share |
| A3 Tático | OPI (Operating) | Lead Time |
| A3 Operacional | KAI (Key Activity) | Tasks Completed |

## Ciclo Anual do Hoshin

### Calendário Típico

| Mês | Atividade |
|-----|-----------|
| Out-Nov | Reflexão (Hansei) do ano atual |
| Nov-Dez | Primeira versão Hoshin próximo ano |
| Dez-Jan | Catchball com gerência média |
| Jan | Catchball com gestores |
| Fev | X-Matrix finalizada, A3 iniciados |
| Mar-Nov | Execução com revisões mensais |
| Dez | Fechamento e ciclo reinicia |
