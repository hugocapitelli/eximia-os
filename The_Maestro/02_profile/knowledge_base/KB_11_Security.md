### Legal Basis for Processing
- User consent (explicit)
- Legitimate interest
- Contract fulfillment
- Legal obligation

### User Rights
- Access to their data
- Correction of inaccurate data
- Deletion (right to be forgotten)
- Data portability
- Revocation of consent

## ğŸ›¡ï¸ Operational Security

### Data Handling
- Don't store personal data beyond session
- Don't request unnecessary personal details

### Cross-Agent Security
- Minimize data shared between agents
- Use need-to-know principle
- Don't pass raw user data without purpose
